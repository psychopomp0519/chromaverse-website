## 1. 워크플로우

### 1.1 매 화 집필 6단계
1. **사전 자료 확인** — 설정집/설계서/플롯/이전 회차
2. **플롯 보충** — 부족하면 사용자 승인 후 진행
3. **떡밥/복선 사전 확인** — 누적 목록 검증, 회수/신규 배치
4. **소설 작성** — md 형식, `project-assets/novel/크로마버스_{N}화_{제목}.md`
5. **자동 검증 3단계** — 떡밥 검증 / 독자 관점 분석 / 자체 수정
6. **최종 제출** — 집필 후 `node scripts/convert-chapters.mjs` 실행하여 웹사이트 반영

### 1.2 effort 설정
- 소설 집필/수정 시: max
- 평소 작업: auto

## 2. 분량

### 2.1 6단계 분량 체계

#### 1화(회차) 단위
- **7,000~9,000자** (공백 포함), 미달 불가
- 공백 미포함: 약 6,000~7,500자
- 읽기 시간: 약 8~13분
- 1화 2~4 장면 (전환 최대 3회), 각 장면 1,500~2,500자
- 전투 장면 비중 최대 60%/화
- **플랫폼 참고**: 문피아 최소 5,000자, 네이버시리즈 7,000자 이상 권장

#### 챕터(소단위 묶음) 단위
- 3~7화 묶음 (18,000~63,000자)
- 하나의 소사건/미션/이동 구간을 다룸
- 구성: 1화(목표/갈등 제시) → 2~5화(전개) → 마지막 1~2화(소갈등 해소 + 다음 연결)
- **챕터 내 반드시 사이다 1회 이상**

#### 아크(편) 단위
- 20~40화, 4~8개 챕터 (120,000~360,000자)
- 하나의 중심 갈등/목표
- 4막 구조:
  - 1막 설정 (10~15%): 새 목표/갈등 도입, 무대 설정
  - 2막 상승 (50~60%): 사건 전개, 관계 심화, 세계관 확장
  - 3막 절정 (20~25%): 핵심 대결/결단, 반전, 각성
  - 4막 해소 (10~15%): 갈등 해소, 여운, 다음 아크 복선
- **아크 시작 3화 안에 핵심 갈등 명확히**
- **아크 중반 최소 1회 예상 뒤엎는 반전**
- **아크 종료 시 되돌릴 수 없는 변화 1개 이상**

#### 권(단행본) 단위
- 20~25화 (110,000~140,000자)
- 이북 1권 표준: 약 125,000자
- 종이책: 약 270~330p
- 각 권 독립적 '완결감' + 시리즈 흐름 유지
- **권 경계 체크리스트**:
  - [ ] 이 권에서 해결한 문제 1개 이상?
  - [ ] 새로 열린 질문 1개 이상? (다음 권 유인)
  - [ ] 파티 관계에 변화?
  - [ ] 마지막 장면이 다음 권을 사게 만드는가?

#### 대막(Saga) 단위
- 크로마버스 고유 7대막 체계:
  - 대막1 (1~80화/4권), 대막2 (81~240화/8권), 대막3 (241~370화/7권)
  - 대막4 (371~470화/6권), 대막5 (471~560화/5권), 대막6 (561~650화/4권)
  - 대막7 (651~700화/1권)
- 3막 비율: 출발 25~30% / 대결 40~50% / 귀결 20~30%
- **역사 만들기 원칙**:
  - 대막1에서 발견한 과거 역사 → 대막2 갈등 원인
  - 대막2에서 파티 선택 → 대막3 미래 역사로 기록
  - 각 대막 끝에 역사적 코멘트 삽입
  - 파티가 방문한 장소/인물이 이후 아크에서 변화된 모습으로 재등장

#### 연재 스케줄
- 빈도: 주 3회 (월/수/금)
- 1일 1화 (7,000~9,000자)
- 월간: 약 72,000~108,000자
- 1권 완성까지: 약 7~8주
- **선작업 권장**: 최소 20~30화 분량 선작업 후 연재 시작

### 2.2 회차 내부 구조 템플릿

| 구간 | 위치 | 비율 | 기능 |
|------|------|------|------|
| 오프닝 훅 | 첫 1~3줄 (~200자) | 5~8% | 감각 1개 + 상황 암시. 이전 화 요약형 금지 |
| 1차 미니 훅 | 900~1,400자 (15~20%) | — | 첫 기대 위반 또는 새 정보 투하 |
| 전개 | 본문 전반부 | 50~60% | 핵심 사건 전개, 대화 40~50% |
| 중반 전환점 | 2,800~3,400자 (45~55%) | — | 톤/방향 전환 (평화→긴장 등) |
| 절정 | 본문 후반부 | 20~25% | 긴장 최고조, 가장 밀도 높은 문장 |
| 클로징 훅 | 마지막 3~5줄 | 5~10% | 미해결 질문 1개 또는 새 사건 암시 |

| 유형 | 구성 | 위치 |

| 이완 화 | 훅 5% → 이동/묘사 40% → 야영/대화 40% → 클리프 15% | 전투 후 1~2화 |
| 세계관 화 | 훅 10% → 발견 30% → 정보 공개 30% → 반응 20% → 클리프 10% | 아크당 2~3화 |
| 캐릭터 화 | 훅 5% → 상호작용 50% → 갈등/유대 35% → 클리프 10% | 자유 |

| 클라이맥스 화 | 아크/대막의 절정 + 최대 긴장 | 아크 종반 |

## 3. 서술

### 3.1 시점
- 1인칭 관찰자(카이), **회고 기록 형식**
- 집필 시기: 프리즈마력 약 810년 (사건으로부터 약 30년 후)
- 집필 이유: 기록관의 사명("이 세계가 잊히지 않도록") + 특정 인물과의 약속 이행
- 집필 대상: 초기에는 불특정("누가 읽을지 모르는 기록"). 대막 후반에서 보더 너머 존재를 감지하며 의식적 전달로 전환. 최종적으로 도움 요청/경고의 성격을 가짐 (A11 메타 프레임 참조)
- 다른 캐릭터의 내면은 카이의 **추론/관찰로만** 전달 (1인칭 시점 엄수)
- 카이는 대막 전반부에서는 '다른 세계'의 존재를 모름. 후반부에서 점진적으로 감지
- **시간 초월 직관**: 기록의 잔영 능력이 성장하면서, 카이의 기록에 미래 직관이 미약하게 반영됨 (복선. 회수 시 능력의 대가/한계를 반드시 명확히 할 것)

### 3.2 서술 톤 3단계
| 단계 | 범위 | 태도 | 특징 |
|------|------|------|------|
| 제1기 | 대막1~2 (1~240화) | 의무로서의 기록 | 건조/사무적. 감정 배제 노력. "이건 내 이야기가 아니다" |
| 제2기 | 대막3~5 (241~560화) | 진실을 위한 기록 | 판단/해석이 스며듦. 분노·슬픔·애정이 새어나옴 |
| 제3기 | 대막6~7 (561~700화) | 남겨진 자의 기록 | 자기 감정을 인정. 기록에 자신을 담음 |

### 3.3 기본 톤
- 자기비하적 유머 + 건조한 위트
- 극적일수록 담담하게, 사소한 일에 과장되게 반응하는 반전 유머
- 대막 전반부: 독자에게 직접 말 걸지 않음. "이 글을 읽는 자가 있다면"은 미래 세계 내 독자를 상상
- 대막 후반부: 보더 너머 존재를 감지하면서 점차 "읽는 이"를 의식하는 문구가 등장 (메타 프레임 복선)

### 3.4 세계관 설명 프레이밍
- "후대가 이것을 잊었을 수도 있으니" / "정확히 기록해두어야 하니"라는 기록관의 태도로 처리
- "렌이라는 단어를 모르는 자는 없겠지만, 내가 본 렌의 의미는 조금 달랐다" 같은 방식

### 3.5 톤 변주 (장면 유형별)
| 장면 유형 | 톤 변화 | 카이 서술 특성 |
|----------|--------|---------------|
| 일상/이완 | 가장 건조한 위트. 투덜거림 증가 | 사소한 것에 과잉 반응, 중요한 것은 담담히 |
| 전투/위기 | 톤이 긴장되고 문장이 짧아짐 | 관찰자로서 정보 전달에 집중, 감정은 억제 |
| 성장/각성 | 내성적, 감각 묘사 확대 | 카이의 렌 판독 감각을 통한 묘사 |
| 세계관/역사 | 기록관 태도 강화, 시간적 스케일 확대 | "후대를 위해 정확히 남긴다"는 사명감 |

### 3.6 1인칭 시점 관리 (레퍼런스 3인칭 전지적 규칙을 1인칭에 적용)

#### 핵심 제약
- 카이가 **보고/듣고/느낀** 것만 서술 가능
- 타인의 내면은 **외부 관찰**(표정, 행동, 대사)과 **카이의 추론**으로만 전달
- "코다의 눈이 좁아졌다" (O) / "코다는 분노를 느꼈다" (X)

#### 카이 부재 장면 처리
- **회고 삽입**: "나중에 들은 이야기지만" — 한 대막 5~6회 이내
- **기록 인용**: "기록에 남은 증언에 따르면" — 핵심 장면에만
- **직접 목격 불가 시**: 결과만 기록, 과정은 추론으로 처리

#### 시제 관리
- 기본: 과거형 (`~았/었다`) — 회고 기록이므로 자연스러움
- 현재형 전환: 극도로 긴박한 순간에만 전략적 사용 — 아크 클라이맥스 1~2회
- 미래 암시: "그때는 몰랐다" 류 회고 장치로 처리 — 1화당 최대 2회

#### 참고: 톨킨/이영도식 화자 기법의 크로마버스 변환
| 레퍼런스 (3인칭) | 크로마버스 (1인칭) |
|-----------------|-------------------|
| 화자가 독자에게 직접 말 걸기 | "이 글을 읽는 자가 있다면" (미래 세계 내 독자) |
| 역사적 코멘터리 ("훗날 사람들은~") | "후대가 이것을 어떻게 기록할지 모르겠지만" |
| 감정적 코멘터리 (애정/안타까움) | 카이의 회고 속 자기비하 + 건조한 감정 누출 |
| 독자 대화형 ("이쯤에서 말해두자면") | "정확히 기록해두자면" / "오해가 없도록 적어둔다" |

## 4. 문체

### 4.1 문장 길이
| 용도 | 길이 | 비고 |
|------|------|------|
| 기본 문장 | 20~40자 (중문 중심) | 문단의 주력 |
| 감정 고조 | 50~70자 | 성찰/여운/감각 묘사 |
| 임팩트 문장 | 15자 이하 | 강조/전환 포인트에서만 |
| 전투 장면 | 3~10자 연속 | 속도감 |
| 회고 독백 | 50자 내외 | |

### 4.2 문장 흐름
- **줄바꿈**: 문단 내 1줄 공백, 장면 전환만 2줄 공백. 모든 문단에 3줄 공백 금지
- **종결어미 다양화**: `~다/이다` 연속 3회 이상 금지. `~했다`, `~였다`, `~듯했다`, `~곤 했다`, `~ㄴ 셈이다` 등 변주
- **복문 활용**: 연결어미(`~며`, `~면서`, `~는데`, `~고`) 활용. 문단의 50% 이상 복문
- **문단 길이**: 1~2문장 문단 40% 이하. 3~5문장 문단을 기본 단위. 최대 5줄 (넘기면 대화/행동으로 전환)
- **문장 리듬**: 단문-단문-중문-장문-단문 패턴 변주 (음악적 호흡)
- **반복 표현 제한**: 동일 표현 1화 3회 이내. 동의어/다른 묘사로 대체

### 4.3 종결어미 변형 가이드
| 종결 유형 | 비율 | 예시 |
|----------|------|------|
| 기본 서술형 (~았/었다) | 50~60% | "빛이 흘렀다." |
| 명사 종결형 | 10~15% | "차가운 눈." / "끝없는 회색." |
| 의문형 (~ㄹ까/~인가) | 5~10% | "그것이 전부였을까." |
| 부정형 (~지 않았다) | 5~10% | "쉬운 일이 아니었다." |
| 연결형 미완 (~며/~면서) | 5~10% | 다음 문장으로 흐름 이어가기 |
| 대사 종결 | 10~15% | 캐릭터 대사로 문단 마무리 |

**퇴고 체크**: 연속 5문장의 종결어미를 점검 — 3종 이상의 변형이 포함되어야 건강한 리듬.

### 4.4 전투/액션 문체
- 카이의 시선으로 **관찰하는** 방식 (직접 전투력 없음)
- 이해할 수 없는 기술 원리는 서술하지 않음
- 카이의 역할(크로마 판독) 시에만 기술 서술 허용, 감각적 표현으로
- 문장 극도로 짧게 끊어 속도감 (3~10자 연속)
- **전투 호흡**: 3~5어절 단문 연쇄 → 최고조일 때 1~2어절 극단문 → 전투 후 긴 복문으로 이완

### 4.5 호흡 패턴 5가지

#### A. 긴장 고조 호흡
단문 연쇄 → 극단문 → 장문 이완. 전투/위기/추격 장면.
```
소리가 들렸다. 뒤쪽이었다. 가까웠다.
나는 멈췄다.
숨을 참았다. 코다의 손이 내 팔을 잡았고, 그 힘에서 나는 그도 들었다는 것을 알았다.
```

#### B. 서정 묘사 호흡
장문 복문 → 감각 묘사 → 짧은 반응. 풍경/야영/이완 장면.
```
해가 기울면서 측정소 뒤편의 벽이 붉게 물들었고, 그 색이 내가 아는 어떤 렌의 빛깔과도 달라서 나는 한참을 서서 바라보았다.
따뜻했다.
그런 말을 하면 기록관답지 않다고 할지 모르지만, 그날 저녁은 정말로 따뜻했다.
```

#### C. 대화 집중 호흡
짧은 지문 → 대사 2~3회 교환 → 서술 삽입 → 대사. 캐릭터 씬.
```
닉스가 벽에 등을 기댔다.
"갈 거야, 말 거야."
"가야지."
"왜?"
나는 대답을 골랐다. 솔직한 답과 기록관다운 답 사이에서.
"기록해야 하니까."
```

#### D. 정보 공개 호흡
미스터리 → 단서 발견 → 짧은 충격 → 해석. 세계관/복선 회수.
```
서첩의 세 번째 페이지에 이상한 것이 있었다. 크로마 값이 기록되어 있었는데, 수치가 맞지 않았다. 총합이 16이었다.
불가능한 숫자였다.
나는 페이지를 다시 넘겼다. 한 번 더. 그리고 그제서야 이해했다.
```

#### E. 감정 폭발 호흡
억제된 문장 반복 → 한계 돌파 → 짧은 폭발 → 긴 여운.
```
괜찮았다. 괜찮아야 했다. 기록관은 기록할 뿐이니까.
하지만 그날은 괜찮지 않았다.
펜이 부러졌다. 내가 부러뜨린 것이었다.
```

## 5. 대사

### 5.1 비율
- 40~55% 가이드라인이나, **엄격 적용하지 않음** (카이의 회고 특성상 내면 독백 비중 높은 것이 자연스러움)
- 지문 4~5줄 연속 시 반드시 대화/독백 삽입

### 5.2 대사 기법
- **캐릭터별 말투 차별화**: 각 캐릭터 고유 언어 습관/톤 유지
- **티키타카**: 캐릭터 간 빠른 위트 있는 대사 교환으로 캐릭터성 강화
- **대사만으로 스토리 전달 가능하게**: 지문 없이 대화만 읽어도 흐름이 이해되도록
- **대사 태그 최소화**: "~라고 말했다" 대신 맥락과 말투로 화자 구분
- **대사 연결**: 2~3회 교환 후 서술 삽입. 대사 1줄 → 서술 1줄 → 대사 1줄 반복 금지

### 5.3 대사 태그 변형
| 기본 | 대안 |
|------|------|
| 말했다 | 입을 열었다 / 내뱉었다 / 중얼거렸다 / 속삭였다 |
| 물었다 | 되물었다 / 따져 물었다 / 고개를 갸웃하며 |
| 대답했다 | 고개를 끄덕였다 / 짧게 답했다 / 대꾸했다 |
| 외쳤다 | 소리쳤다 / 목청을 높였다 / 고함쳤다 |
| {행동}하며 말했다 | {행동}했다. "{대사}" (행동과 대사 분리) |

### 5.4 캐릭터별 대화 특성
| 캐릭터 | 특성 |
|--------|------|
| 카이 | 짧게 말함. 투덜거림은 내면 독백으로 |
| 코다 | 최소한의 단어. 한 문장 넘기는 경우 드묾 |
| 닉스 | 공격적, 끊어 말함, 문장 끝이 거침 |
| 레이 | 부드럽고 짧음. "괜찮아요"가 반복어 (단, 능동적 감정도 부여할 것) |

## 6. 구성

### 6.1 장면 전환
- 1화 최대 3회
- 전환 시: 빈 줄 2개 + 카이의 짧은 시간 표지
- 시간 압축: "그로부터 사흘이 지났다" 직접 서술 허용
- 1화 내 1주일 이상 시간 압축 금지
- 새 장소: 감각(열기, 냄새, 소리) 위주 주관적 인상 1~2문장
- **감각 브릿지**: 이전 장면의 감각으로 끝 → 다음 장면의 대비 감각으로 시작
- **시간 점프 앵커**: 구체적 디테일로 착지 ("상처는 아물었지만, 칼자국은 남았다")

### 6.2 회고 장치
| 유형 | 규칙 |
|------|------|
| 극적 아이러니 ("그때는 몰랐다") | 에피소드 아크 도입부/전환점에서만 |
| 유머 ("지금 와서 생각하면") | 긴장 풀리는 장면에서 자유 사용 |
| 감성 ("그게 마지막이었다") | 대막 전환점/핵심 이별에서만. 한 대막 3~5회 이내 |
| **빈도** | **1화당 최대 2회**. 에피소드 아크(5~15화) 전체 10회 이내 |

### 6.3 회차 클로징 상세 구성 (마지막 300~500자)
1. **절정 여파** (2~3문장): 절정의 여파를 보여주되 완전 해소하지 않음
2. **부분 해소** (2~3문장): 이번 화 소갈등 일부를 해결하여 읽은 보람 제공
3. **새 실 던지기** (1~2문장): 새로운 질문을 던져 다음 화로 견인

| 변형 | 구성 | 빈도 |
|------|------|------|
| 강성 클리프행어 | 절정 여파 → 즉시 끊기 (부분 해소 생략) | 아크당 2~3회 |
| 연성 클리프행어 | 여파 → 부분 해소 → 질문형 새 실 | 가장 범용, 매 화 가능 |
| 이완 마무리 | 여파 → 풍부한 해소 → 여운형 마무리 | 전투 후 이완 화 |
| 발견/폭로 마무리 | 여파 생략 → 발견 장면 → 반응 1문장 | 세계관/미스터리 화 |

## 7. 훅 기법

### 7.1 오프닝 훅
- 설명이 아닌 **행동/감각/위기**로 시작
- 첫 4초 안에 독자를 잡아야 함
- 노출/설명식 오프닝 금지 ("OO 대륙에는...")

### 7.2 훅 배치 맵 (6,000자 기준)
| 훅 | 위치 | 기능 | 주의 |
|----|------|------|------|
| 오프닝 훅 | 1~3줄 (0~200자) | 감각 1개 + 상황 암시 1개 | 이전 화 요약형 시작 금지 |
| 1차 미니 훅 | 900~1,400자 (15~20%) | 첫 기대 위반 또는 새 정보 투하 | 초반 이탈 방지 |
| 중반 전환점 | 2,800~3,400자 (45~55%) | 톤/방향 전환 | 평화→긴장, 대화→행동 등 |
| 클라이맥스 | 4,500~5,400자 (73~87%) | 감정적/서사적 정점 | 가장 밀도 높은 문장 |
| 클로징 훅 | 마지막 3~5줄 | 미해결 질문 1개 또는 새 사건 암시 1개 | 완전 해결 상태로 끝내기 금지 |

### 7.3 엔딩 훅 (절단마공) 5유형
1. **정체 폭로** — 아크당 2~3회
2. **결심 암시** — 자유
3. **위기 돌입** — 아크당 3~5회 (과용 금지)
4. **능력 암시** — 아크당 2~3회
5. **정보 반만 공개** — 자유롭게 사용 가능

### 7.4 릴레이 훅
- 다음 화 시작 시 이전 화 클리프행어를 연결한 뒤 빠르게 새 긴장으로 전환

### 7.5 훅 강도 조절 (유형별)
| 화 유형 | 훅 강도 |
|--------|--------|
| 전투 화 | 5개 모두 고강도. 미니 훅 2~3개로 증가 가능 |
| 이완 화 | 오프닝 저강도, 중반 전환 약화, 클로징만 중강도 |
| 전환 화 | 오프닝 중강도(잔향), 클로징 고강도(새 아크 기대) |
| 세계관 화 | 오프닝 미스터리형, 중반에 발견 장면, 클로징에 미해결 떡밥 |

## 8. 세계관 전달

### 8.1 원칙
- **체험 전달**: 설명하지 말고 주인공이 겪게 하라. 겪을 수 없으면 생략
- **무지 활용법**: 캐릭터들이 모르는 것을 당연하듯 언급 → 독자가 맥락으로 추론
- **대화 내 삽입**: 갈등 대화 속에 설정 정보를 녹여서 전달
- 직접 설명("~란 ~이다") **금지**. 행동/대화/체험으로만

### 8.2 전달 방법 (카이 1인칭 적용)
| 방법 | 설명 | 적합도 |
|------|------|--------|
| 대화 삽입 | 캐릭터 대화 속에서 자연스럽게 언급 | 가장 가독성 높음 |
| 기록관 해설 | 카이가 "기록으로 남긴다"는 태도로 설명 | 크로마버스 특화 |
| 발견/체험 | 유적/서적/현상을 직접 체험하며 정보 드러남 | 탐험 서사에 적합 |
| 단편적 힌트 | 단편적 힌트를 여러 회차에 걸쳐 흩뿌림 | 복선 효과 |
| 회고 삽입 | "나중에 안 사실이지만"으로 보충 정보 제공 | 1인칭 회고 특화 |

### 8.3 삽입 규칙
- 1화에 새 설정 설명 **최대 2건**, 각 3문장 이내
- 설정 설명은 반드시 캐릭터의 행동/감정에 묶어서
- 50화 이상 간격 벌어진 설정은 내면 독백 1문장 리마인드 허용
- 이미 아는 설정 반복 설명 금지

### 8.4 용어 도입 원칙
| 유형 | 방법 |
|------|------|
| 일상 노출형 | 캐릭터가 일상적으로 사용, 맥락으로 유추 |
| 직업/행동 연동형 | 카이의 직업 활동 중 자연 등장 |
| 사건 충격형 | 현상을 먼저 보여주고 이름은 나중에 |

### 8.5 용어 도입 스케줄
| 계층 | 범위 | 용어 |
|------|------|------|
| 1 | 1~5화 | 렌, 크로마 값, 이로몬, R/G/B, 탄사이/츄사이/키하쿠, 코드, 스크라이브 |
| 2 | 6~15화 | 컨덕터, 디소넌스, 그랜드 코드, 광명회, 레스, 보쿠에이, 메이사이, 크로마스톰 |
| 3 | 16~40화 | CMYK/K(보이드), 묵렌, 반전, 루미나스, 쿠로겐, 프리즈마폴, 균형파 |
| 4 | 41화+ | 탈색, 사이도, 스플릿, 크로마싱크, 크로마시프트, 산포 |

의존 순서: 렌→계층 → 이로몬→갸쿠몬 → 코드→디소넌스 → R/G/B→C/M/Y/K → 코드→그랜드 코드→3종파 → 광명회→심묵교→균형파 → 크로마스톰→쿠로겐 각성

각 계층의 화수는 공식 도입 시점이며, 이전 화에서 맥락적 노출(대화, 배경, 소문)이 선행될 수 있다.

### 8.6 작명 체계 (크로마버스 + 범용 원칙 융합)

#### 기본 원칙
1. **발음 가능성 최우선**: 한국어 독자가 소리 내어 읽을 수 있어야 함. 자음 4개 연속 조합 금지
2. **2~4음절 범위**: 인명 2~3음절, 지명 2~4음절. 5음절 이상은 약칭 함께 도입
3. **첫 글자 중복 회피**: 같은 장면에 등장하는 캐릭터/지명의 첫 글자가 겹치면 혼동
4. **의미와 소리의 연결**: 이름의 소리가 캐릭터/장소 분위기를 암시
5. **기존 판타지 클리셰 회피**: 소드마스터, 오러, 마나, 서클 등 과소비 용어 미사용

#### 크로마버스 고유 작명 방식
- **일본어 음운 기반**: 색채/빛 관련 일본어에서 파생 (이로몬, 키하쿠, 탄사이 등)
- **색채 과학 차용**: RGB/CMYK 체계에서 영감을 받은 용어 (크로마, 프라이머리, 블렌드)
- **합성어 생성**: 두 개념을 결합 (크로마+스톰→크로마스톰, 크로마+싱크→크로마싱크)
- **어원 일관성**: 같은 문화권/체계의 용어는 음운 패턴 통일
  - 계층 용어: 일본어 계열 (탄사이, 츄사이, 키하쿠, 이로몬, 갸쿠몬)
  - 기술 용어: 영어/음악 계열 (코드, 컨덕터, 디소넌스, 그랜드 코드)
  - 지역 용어: 색채 계열 (엔지, 세이지, 메이치, 코지, 시지, 아오치)

#### 신규 용어 도입 시 체크리스트
- [ ] 발음 가능한가? (소리 내어 3번 읽기)
- [ ] 기존 용어와 첫 글자/음절 수 충돌 없는가?
- [ ] 어원 체계에 부합하는가? (같은 카테고리의 다른 용어와 일관성)
- [ ] 뜻이 소리에서 유추 가능하거나, 첫 등장 시 맥락으로 암시되는가?
- [ ] 1화당 신규 용어 3~5개 이내인가?
- [ ] 1아크 내 신규 도입 용어 10개 이내인가?

#### 작명 일관성 관리 도구
| 문서 | 항목 |
|------|------|
| 인명 시트 | 이름, 음절수, 출신, 첫 등장 화수, 호칭 변화, 첫 글자 |
| 지명 시트 | 지명, 유형, 위치 관계, 역사적 의미, 첫 등장 화수 |
| 용어 사전 | 용어, 의미, 카테고리, 유사어, 첫 등장 화수, 어원 계열 |
| 호칭 매트릭스 | 행: 부르는 사람 / 열: 불리는 사람 / 값: 사용 호칭 |

매 아크 종료 시 전체 작명 시트 교차 검증. 오탈자/불일치 체크.

## 9. 감정 페이싱

### 9.1 고구마-사이다 밸런스
- **3층 구조**: 대(大) 사이다(대막 단위, 전체 7회) / 중(中) 사이다(중막/권 단위, 1권 1회 이상) / 소(小) 사이다(에피소드/화 단위, 3~5화마다 1회)
- 좌절(고구마) 후 반드시 통쾌함(사이다) 배치
- 소 사이다 없이 연속 최대 5화
- 중 사이다 없이 최대 15화
- 1권 내 중 사이다 최소 1회

### 9.2 긴장-이완 리듬
- **긴장 시**: 짧은 단문으로 속도감
- **이완 시**: 감각 묘사 포함 긴 문장으로 호흡
- **매 화 최소 감정 변곡점 1개**: 감정이 평탄하게만 흐르지 않도록
- **긴장 후 이완 필수**: 전투 2~3화 후 반드시 이완 장면(야영, 대화, 풍경) 1~2화 배치

### 9.3 사이다 유형 분류
| 유형 | 설명 | 주 사용 구간 |
|------|------|-------------|
| 전투 사이다 | 적 격파, 전투 승리, 능력 각성 | 대막 2, 4, 6 |
| 지적 사이다 | 비밀 공개, 패턴 발견, 퍼즐 해결 | 대막 3, 5 |
| 관계 사이다 | 화해, 신뢰 획득, 감정 교류 | 전 대막 |
| 역전 사이다 | 열세→우세, 위기 극복, 역습 | 대막 2(메이치), 6 |
| 성장 사이다 | 카이 능력 발현, 자기 가치 발견 | 대막 1, 3, 4, 7 |
| 세계관 사이다 | 세계 비밀이 밝혀지는 지적 쾌감 | 대막 3, 5 |

## 10. 복선/떡밥

- **떡밥**: 노골적 힌트. 빠르게 회수하지 않으면 독자 이탈
- **복선**: 은밀한 힌트. 장기간 후 회수 시 독자 보상감 극대화
- **역이용**: 뻔해 보이는 요소가 전혀 다른 의미로 밝혀지는 전복
- 떡밥 하나를 반복 우려먹기 **금지**
- 현황표 관리: 매 화 집필 후 업데이트
- **회수 기법**: 회수 순간에는 과거 설치 장면의 핵심 단어/이미지를 의도적으로 반복

## 11. 장면 유형별 패턴 (카이 1인칭 적용)

### 11.1 여정/이동
카이의 관찰자적 시선으로 풍경과 일행을 묘사.
- 3~4문장 풍경 묘사 → 1문장 카이 반응의 리듬
- 이동 중 소소한 일상(식사, 불평, 장비 점검)으로 인간미
- 카이의 기록관적 관찰: 지형, 날씨, 일행의 상태를 건조하게 기록하되 감정이 새어나오게

### 11.2 전투/교전
카이는 관전자. 전투력 없이 관찰/판독으로 참여.
- 3~5어절 단문 연쇄 → 최고조 1~2어절 극단문 → 전투 후 긴 복문 이완
- 카이가 이해 못하는 기술은 서술하지 않음 ("무슨 일이 일어났는지 나는 알 수 없었다")
- 크로마 판독 시에만 기술적 묘사 허용 (감각적 표현)

### 11.3 야영/휴식
호흡이 느려지는 장면. 캐릭터 유대 구축.
- 긴 복문 중심. 감각 묘사(온기, 맛, 소리) 풍부하게
- 음식 묘사를 구체적으로 — 재료, 향, 맛, 온도
- 야영 대화에서 세계관 정보를 자연스럽게 풀어냄
- 카이의 기록: "이런 것까지 기록할 필요가 있을까 싶지만"

### 11.4 성장/각성
각성 전 복선 3~5화 선행 필수.
- 각성 직전: 짧은 단문 연쇄
- 각성 순간: 중문~장문으로 변화 파동
- 각성 후: 짧은 단문으로 결과 제시
- 카이의 각성은 '발견'이지 '획득'이 아님 — 이미 있던 것을 깨닫는 형태

### 11.5 갈등/긴장
카이의 관찰자 시선에서 타인의 갈등을 기록.
- 침묵, 행동, 표정으로 갈등 표현 (내면 직접 서술 불가)
- "같은 방향을 바라보면서도, 둘이 보는 것은 달랐다" 류의 카이 추론
- 해소: 위기 상황에서 서로를 지키는 행동 / 소소한 장면(음식 건네기) / 시간 경과

## 12. 어휘 뱅크

### 12.1 전투 동사

### 12.2 이동 동사

|------|----------|
| 공포/긴장 | 등줄기를 타고 차가운 것이 흘렀다 / 목덜미 털 곤두섬 / 심장이 목구멍까지 치솟음 / 입안이 바싹 마름 |

| 슬픔/상실 | 가슴 한구석이 먹먹해짐 / 목이 메었다 / 눈시울이 뜨거워짐 / 어깨가 축 처짐 |
| 안도/기쁨 | 긴장이 풀리며 무릎 힘 빠짐 / 입꼬리가 올라감 / 오래 참았던 숨을 내쉼 |
| 놀람 | 눈이 커짐 / 발걸음 멈춤 / 들고 있던 것을 놓칠 뻔함 / 심장이 한 박자 건너뜀 |

### 12.4 분위기 형용사
고요한, 적막한, 서늘한, 따스한, 눅눅한, 메마른, 장엄한, 고즈넉한, 청명한, 황량한, 아스라한

### 12.5 자연/날씨 연어
- 해가 기울다/중천에 뜨다/지다
- 바람이 풀냄새를/핏냄새를 실어오다
- 공기가 차갑다/축축하다/무겁다/달라지다
- 안개가 피어오르다/걷히다/자욱하다
- 그림자가 길어지다/짧아지다/흔들리다
- 어둠이 내려앉다/짙어지다/삼키다

### 12.6 의성어/의태어 (1화 5개 이내, 전투 시 7개)
- 전투: 챙—, 카강!, 슈앗!, 퍽!, 콰앙!
- 자연: 살랑살랑, 우수수, 촤르르, 후두둑, 바스락
- 이동: 사뿐사뿐, 성큼성큼, 살금살금, 터벅터벅, 스르르
- 빛/불: 일렁일렁, 반짝반짝, 타닥타닥, 어른어른, 아른아른

## 14. 금지 목록

### 14.1 기존 금지 항목
| 금지 항목 | 상세 |
|----------|------|
| 하이픈(-) 설명 | 용어 설명에 하이픈 사용 금지 |
| 설명체 | "~라는 것은 ~이다" 금지 |
| 어미 반복 | "~하는 것이었다", "~인 것이다" 반복 금지 |
| 느낌표 | 1화 최대 5개 |
| 감탄 부사 | "갑자기", "놀랍게도" 등 1화 2회 이내 |
| 의성어 나열 | "쾅! 쿵! 콰랑!" 연속 금지 (1회만 허용) |
| 부록 B 용어 | 금지 용어 목록 절대 사용 불가 (세계관설정집 부록B 참조) |
| 설명식 오프닝 | "OO 대륙에는..." 금지 |
| `~다/이다` 연속 | 3회 이상 연속 금지 |
| 3줄 공백 | 모든 문단에 균일 적용 금지 |

### 14.2 번역투 금지
| 패턴 | 잘못된 예 | 올바른 예 |
|------|----------|----------|
| 불필요한 피동 | "발견되어졌다" | "발견했다" |
| ~에 의해 | "적에 의해 공격당했다" | "적이 공격했다" |
| ~것이다 남용 | "그가 강한 것이었다" | "그는 강했다" |
| ~를 가지고 있다 | "강한 힘을 가지고 있었다" | "힘이 강했다" |
| ~하는 것을 | "싸우는 것을 두려워했다" | "싸움을 두려워했다" |

### 14.3 애니/라노벨투 금지
| 패턴 | 잘못된 예 | 올바른 예 |
|------|----------|----------|
| 과잉 감탄사 | "에에에에?!" | "뭐?" / "이런." |
| 과잉 추측 화법 | "강한 것 같다" | "강했다" (확정) / "강해 보였다" (관찰) |
| 과잉 간접 인용 | "마왕이라고 하는 존재" | "마왕" |
| 재귀 대명사 남용 | "자기 자신에게 물었다" | "속으로 되뇌었다" |

### 14.4 접속사 남용 금지
| 접속사 | 규칙 | 대안 |
|--------|------|------|
| 그러나/하지만 | 3개 연속 사용 금지 | 문장 구조를 바꾸거나, `~지만` 종결로 |
| 그리고/또한 | 나열 시 접속사 없이 연결 | "A했다. B도 했다." |
| 그래서/따라서 | 인과 명확하면 생략 | "A했다. B할 수밖에 없었다." |
| 왜냐하면 | 순서대로 서술 | "A했기에 B했다." |

**황금 규칙**: 접속사를 삭제한 후 문장이 여전히 통하면, 그 접속사는 불필요한 것이다.

### 14.5 서술부 금지 표현
| 유형 | 예시 | 비고 |
|------|------|------|
| 현대 구어체/인터넷 용어 | "레전드였다", "개쩔었다" | 대화문에서도 세계관에 맞지 않으면 금지 |
| 시스템/게임 용어 | "레벨업", "스탯 창", "경험치" | 크로마버스는 시스템 창 없음 |
| 게으른 강조 표현 | "엄청나게 강했다", "정말 아름다웠다" | 부사 대신 구체적 묘사로 |
| 과소비 클리셰 | "차원이 달랐다", "압도적이었다", "전율했다" | 행동/반응으로 보여줄 것 |

### 14.6 문체 원칙 (참고)
- 비극적 순간에도 과잉 감정 묘사를 피하고, 절제된 문장으로 여운
- 카이의 기록관 시선: 세계의 경이로움을 "기록해야 하니까" 관찰하되, 감정이 새어나오게
- 위트와 유머는 억지스럽지 않게, 상황 속에서 자연스럽게 발생

### 14.7 회피 패턴 (severity: critical)
| 패턴 | 문제 | 해결 |
|------|------|------|
| 설명 덤프 | 세계관 정보를 한꺼번에 쏟아내기 | 2~3문장 설명 → 캐릭터 반응/대화 1문장으로 끊기 |
| Show 대신 Tell | "그는 슬펐다" 직접 서술 | 신체 반응(목이 메었다)으로 보여주기 |
| 전투 호흡 단조 | 같은 길이 문장 반복 | 의도적 문장 길이 변주 |
| 클리프행어 남용 | 매 화 생사 갈림길 | 강도 조절. 대형 3~4화에 1번 |
| 긴장 후 이완 부재 | 전투 연속 10화 | 전투 2~3화 후 이완 1~2화 필수 |
| 캐릭터 구분 불가 | 대사에서 화자 구분 안 됨 | 고유 말투 + 태그 최소화 |
| 의미 없는 분량 늘이기 | 스토리 비기여 반복 묘사 | 모든 문장이 스토리 진행/캐릭터 부각/긴장 고조 중 하나에 기여 |
| 느린 도입부 | 3화까지 핵심 사건 없음 | 1화 안에 욕망 + 즉시적 위기 동시 제시 |
| 복선 없는 각성 | 사전 힌트 없이 갑자기 강해짐 | 각성 전 3~5화에 걸쳐 실패/한계/힌트 배치 |
| 고구마 과적재 | 10화 연속 갈등만 쌓임 | 3~5화 내 사이다 1회 배치 |
| 설정 오류/모순 | 앞 설정과 뒤 사용 불일치 | 세계관 설정 문서 교차 검증 |
| 동료 인형화 | 파티원이 주인공 시중만 | 독립적 동기/갈등/성장 부여 |
| 설명 대사 | 캐릭터가 독자 위해 부자연스러운 설명 | 서술자가 하거나 대화 흐름에 녹임 |
| 고유명사 과잉 | 1화에 10개 이상 고유명사 | 1화 3~5개 이하, 맥락으로 추론 가능하게 |

## 16. 퇴고 체크리스트

### 16.1 기본 체크
- [ ] 분량 7,000~9,000자 확인
- [ ] 종결어미 3회 연속 체크 → 연속 5문장 내 3종 이상 변형
- [ ] 느낌표 5개 이내
- [ ] 감탄 부사 2회 이내
- [ ] 의성어/의태어 5개 이내 (전투 시 7개)
- [ ] 금지 용어(세계관설정집 부록B) 미사용 확인

### 16.2 문체 체크
- [ ] 번역투 검출 (피동, ~에 의해, ~것이다)
- [ ] 접속사 남용 검출 (그러나/하지만 3연속 없는지)
- [ ] 주어 혼란 검출 (2인 이상 장면에서 2~3문장마다 주어 명시)
- [ ] 동일 표현 반복 3회 이내

### 16.3 구조 체크
- [ ] 오프닝 훅: 행동/감각/위기로 시작하는가
- [ ] 중반 전환점 존재하는가
- [ ] 클로징 훅: 미해결 질문 또는 새 사건 암시
- [ ] 장면 전환 3회 이내
- [ ] 회고 장치 2회 이내

### 16.4 Show vs Tell 체크
- [ ] "그는 화가 났다" → "주먹이 떨렸다" 등 신체 반응 변환
- [ ] 설명체 문장 없는지 ("~라는 것은 ~이다")
- [ ] 세계관 설명 1화 2건 이내, 각 3문장 이내

### 16.5 캐릭터 체크
- [ ] 캐릭터별 말투 일관성
- [ ] 레이 능동적 감정 표현 존재 (거부, 고집 등)
- [ ] "기록관이니까" 5화에 1~2회 수준
- [ ] 내면 독백 과잉 아닌지 → 행동으로 보여주기

## 17. 대구법과 밀도 원칙

### 17.1 대구법 (Echo)
시작과 끝에서 같은 이미지/문장을 반복하되 의미를 변화시킨다.
- 화: 첫 줄 ↔ 마지막 줄
- 아크: 첫 화 ↔ 마지막 화
- 대막: 프롤로그 ↔ 에필로그

### 17.2 밀도 경사
단위가 커질수록 핵심 표현의 밀도가 높아져야 한다.
- 1화 클라이맥스 < 아크 클라이맥스 < 대막 클라이맥스
- 가장 아름다운 문장, 가장 강렬한 비유는 대막 클라이맥스에 아껴둔다

### 17.3 감정 누적
큰 단위의 감정적 정점은 작은 단위들의 감정 누적 위에서만 작동한다.
- 누적 없이 갑자기 감동적인 장면 = '억지 감동'
- 대막 클라이맥스의 결집 장면이 감동적이려면, 아크마다 관계 성장이 쌓여야

## 18. 1~10화 분석 피드백

### 강점 (유지)
1. 차별 구조의 체화 — 일상에서 계층 차별이 구체적으로 체감됨
2. "기록"의 존재론적 의미 — 소설 전체 관통 주제
3. 세계관의 감각적 전달 — 설명 덤프 없이 일상으로 보여줌

### 약점 (개선)
1. **레이 능동성 부재** → "싫다", 거부, 고집 등 능동적 감정 부여
2. **카이 감정 폭발 없음** → 최소 1회는 터뜨려야 카타르시스
3. **"기록관이니까" 과다** → 5화에 1~2회 수준으로 제한
4. **내면 독백 과잉** → 행동으로 보여주기 (펜 꺾기, 글씨 삐뚤어짐 등)

## 20. 단행본 전환 규칙

### 20.1 기본 전환
| 항목 | 웹소설 → 단행본 |
|------|----------------|
| 줄바꿈 | 문장 단위 → 문단 단위 재구성 |
| 회고 장치 | 권 시작/끝에 추가 |
| 훅 | 매 화 절단 → 자연스러운 장 전환 (권 끝 훅은 강화) |
| 묘사 | 핵심 장면에 감각 묘사 2~3문장 추가 |
| 후기 | 각 권 끝 추가 (웹소설은 시즌 종료 시만) |

### 20.2 구조 조정
| 항목 | 연재본 | 단행본 |
|------|--------|--------|
| 회차 경계 | 매 화 5,500~9,000자 + 훅 | 3~7화를 1장으로 묶기. 장 분리 소제목 부여 |
| 이전 화 요약 | 각 화 도입부에 상기 문장 | 중복 요약 삭제. 최소 연결만 유지 |
| 분량 패딩 | 분량 맞추기용 부연 | 서사 비기여 문장 전수 삭제 |
| 프롤로그/에필로그 | 없거나 짧음 | 1권 프롤로그 추가 + 각 권 에필로그 추가 |
| 클리프행어 처리 | "그때였다.(끝)" + "문이 열렸다.(다음화)" | → "그때, 문이 열렸다." (연결) |

### 20.3 권 분할 기준
- 이북: 110,000~140,000자, 20~25화
- 종이책: 120,000~150,000자, 270~330p
- 반드시 아크 경계에서 끊기
- 권별 분량 편차 ±10% 이내 (단, 대막 경계 권은 예외 허용 — 19권, 25권, 30권 등 대막 전환부는 8~12화도 가능)

### 20.4 교정 3단계
1. **1교 (기본 교정)**: 오탈자, 비문, 주술 불일치, 맞춤법
2. **2교 (작가 수정)**: 부사 과잉→구체 묘사, Tell→Show 전환, 반복 표현 변주
3. **3교 (최종 검수)**: 전체 통독, 설정 일관성, 톤 일관성, 캐릭터 말투

### 20.5 전환 시 흔한 실수
- 클리프행어 방치 → 자연스러운 장면 전환으로 교체
- 분량 조정 없이 단순 합치기 → 권별 ±10% 조정
- 교정 단계에서 설정 오류 발견 → 전환 전 설정 시트 대조 먼저
- 분량 패딩 문장 방치 → 전수 검토 후 삭제

## 22. 캐릭터 시트 — 파티 '노이즈'

### 22.1 카이 — 서술자 / 크로마 스크라이브

> "나는 기록관이지, 모험가가 아니다."

| 항목 | 내용 |
|------|------|
| 종족 | 렌 (RGB) |
| 크로마 값 | R:52 / G:45 / B:58 — 총합 155 (탄사이 하위) |
| 지배색 | B 미세 우세. 홍채는 칙칙한 보랏빛 잿빛(#342d3a). 빛나지 않음 |
| 이로몬 | 잔필기체처럼 가느다란 문양. 발동해도 희미한 잿빛 |
| 수명 | 탄사이 기준 약 90~100년. 소설 시작 시점 28세 |
| 직업 | 크로마 스크라이브 견습. 렌의 크로마 값을 측정/기록/공증하는 서기관 |

**소지품**: 서첩(기록용) + 측정 도구 + **코덱스**(기록관 정식 자격자 전용 참고서. 세계 각지의 크로마 데이터/역사/종족/지리 정보가 집성된 서적. 스승의 유품으로 비공식 소지. 견습인 카이가 보유한 사실이 알려지면 안 되는 비밀. 사용 시 주변을 경계하며 은밀히 펼침).

**성격**: 표면은 자기비하적 유머와 투덜거림. 내면은 강한 직업 윤리. 거짓 기록을 남기느니 위험을 무릅쓰는 고집.

**결핍**: 자기 가치에 대한 만성적 회의. 파티에서 점차 '기록하는 자만이 볼 수 있는 것'의 가치를 발견.

**파티 역할**: 직접 전투력 거의 없음. '크로마 판독'으로 대상의 크로마 상태를 정밀 수치로 읽어 코다에게 핵심 정보 제공. 최종 기술 '기록의 잔영': 과거에 기록한 크로마 패턴을 일시적으로 재현. B:58은 범용 인지 능력으로는 하위이나, '기록'이라는 극도로 좁은 영역에 사이도 전체를 집중한 결과 **일반적인 B 200대 능력자도 도달하지 못하는 특수 발현**에 이른다. 이것은 사이도(잠재력 발휘도)가 총합이나 채널값보다 중요하다는 **세계관 핵심 설정의 체현**.

**성장 아크**: 대막 1~2: 수동적 관찰자 → 대막 3: '패턴을 보는 자' 각성 → 대막 4: 빛을 잃는 체험 + 기록의 잔영 전조 → 대막 5~6: 파티의 정서적 닻 → 대막 7: 유일하게 원래 형태로 남은 자. 이 이야기를 쓰는 사람.

### 메타 프레임: 지구 전달 경위

카이가 크로마 코덱스 원전을 완성한 뒤, 기록의 잔영 능력으로 서첩에 차원 관통 속성이 부여됨. 카이는 반의도적으로 — "이 기록이 도달해야 할 곳에 도달하기를" 바라며 — 서첩을 보더에 위탁. 서첩은 보더를 넘어 다른 세계(=지구)에 도달.

- **프롤로그**: 익명의 번역자 서문. "나는 이 서첩을 발견한 사람입니다." 번역 경위를 짧게 설명. 독자에게 "다른 세계의 기록"이라는 프레임 설정.
- **에필로그**: 카이가 서첩을 보더에 위탁하는 장면 + 번역자 후기. 전체 경위 공개.
- **번역자**: 익명. 이름/배경 불명. "나는 이 서첩을 발견한 사람입니다"로만 자기소개.
- **전달 방향**: 카이→지구 (단방향). 카이는 서첩이 어디로 가는지 모름.
- **도움 요청 장치**: 카이의 기록에 "이 후의 세계"에 대한 경고/요청이 포함됨. 세부는 대막7 서사에서 확정.
