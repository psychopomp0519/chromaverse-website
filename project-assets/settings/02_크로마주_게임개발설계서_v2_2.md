<a id="수익-모델"></a>### 1\.2 수익 모델

- 절대 금지: 전투력 직접 판매\(P2W\)\. 과금이 전투 수치에 영향 없음\.
- 외형 과금: 이로몬 스킨, 의상, 탈것 외형, 하우징 가구, 이펙트 변경\.
- 편의 과금: 창고 확장, 인벤 슬롯, 빠른 이동권, 사이도 경험 부스터\(상한 변경 아님\)\.
- 인게임 대체: 모든 과금 아이템은 고순도 레스로도 구매 가능\.
- 가차: 천장 80회, 중복 시 색채 파편으로 원하는 아이템 교환\. 검열판은 선택형\.

<a id="캐릭터-생성-및-성장"></a>## 2\. 캐릭터 생성 및 성장

<a id="생성-흐름"></a>### 2\.1 생성 흐름

- 
	1. 색채 부여: 랜덤 RGB \(총합 350~480, 개별 40~220\)\. 리롤 3회\.
	2. 체계 선택: RGB\(빛\) 또는 CMYK\(어둠\)\. CMYK 선택 시 R→C, G→M, B→Y 변환\. K=\(765\-총합\)/5\. 이에 따라 게임 내 CMYK 캐릭터의 K값 범위는 57\(총합 480\)~83\(총합 350\)이다\. 수치 범위가 좁지만 공명 증폭 등 특수 상황에서 K값 1포인트당 약 0\.5%의 배율 차이가 누적되어 의미 있는 격차를 만든다\.
	3. 초기 활동: 튜토리얼\. 사이도 10%→25%\. 생활직업\+전투직업 자유 병행\.

<a id="파티-46인"></a>### 6\.1 파티 \(4~6인\)

- 던전 4단계: 일반/정예/영웅/전설\. 색채 상성 기믹 \+ 페이즈 전환 \+ 전멸기\.
- 코드 공명 보너스: 상보 공명\(R\+G\+B 균형\)=방어\+10%, 단색 공명\(동채널 3\+\)=데미지\+20%\.

<a id="레이드-1224인"></a>### 6\.2 레이드 \(12~24인\)

- 4유형: 소규모\(12인\)/표준\(18인\)/대규모\(24인\)/세계\(24인\+서버지원\)\.
- 레조넌스 체인: 다수가 특정 순서로 스킬 사용 시 폭발적 추가효과\.
- 보스 등급: 정예 → 지배자 → 고대자 → 세계급\.

<a id="공성전-2050-vs-2050"></a>### 6\.3 공성전 \(20~50 vs 20~50\)

- 코드 단위 거점 쟁탈\. 준비 10분 \+ 전투 40분\.
- 3단계: 외벽 돌파 → 내부 침투 → 크로마코어\(색채 핵\) 공략\.
- 크로마코어: 3중 보호막\(방어측 색채 합산 유지\)\. 방어측 전사자 증가 시 보호막 감소\.
- 인원 차이 보정: 5명당 소수측 \+8%, 최대 \+40%\.

<a id="월드-구조"></a>## 12\. 월드 구조

- 탐험 포인트: 색채 퍼즐, 숨겨진 동굴, 산란의 기억 수집\.
- 환경 상호작용: 아도르로 얼음 녹이기, 비탈리스로 식물 성장, 코그니스로 숨겨진 경로 감지\.
- 날씨/시간: 게임 내 1일=현실 2시간\. 밤: 보이드 \+10%, 은신 탐지 \-30%\.

<a id="반전-선택-캐릭터-데이터-이전"></a>### 13\.1 반전 선택 캐릭터 데이터 이전

서막에서 반전 스토리를 선택한 캐릭터의 경우:

- RGB→CMYK\(또는 역방향\) 전환 데이터가 대규모 버전에 그대로 이전된다\.
- 서막에서 해금 불가능한 히든 직업\(크로마블렌드, 인버터 등\)의 전제 조건\(반전 경험\)은 대규모 버전에서 인정된다\.
- 서막에서 획득한 반전 관련 퀘스트 진행도, 균형파 연구소 방문 기록 등도 이전된다\.

<a id="세계관-용어의-게임-내-노출-지침"></a>### 13\.2 세계관 용어의 게임 내 노출 지침

세계관설정집의 핵심 용어 중 게임 시점\(프리즈마력 1500년\)에서 ’전설’이나 ’학술 논쟁’으로 존재하는 개념들은 NPC 대사, 고서적 아이템 텍스트, 퀘스트 지문에서 단정적이지 않은 톤으로 노출한다\. ’잠재 현현’은 균형파 학자 NPC가 드물게 언급하는 가설로, ’산포’는 코덱스 원전 단편에서 발견되는 고대 용어로, ’네 명의 선구자’는 진위 논쟁이 있는 전설로 취급한다\. 플레이어가 히든 퀘스트를 진행하면서 이 전설들의 신빙성을 확인해 나가는 구조가 이상적이다\.
