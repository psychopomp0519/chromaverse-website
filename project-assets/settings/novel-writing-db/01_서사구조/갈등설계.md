# 갈등 설계

> 서사의 엔진. 갈등 없는 이야기는 엔진 없는 차와 같다.
> 갈등의 유형, 층위, 에스컬레이션, 해소 기법을 총정리.

---

## 1. 갈등의 7가지 유형

### 1.1 인간 vs 인간 (Man vs Man)
- **가장 보편적**이고 직관적인 갈등
- 물리적 대립(전투), 심리적 대립(권모술수), 이념적 대립(가치관 충돌)
- **[인기]**: 악역에게도 이해 가능한 동기 부여 → 갈등의 깊이 증가
- **[비인기]**: 악역이 단순한 악 → "왜 저러는 거야" 납득 불가

### 1.2 인간 vs 자연 (Man vs Nature)
- 재해, 생존, 탐험 서사의 핵심
- 자연에 의지를 부여하지 않으되, 압도적 위협으로 설정
- **[팁]**: 자연 속에서 인간의 한계와 의지를 대비시킬 때 가장 강력

### 1.3 인간 vs 사회 (Man vs Society)
- 개인 vs 체제/관습/법/여론
- 사회 비판, 혁명, 차별 극복 서사
- **[인기]**: 독자가 "나도 저렇게 느낀 적 있다"고 공감할 때 폭발적 몰입
- **[비인기]**: 사회 문제를 단순화/희화화 → 설교적/피상적

### 1.4 인간 vs 자기 자신 (Man vs Self)
- 내면의 두려움, 트라우마, 도덕적 딜레마
- **모든 좋은 서사에 최소 하나는 존재해야 하는 갈등**
- **[인기]**: 외적 갈등의 해결이 내적 갈등의 극복과 동기화
- **[비인기]**: 내적 갈등 없이 외적 문제만 해결 → 감동 없음

### 1.5 인간 vs 기술/문명 (Man vs Technology)
- SF, 디스토피아, 사이버펑크의 핵심
- AI, 감시 사회, 기술 의존, 인간성 상실
- **[팁]**: 기술 자체보다 기술이 인간 관계에 미치는 영향에 초점

### 1.6 인간 vs 운명/신 (Man vs Fate)
- 예언, 저주, 신의 의지에 맞서는 서사
- 비극(운명에 패배)과 영웅담(운명 극복) 양쪽 가능
- **[주의]**: 운명이 너무 강력하면 독자가 무력감 → 주인공의 "선택"이 의미를 가져야 함

### 1.7 인간 vs 미지 (Man vs Unknown)
- 호러, 코스믹 호러, 미스터리의 핵심
- 이해할 수 없는 것에 대한 공포
- **[인기]**: 미지가 서서히 윤곽을 드러내되 완전히 설명되지 않을 때
- **[비인기]**: 미지가 결국 별것 아닌 것으로 밝혀짐 → 허탈감

## 2. 갈등의 층위 설계

### 2.1 3층 갈등 모델 (모든 좋은 작품의 공통점)

```
표층(External): 눈에 보이는 갈등 — 전투, 추격, 경쟁, 사건
  ↕ 연동
중층(Interpersonal): 인물 간 갈등 — 신뢰, 배신, 오해, 경쟁
  ↕ 연동
심층(Internal): 내면 갈등 — 두려움, 정체성, 도덕적 딜레마
```

**[필수]** 3층이 모두 존재하고 서로 연동될 때 최고의 서사 밀도

**예시 — 인기 패턴**:
| 표층 | 중층 | 심층 |
|------|------|------|
| 마왕을 물리쳐야 함 | 동료가 배신자일 수 있음 | "나는 영웅이 될 자격이 있는가" |
| 회사에서 프로젝트 성공 | 상사/동료와 갈등 | "성공이 내 인생의 전부인가" |
| 살인범을 찾아야 함 | 용의자가 소중한 사람 | "진실이 모두를 행복하게 하는가" |

**예시 — 비인기 패턴**:
- 표층만 존재: "몬스터 잡고 레벨업" (반복, 지루)
- 심층만 존재: "주인공이 계속 고민만 함" (답답, 진전 없음)

---

### 2.2 갈등 연동의 핵심 원칙

**원칙 1: 외적 갈등의 해결이 내적 갈등을 해결하게 만들어라**
- 마왕을 이기는 것이 "나는 영웅인가"의 답이 되는 구조
- 사건 해결이 곧 성장 완성

**원칙 2: 내적 갈등이 외적 갈등의 장애물이 되게 하라**
- 두려움 때문에 결정적 순간에 주저 → 위기 확대
- 내면을 극복해야 외부를 해결할 수 있는 구조

**원칙 3: 중층 갈등으로 양쪽을 연결하라**
- 동료와의 갈등이 전투(표층)에서 약점이 되고, 신뢰 문제(심층)를 부각

## 3. 갈등 에스컬레이션

### 3.1 에스컬레이션 5단계 모델

```
1단계: 불편 (Discomfort)
   → "뭔가 이상하다" — 작은 균열, 미세한 변화
2단계: 사건 (Incident)
   → "문제가 터졌다" — 무시할 수 없는 사건 발생
3단계: 오해/확대 (Misunderstanding/Escalation)
   → "상황이 꼬인다" — 해결 시도가 오히려 악화
4단계: 위기 (Crisis)
   → "돌이킬 수 없다" — 관계/상황이 파국 직전
5단계: 폭발/해소 (Explosion/Resolution)
   → "결정적 순간" — 대결/고백/진실/선택
```

**[인기]** 핵심: 각 단계 사이에 "잠깐의 안정"을 배치 → 다음 에스컬레이션의 충격 극대화

**[비인기]** 실패:
- 1→5 급점프: "갑자기 다 터짐" → 감정 준비 부족
- 1→2→1→2 반복: "계속 같은 수준" → 지루함
- 5단계 도달 후 꺼짐: "기대만큼 안 터짐" → 허탈

---

### 3.2 에스컬레이션 기법

**스테이크(Stakes) 상승법**:
| 단계 | 걸린 것 | 예시 |
|------|---------|------|
| 개인적 | 자존심, 체면 | 시합에서 지면 창피 |
| 관계적 | 우정, 사랑, 신뢰 | 지면 동료가 위험 |
| 생존적 | 생명, 안전 | 지면 죽는다 |
| 사회적 | 공동체, 국가 | 지면 마을이 멸망 |
| 존재적 | 세계, 존재 의미 | 지면 세계가 끝난다 |

**[필수]** 스테이크는 반드시 우상향 — 한번 올린 스테이크를 내리면 긴장 붕괴

**타임 프레셔(시간 압박)**:
- "3일 안에 해독제를 구해야 한다"
- 시간 제한은 가장 효율적인 에스컬레이션 장치
- **[주의]**: 시간 제한을 설정하고 무시하면 신뢰 상실

**딜레마(이중 구속)**:
- "A를 구하면 B가 죽는다"
- 정답이 없는 선택 → 캐릭터의 가치관 노출
- **[인기]**: 독자도 고민하게 만드는 딜레마가 최고

## 4. 갈등 해소 패턴

### 4.1 만족스러운 해소의 조건

**[필수]** 3요소:
1. **납득 가능성**: 해결 방법이 이야기 안에서 논리적
2. **감정적 보상**: 해결 과정에서 카타르시스 발생
3. **변화 확인**: 해결 후 인물/세계가 이전과 달라짐

### 4.2 해소 유형

| 유형 | 설명 | 적합한 장르 |
|------|------|------------|
| 승리형 | 주인공이 적/문제를 극복 | 액션, 성장물, 영웅서사 |
| 타협형 | 양측이 일부씩 양보 | 현실물, 성인 드라마 |
| 희생형 | 무언가를 대가로 치름 | 비극, 감동물, 전쟁물 |
| 인식 전환형 | 문제 자체의 의미가 바뀜 | 문학, 성장물, 철학적 서사 |
| 미해결형 | 갈등이 남아 여운 | 문학, 리얼리즘, 시리즈 다음 권 |
| 반전형 | 전제가 뒤집힘 | 미스터리, 스릴러 |

### 4.3 해소의 금기

**[금기]** 데우스 엑스 마키나:
- 갑자기 등장한 존재/능력이 모든 것을 해결
- **예외**: 의도적 패러디이거나, 그 "갑작스러운 것"이 사전에 복선으로 깔려 있었을 때

**[금기]** 무력화 해소:
- "갈등이 사실 오해였어요" → 70%의 서사가 무의미해짐
- 오해 기반 갈등은 해소 시 새로운 갈등으로 전환해야 함

**[금기]** 설교형 해소:
- 캐릭터가 갑자기 깨달음을 얻어 장문의 연설 → "갑자기?"
- 깨달음은 행동으로 보여주고, 말은 최소화

## 5. 장르별 주력 갈등

| 장르 | 주력 갈등 유형 | 보조 갈등 |
|------|--------------|----------|
| 판타지 | 인간 vs 인간/운명 | 자기 자신, 사회 |
| 로맨스 | 인간 vs 인간 (연애) | 자기 자신, 사회 |
| 액션/무협 | 인간 vs 인간 (전투) | 자기 자신, 운명 |
| 미스터리 | 인간 vs 미지 | 인간, 자기 자신 |
| 호러 | 인간 vs 미지/자연 | 자기 자신 |
| SF | 인간 vs 기술/사회 | 자기 자신, 미지 |
| 일상물 | 인간 vs 사회/자기 자신 | 인간 |
| 생존물 | 인간 vs 자연 | 인간, 자기 자신 |

## 6. 갈등 설계 체크리스트

- [ ] 표층/중층/심층 갈등이 모두 존재하는가?
- [ ] 세 층위가 서로 연동되는가?
- [ ] 갈등의 스테이크가 점진적으로 상승하는가?
- [ ] 주인공이 갈등의 해결에 능동적으로 참여하는가? (수동적 관찰자가 아닌가?)
- [ ] 갈등 해소가 사전에 깔린 단서/복선으로 납득 가능한가?
- [ ] 해소 후 인물/세계에 의미 있는 변화가 발생하는가?
- [ ] 독자가 갈등의 양쪽 입장을 이해할 수 있는가?
- [ ] 메인 갈등 외에 서브 갈등이 서사를 풍부하게 하는가?
