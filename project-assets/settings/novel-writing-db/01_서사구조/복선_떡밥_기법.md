# 복선/떡밥 기법

> 독자에게 "그때 그거였어!" 순간을 선사하는 기술.
> 복선 설치/회수, 떡밥 운용, 미스디렉션, 체호프의 총 활용법.

---

## 1. 복선과 떡밥의 정의

| 용어 | 정의 | 목적 |
|------|------|------|
| **복선(伏線)** | 미래 사건의 단서를 사전에 배치 | 사건 발생 시 "납득감" 제공 |
| **떡밥** | 독자의 궁금증을 유발하는 정보 조각 | "다음 회차를 클릭하게" 만듦 |
| **미스디렉션** | 독자의 시선을 의도적으로 다른 곳으로 유도 | 반전의 충격 극대화 |
| **체호프의 총** | "1막에 나온 총은 3막에서 발사되어야 한다" | 설정의 경제성, 약속 이행 |

**복선 vs 떡밥의 차이**:
- 복선: 독자가 읽을 때는 모름 → 나중에 "아!" (소급적 인지)
- 떡밥: 독자가 읽을 때 궁금 → 나중에 "그거!" (전향적 기대)
- **최고의 장치**: 복선이자 떡밥 — 궁금증도 유발하고, 나중에 의미도 달라지는 것

## 2. 복선 설치 기법

### 2.1 설치 방법 분류

**A. 직접 언급형** (가장 단순)
- 캐릭터가 직접 말하거나, 서술자가 언급
- 위험: 너무 노골적이면 예측 가능 → 미스디렉션과 병행 필요
```
예: "이 검은 아무도 뽑지 못했다" → 나중에 주인공이 뽑음
```

**B. 배경 녹임형** (가장 효과적)
- 풍경/환경/소품 묘사 속에 자연스럽게 삽입
- 독자가 "그냥 분위기 묘사구나" 하고 넘어감 → 나중에 결정적 역할
```
예: "벽에 걸린 낡은 초상화" → 100화 후 그 인물이 핵심
```

**C. 행동/습관형**
- 캐릭터의 반복 행동이나 습관이 나중에 의미를 가짐
```
예: "그는 항상 왼손을 주머니에 넣고 있었다" → 숨기고 있던 것의 복선
```

**D. 대사 삽입형**
- 일상적 대화 속 한 문장이 나중에 핵심 단서
```
예: "할아버지가 늘 말씀하셨지, '바다 밑에는 길이 있다'고" → 해저 통로 복선
```

**E. 감각/감정형**
- 설명 불가능한 감각이나 감정이 나중에 능력/진실의 복선
```
예: "이유 없이 그 방에 들어가면 머리가 아팠다" → 봉인된 마법의 복선
```

**F. 구조적 복선**
- 이야기의 구조 자체가 복선 (프레임 서사의 현재 시점이 미래 사건 암시)
```
예: "그때의 나는 이것이 마지막 평화로운 아침이 될 줄 몰랐다"
```

---

### 2.2 복선 설치의 황금 비율

**[인기]** 인기작의 복선 밀도:
- 1회차당 복선/떡밥 설치: 1~3개
- 회수까지의 거리: 근거리(3~10화) 60%, 중거리(11~50화) 30%, 장거리(50화+) 10%
- **근거리 복선이 독자 만족의 핵심** — 장거리만 쌓으면 "떡밥만 던지고 안 풀어줌"

**[비인기]** 실패 패턴:
- 복선 과다: 매 장면이 복선 → 독자가 피로, "뭐가 중요한 거야"
- 복선 부재: 사건이 갑자기 발생 → "복선 없이 반전" = 억지 전개
- 설치 후 망각: 복선을 깔고 회수 안 함 → 설정 구멍으로 인식

---

### 2.3 복선의 3단계 설치법 (장거리 복선)

```
1차 설치 (씨앗): 자연스럽게, 독자 인식 최소화
   → "그 마을의 우물은 유난히 깊었다"

2차 강화 (물주기): 다른 맥락에서 한 번 더 언급
   → "옛 기록에 이 지역의 지하수맥이 특이하다는 내용이"

3차 회수 (개화): 결정적 순간에 의미 폭발
   → "우물이 깊었던 이유는 — 지하에 봉인된 것이 있었기 때문"
```

**[인기]** 핵심: 2차 강화가 있어야 회수 시 "아 그거!" 반응 발생. 1차만 깔면 너무 오래돼서 잊힘.

## 3. 떡밥 운용 기법

### 3.1 떡밥 유형

| 유형 | 설명 | 효과 |
|------|------|------|
| 질문형 | "그는 왜 그날 웃었을까?" | 호기심 유발 |
| 비밀형 | "아직 아무도 모르는 진실이 있다" | 기대감 형성 |
| 모순형 | 캐릭터 행동/설정에 앞뒤가 안 맞음 | 추리 욕구 자극 |
| 예언형 | "세 번째 달이 뜨면 ~" | 시간적 기대감 |
| 아이템형 | 정체불명의 물건/능력 등장 | "이거 뭐지?" |
| 인물형 | 미등장 인물에 대한 언급 | "누구야?" |

### 3.2 떡밥의 유통기한

**[필수]** 떡밥에는 유통기한이 있다:

| 거리 | 유통기한 | 관리법 |
|------|---------|--------|
| 근거리 | 3~10화 | 던지면 빠르게 회수, 독자 만족의 주력 |
| 중거리 | 11~50화 | 중간에 1~2회 리마인드 필수 |
| 장거리 | 50화+ | 정기적 리마인드(50화마다) + 부분 회수 |

**리마인드 기법**:
- 캐릭터가 해당 사건을 다시 언급
- 유사한 상황 발생 → 독자가 자연스럽게 연상
- 새로운 단서 추가 → 떡밥을 "업데이트"

**[비인기]** 떡밥 방치의 3대 증상:
1. 독자가 떡밥의 존재를 잊음 → 회수해도 감흥 없음
2. 떡밥이 너무 많이 쌓임 → "대체 뭐가 뭔지 모르겠다"
3. 회수가 기대 이하 → "이게 다야?" 반응

---

### 3.3 떡밥 회수의 만족도 공식

**만족도 = (궁금증 크기 × 회수 충격) ÷ 대기 시간**

- 궁금증 크기: 떡밥이 독자에게 얼마나 궁금했는가
- 회수 충격: 답이 얼마나 예상 밖이면서도 납득 가능한가
- 대기 시간: 너무 길면 만족도 하락 (유통기한 초과)

**[인기]** 최고의 회수:
- 예상 가능한 답: 20% (독자가 맞춤 → 성취감)
- 예상 밖이지만 납득 가능한 답: 70% (놀라움 + 감탄)
- 완전히 예상 밖인 답: 10% (충격, 단 과하면 억지)

## 4. 미스디렉션 기법

### 4.1 미스디렉션 유형

**A. 레드 헤링 (Red Herring)**
- 가짜 단서를 진짜처럼 배치
- 독자가 잘못된 추리를 하게 유도
- 진짜 답이 밝혀질 때 "아, 속았다!" 쾌감

**B. 시선 유도 (Misdirection of Attention)**
- 중요한 정보를 중요하지 않은 것 사이에 끼워넣음
- 독자의 주의가 다른 곳에 있을 때 핵심 정보 슬쩍 공개
- "나중에 다시 읽으면 다 나와 있었어!" 효과

**C. 신뢰할 수 없는 화자**
- 서술자가 의도적/비의도적으로 정보를 왜곡
- 독자가 서술자를 신뢰했다가 진실 발견 시 충격
- **[주의]**: 서술자의 왜곡에도 "규칙"이 있어야 함 (거짓말을 하되 일관된 방식으로)

**D. 장르적 기대 역이용**
- 독자가 장르 문법상 기대하는 전개를 이용
- "분명 이 캐릭터가 악역이겠지" → 아님
- "주인공이 이기겠지" → 패배 → 다른 방식으로 극복

### 4.2 미스디렉션의 윤리

**[필수]** 공정한 미스디렉션의 조건:
1. 진짜 단서도 함께 존재해야 함 (독자에게 맞출 기회 제공)
2. 밝혀진 후 다시 읽으면 모든 단서가 보여야 함
3. 미스디렉션이 캐릭터 행동의 논리를 깨면 안 됨

**[금기]** 불공정한 미스디렉션:
- 독자에게 거짓 정보를 "서술 차원에서" 제공 (화자가 아닌 서술 자체가 거짓)
- 핵심 정보를 의도적으로 숨기고 "반전이다" 주장
- 모든 단서가 미스디렉션이고 진짜 단서가 없음

## 5. 체호프의 총 — 설정 경제학

### 5.1 원칙
> "1막에 벽에 총이 걸려 있다면, 3막에서 그것은 발사되어야 한다."

**확장 해석**:
- 의미 있게 언급된 모든 것은 서사에서 역할을 해야 한다
- 역할 없이 언급된 것은 삭제하거나 역할을 부여해야 한다
- 예외: 세계관 분위기 조성용 묘사 (그러나 이것도 반복 언급되면 의미 부여 필요)

### 5.2 활용법

**정방향**: 설정을 먼저 배치 → 나중에 활용
```
1화: 주인공이 할아버지의 오래된 시계를 물려받음
50화: 그 시계가 실은 시간을 멈추는 기능이 있었음
```

**역방향**: 필요한 장치를 먼저 결정 → 사전에 자연스럽게 배치
```
50화에서 시간 정지 기능이 필요 → 1화에 시계 등장시킴 (역설계)
```

**[인기]** 베스트 프랙티스:
- 모든 주요 반전/해결의 장치는 최소 30% 이전에 설치
- 설치 시점에는 "당연한 것"처럼 보이게
- 활용 시점에는 "아 그거!" 반응이 나오게

## 6. 복선/떡밥 관리 시스템

### 6.1 추적표 템플릿

```markdown
| ID | 유형 | 내용 | 설치 화 | 강화 화 | 회수 화 | 상태 | 연결 떡밥 |
|----|------|------|--------|--------|--------|------|----------|
| F-001 | 복선 | 우물의 깊이 | 3화 | 15화, 30화 | 45화 | 회수완료 | F-005 |
| T-001 | 떡밥 | 미지의 인물 언급 | 5화 | — | — | 미회수 | — |
```

### 6.2 관리 원칙

1. **미회수 떡밥 한계**: 동시에 미회수 상태인 떡밥은 5~7개 이하 권장
2. **회수율 목표**: 전체 떡밥의 90% 이상은 완결 전 회수
3. **의도적 미회수**: 속편/스핀오프를 위한 것은 명시적으로 표기
4. **매 10화마다**: 미회수 목록 점검, 리마인드 필요 여부 판단

## 7. 장르별 복선/떡밥 특성

| 장르 | 주력 기법 | 특징 |
|------|----------|------|
| 미스터리 | 단서형 복선 + 레드 헤링 | 공정한 퍼즐이 핵심, 독자와의 두뇌 게임 |
| 판타지 | 세계관형 복선 + 예언형 떡밥 | 세계의 비밀이 점진적으로 공개 |
| 로맨스 | 감정형 복선 + 오해형 떡밥 | 관계의 진전/후퇴가 복선의 대상 |
| 호러 | 분위기형 복선 + 모순형 떡밥 | "뭔가 이상한데" 감각이 복선 |
| 회귀물 | 미래 지식형 복선 | 주인공만 아는 정보가 복선이자 떡밥 |
| 시스템물 | 스킬/아이템형 복선 | 습득한 능력이 나중에 결정적 역할 |
