# 플롯 구조 패턴

> 모든 장르/매체에 적용 가능한 서사 구조의 뼈대.
> 각 구조의 특성, 적합한 장르/매체, 인기작 활용 사례, 비인기작 실패 사례를 정리.

---

## 1. 기본 구조 패턴

### 1.1 3막 구조 (Three-Act Structure)

**구성**: 설정(25%) → 대립(50%) → 해결(25%)

| 막 | 역할 | 핵심 요소 |
|----|------|----------|
| 1막: 설정 | 세계/인물 소개, 일상 제시, 촉발 사건 | 촉발 사건(Inciting Incident), 1차 전환점 |
| 2막: 대립 | 갈등 심화, 시련과 성장, 중간점 반전 | 중간점(Midpoint), 위기 고조, 2차 전환점 |
| 3막: 해결 | 클라이맥스, 갈등 해소, 여운 | 절정(Climax), 해결(Resolution), 에필로그 |

**[인기]** 성공 패턴:
- 1막에서 "일상 → 파괴"의 낙차가 클수록 독자 몰입 강함
- 2막 중간점에서 "게임 체인저" 이벤트로 서사 방향 전환
- 3막 클라이맥스 전 "가장 어두운 순간(Dark Night of the Soul)" 배치

**[비인기]** 실패 패턴:
- 1막이 30% 이상 — 설정 과잉, 독자 이탈
- 2막 사기(Sagging Middle) — 중간점 없이 늘어지는 전개
- 3막이 급격히 짧아짐 — "용두사미" 느낌

**적합**: 장편소설, 단행본, 영화 각본, 웹소설 시즌제

---

### 1.2 5막 구조 (프라이타크 피라미드)

**구성**: 발단 → 상승 → 절정 → 하강 → 결말

| 막 | 기능 | 감정 곡선 |
|----|------|----------|
| 발단(Exposition) | 배경/인물 설정 | 기대감 형성 |
| 상승(Rising Action) | 갈등 도입, 복잡도 증가 | 긴장 상승 |
| 절정(Climax) | 최고 긴장, 결정적 대립 | 정점 |
| 하강(Falling Action) | 결과 전개, 실마리 정리 | 이완 시작 |
| 결말(Denouement) | 모든 갈등 해소, 새 균형 | 만족/여운 |

**[인기]** 성공 패턴:
- 상승 구간에 "미니 절정"을 2~3개 배치 → 텐션 유지
- 절정에서 "예상 가능하되 예상 못한 방식"으로 해결
- 하강이 지루하지 않도록 "마지막 반전" 또는 "감정적 정리" 삽입

**적합**: 정통 문학, 단편소설, 연극, 비극 서사

---

### 1.3 기승전결 (起承轉結)

**구성**: 기(도입) → 승(전개) → 전(전환) → 결(마무리)

| 단계 | 역할 | 비율 |
|------|------|------|
| 기(起) | 상황 제시, 문제 도입 | 10~15% |
| 승(承) | 상황 발전, 심화 | 30~40% |
| 전(轉) | 예상 밖 전환, 반전 | 30~40% |
| 결(結) | 결론, 여운 | 10~20% |

**핵심**: '전'에서의 전환이 생명 — 단순 전개가 아닌 시각/상황의 급변

**[인기]** 성공 패턴:
- '전'에서 독자의 전제를 뒤집는 정보 공개
- '결'이 짧되 강렬한 여운 — "마지막 한 문장의 힘"

**[비인기]** 실패 패턴:
- '전'이 없고 '승'이 그대로 '결'로 이어짐 → 밋밋한 서사
- '기'가 너무 길어 독자가 '승'에 도달하기 전 이탈

**적합**: 단편소설, 웹소설 1회차, 에피소드 단위 구성, 동아시아 서사

---

### 1.4 키시쇼텐케츠 (起承転結) — 일본식 변형

**특징**: 갈등 없이도 서사가 성립하는 구조. '전'에서 전혀 다른 요소를 도입.

**예시 구조**:
1. 기: A 상황 제시
2. 승: A 상황 발전
3. 전: 전혀 관련 없어 보이는 B 도입
4. 결: A와 B가 연결되어 새로운 의미 형성

**[인기]** 성공 패턴:
- 일상물, 힐링물에서 갈등 없이 감동 전달
- 에피소드 단위 소품에 최적화

**적합**: 일상물, 힐링, 단편, 4컷 만화적 서사

---

### 1.5 영웅의 여정 (Hero's Journey / 캠벨 모노미스)

**12단계 구조**:

```
일상 세계 → 모험의 부름 → 부름의 거부 → 조력자와의 만남
    → 첫 번째 관문 → 시련/동맹/적 → 동굴 깊은 곳 접근
    → 시련(죽음과 부활) → 보상 → 귀환의 길
    → 부활(최종 시련) → 영약을 가지고 귀환
```

**[인기]** 성공 패턴:
- 판타지/액션의 골든 스탠다드 — 독자가 본능적으로 익숙
- "조력자" 캐릭터가 독자 대리인 역할 → 세계관 자연 설명
- "부름의 거부"가 캐릭터에 인간미 부여

**[비인기]** 실패 패턴:
- 12단계를 기계적으로 따름 → 뻔한 전개
- "시련" 없이 주인공이 쉽게 성장 → 카타르시스 부재
- 귀환이 원래 세계와 단절 → 변화의 의미 없음

**[주의]**: 이 구조를 "체크리스트"가 아닌 "골격"으로 사용해야 함. 단계 생략/변형/순서 변경이 오히려 신선함을 줌.

**적합**: 판타지, 모험, 성장물, 게임 서사

### 1.6 Save the Cat! 비트시트

**15비트 구조** (장편/시리즈 특화):

| 비트 | 위치 | 기능 |
|------|------|------|
| 오프닝 이미지 | 1% | 주제의 시각적 은유 |
| 주제 제시 | 5% | 핵심 질문 던지기 |
| 설정 | 1~10% | 일상, 결핍, 필요 제시 |
| 촉매 | 10% | 모든 것을 바꾸는 사건 |
| 망설임 | 10~20% | 변화에 대한 두려움 |
| 2막 돌입 | 20% | 새로운 세계 진입 |
| B스토리 | 22% | 서브플롯 시작 (주로 관계) |
| 재미와 게임 | 20~50% | 장르의 약속 이행 |
| 중간점 | 50% | 가짜 승리/가짜 패배 |
| 악당 역습 | 50~75% | 상황 악화 |
| 모든 것을 잃다 | 75% | 최저점, 상징적 죽음 |
| 어둠의 밤 | 75~80% | 절망, 내면 대면 |
| 3막 돌입 | 80% | 최종 결의 |
| 피날레 | 80~99% | 최종 대결, 변화 증명 |
| 최종 이미지 | 99~100% | 오프닝과 대비되는 변화된 세계 |

**[인기]** 핵심: "재미와 게임(Fun & Games)" 구간이 장르의 약속을 이행하는 부분 — 판타지면 모험, 로맨스면 밀당, 미스터리면 단서 추적

**적합**: 모든 장르의 장편, 시리즈 각 권의 구성

## 2. 웹소설 특화 구조 패턴

### 2.1 회차 반복 루프 구조

**패턴**: 목표 설정 → 도전 → 위기 → 극복/성장 → 보상 → 새 목표

```
[회차 1~3] 목표 A 설정 + 도전
[회차 4~5] 위기 + 고구마
[회차 6~7] 극복 + 사이다
[회차 8~10] 보상 + 새 목표 B 암시
→ 루프 반복 (스케일 확대)
```

**[인기]** 성공 패턴:
- 루프마다 스케일이 점진적으로 확대 (개인→마을→국가→세계)
- 각 루프에 서브플롯 1개씩 진행 → 다층적 서사
- 보상 구간에서 "다음 루프의 씨앗" 배치

**[비인기]** 실패 패턴:
- 루프 간 스케일 변화 없음 → 반복감, 정체
- 위기의 강도가 루프마다 동일 → 긴장감 하락
- 보상이 항상 "레벨업/아이템" → 단조로움

---

### 2.2 파워 커브 구조 (시스템물/성장물)

**패턴**: 약체 시작 → 단계적 성장 → 벽 → 돌파 → 새 영역

```
레벨 1~10: 생존 단계 (독자 감정이입)
레벨 11~30: 성장 쾌감 (사이다)
레벨 31~50: 정체 + 신규 위협 (텐션 유지)
레벨 51~70: 폭발적 성장 (카타르시스)
레벨 71~90: 상위 존재와 대면 (새 스케일)
레벨 91~100: 최종 도달 (완결)
```

**[인기]** 핵심 원칙:
- "1회 강함 → 2~3회 위기 → 1회 폭발"의 리듬
- 성장이 단순 수치가 아닌 "이해/각오/관계"와 연동
- 각 단계의 "천장"이 다음 단계의 "바닥"과 겹침

---

### 2.3 시간 루프 / 회귀 구조

**기본 패턴**: 실패 → 회귀 → 재도전 (변수 변경) → 점진적 성공

**회귀물 핵심 설계**:

| 요소 | 인기 패턴 | 비인기 패턴 |
|------|----------|------------|
| 회귀 동기 | 절절한 후회/결의 | 모호한 이유, "그냥 돌아옴" |
| 기억 활용 | 선택적, 실수도 함 | 완벽한 미래 기억 → 긴장감 0 |
| 나비효과 | 하나 바꾸면 다른 것이 틀어짐 | 모든 것이 계획대로 |
| 감정 | 회차마다 감정적 대가 존재 | 감정 소모 없이 최적 루트 |
| 주변 인물 | 주인공의 변화를 감지 | NPC처럼 동일 행동 반복 |

---

### 2.4 던전/퀘스트 구조

**패턴**: 의뢰 수주 → 준비 → 진입 → 탐색 → 보스 → 보상/후일담

**[인기]** 성공 요소:
- 던전마다 "테마"가 명확 (얼음, 미로, 시간 등)
- 보스전이 단순 전투가 아닌 "퍼즐 + 전투"
- 후일담에서 캐릭터 관계 발전

## 3. 다선 서사 패턴

### 3.1 교차 시점 (인터위빙)

**구조**: A시점 → B시점 → C시점 → 합류

**[인기]** 핵심:
- 각 시점이 독립적으로도 흥미로움
- 시점 전환 시 "클리프행어" 배치
- 합류 지점에서 독자가 "아!" 하는 퍼즐 완성감

**[비인기]** 실패:
- 시점 간 흥미도 편차 → 특정 시점 건너뛰기
- 합류가 너무 늦거나 억지스러움

### 3.2 프레임 서사 (액자 구조)

**구조**: 현재 시점(프레임) 안에 과거/다른 이야기(내부 서사)

**효과**:
- 결과를 먼저 알려주고 과정의 궁금증 유발
- 서술자의 신뢰성 의문 제기 가능 (신뢰할 수 없는 화자)
- 시간적 거리감으로 감정적 깊이 추가

### 3.3 군상극 (앙상블)

**구조**: 중심인물 없이 다수의 인물이 동등한 비중

**[인기]** 핵심:
- 인물별 "고유 갈등" + "공유 갈등"의 이중 구조
- 인물 간 만남/충돌이 서사를 추진
- 각 인물의 결말이 전체 테마에 수렴

## 4. 비선형 구조

### 4.1 역순 구조 (리버스 크로놀로지)
- 결말 → 원인으로 거슬러 올라감
- 미스터리/스릴러에 효과적
- 매 단계마다 "왜?"의 답이 새로운 "왜?"를 생성

### 4.2 모자이크 구조
- 시간순 무시, 장면 단위 배치
- 독자가 퍼즐을 맞추는 경험
- **[주의]**: 독자 피로도 높음, 단편이나 실험적 문학에 적합

### 4.3 순환 구조
- 시작과 끝이 같은 이미지/상황
- 변화의 유무로 테마 전달 (변했다 vs 변하지 않았다)
- 문학성 높은 작품에 효과적

## 5. 구조 선택 가이드

| 작품 유형 | 추천 구조 | 이유 |
|----------|----------|------|
| 웹소설 연재 (200화+) | 회차 루프 + 3막 (시즌 단위) | 연재 리듬 유지, 중독성 |
| 웹소설 성장물 | 파워 커브 + 영웅의 여정 | 성장 쾌감, 서사 골격 |
| 로맨스 (웹소설) | 3막 + B스토리 강화 | 관계 발전이 메인 플롯 |
| 단편소설 | 기승전결 / 키시쇼텐케츠 | 압축적, 반전에 유리 |
| 단행본 시리즈 | Save the Cat 비트시트 | 권별 완결성 + 시리즈 연결 |
| 문학 소설 | 5막 / 비선형 | 테마 탐구, 구조적 실험 |
| 미스터리 | 역순 + 3막 | 퍼즐 구성, 반전 |
| 군상극 | 교차 시점 + 앙상블 | 다양한 시각, 풍부한 서사 |
| 회귀물 | 시간 루프 + 파워 커브 | 장르 문법 충족 |

## 6. 구조적 실패 진단 체크리스트

| 증상 | 원인 | 처방 |
|------|------|------|
| "재미는 있는데 뭔가 허전" | 테마/내적 갈등 부재 | B스토리(내면) 강화 |
| "지루하다" | 2막 사기(중간 처짐) | 중간점 반전 삽입, 서브플롯 교차 |
| "전개가 급하다" | 설정 부족 or 해결 급진 | 1막 확장 or 3막 분할 |
| "뻔하다" | 구조를 너무 충실히 따름 | 단계 생략/순서 변경/기대 전복 |
| "뭔 소리야" | 비선형이 과도 | 시간 앵커 삽입, 독자 단서 강화 |
| "캐릭터가 끌려다닌다" | 플롯 우선, 캐릭터 종속 | 캐릭터 동기에서 플롯 유도 |
| "용두사미" | 3막 에너지 부족 | 클라이맥스 전 "최저점" 강화 |
