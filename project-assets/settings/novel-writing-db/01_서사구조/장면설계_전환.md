# 장면 설계와 전환 기법

> 소설의 최소 단위는 "장면(Scene)"이다.
> 장면을 잘 설계하고 자연스럽게 전환하는 것이 몰입의 핵심.

---

## 1. 장면(Scene)의 구조

### 1.1 장면의 정의

장면 = **하나의 시공간에서 하나의 목표를 추구하는 연속된 서술 단위**

장면이 바뀌는 기준:
- 시간이 크게 이동할 때
- 공간이 바뀔 때
- 시점 인물이 바뀔 때
- 감정/목적의 방향이 크게 전환될 때

### 1.2 Scene-Sequel 패턴 (장면-반응 패턴)

**[인기]** 인기작의 90%가 이 패턴을 (의식적이든 무의식적이든) 사용

**Scene (능동 장면)**:
```
목표(Goal) → 갈등(Conflict) → 재앙(Disaster/결과)
- 인물이 무언가를 원한다
- 장애물에 부딪힌다
- 예상 밖의 결과가 나온다
```

**Sequel (반응 장면)**:
```
반응(Reaction) → 딜레마(Dilemma) → 결정(Decision)
- 결과에 대해 감정적으로 반응한다
- 다음에 어떻게 할지 고민한다
- 새로운 목표를 세운다 → 다음 Scene으로
```

**[팁]** Scene과 Sequel의 비율:
| 장르 | Scene : Sequel | 특징 |
|------|---------------|------|
| 액션/무협 | 7:3 | 빠른 전개, 반응은 짧게 |
| 로맨스 | 4:6 | 감정 반응이 핵심 |
| 판타지(균형) | 6:4 | 행동과 성찰 균형 |
| 스릴러 | 8:2 | 멈추면 긴장 해소됨 |
| 일상물 | 3:7 | 관찰과 성찰 중심 |

### 1.3 장면의 5요소

| 요소 | 설명 | 체크 |
|------|------|------|
| **목적** | 이 장면이 존재하는 이유 (플롯/캐릭터/세계관 중 최소 1개) | 목적 없으면 삭제 |
| **갈등** | 이 장면 내의 미시적 긴장감 | 갈등 없으면 축소 |
| **변화** | 장면 전후로 뭔가가 달라져야 함 (정보/감정/관계/상황) | 변화 없으면 통합 |
| **감각** | 독자가 "거기 있는 느낌"을 받는 묘사 | 최소 2감각 |
| **출구** | 다음 장면으로 이어지는 동력 (궁금증/긴장/기대) | 출구 없으면 보강 |

**[비인기]** 실패 패턴:
- 목적 없는 장면 나열 → 독자: "이게 왜 나오지?"
- 변화 없는 장면 반복 → 독자: "진행이 안 되네"
- 갈등 없는 일상 장면이 3개 이상 연속 → 독자 이탈

## 2. 장면 전환 기법

### 2.1 전환의 원칙

**[필수]** 3가지 원칙:
1. **방향성**: 독자가 "왜 여기로 왔는지" 알아야 함
2. **연결성**: 이전 장면과 감정/사건/주제로 연결되어야 함
3. **속도감**: 전환 자체가 페이싱의 일부

### 2.2 전환 유형 8가지

| 유형 | 기법 | 효과 | 예시 |
|------|------|------|------|
| **하드컷** | 줄바꿈 + 시공간 명시 | 빠른 전환, 긴장 유지 | "***\n이튿날 아침, 측정소." |
| **감각 브릿지** | 같은 감각으로 연결 | 자연스러운 흐름 | 장면A 비소리 → 장면B 빗물 흔적 |
| **대사 브릿지** | 이전 대사가 다음 장면의 트리거 | 인과관계 강화 | "내일 보자" → 다음 날 장면 |
| **감정 브릿지** | 같은 감정, 다른 상황 | 주제 연결 | 장면A 슬픔 → 장면B 다른 인물의 슬픔 |
| **대비 전환** | 정반대 분위기로 점프 | 충격/대비 효과 | 평화로운 식사 → 폐허 |
| **시간 점프** | "3일 후", "한 달이 지났다" | 불필요한 시간 건너뛰기 | "그로부터 사흘이 흘렀다." |
| **사물 연결** | 같은 사물이 두 장면에 등장 | 시각적 연결 | 장면A 편지 쓰기 → 장면B 편지 읽기 |
| **회상 진입/복귀** | 트리거 → 과거 → 현재 복귀 | 배경 설명, 감정 심화 | 냄새가 기억을 촉발 → 과거 장면 → 현재로 |

### 2.3 전환 시 금기

| 금기 | 이유 | 대안 |
|------|------|------|
| **아무 연결 없는 장면 나열** | 독자 혼란, 몰입 파괴 | 최소 1가지 브릿지 사용 |
| **전환 후 설명 덤프** | "여기는 ~이고, ~한 곳이다" | 행동/대화 속에 녹이기 |
| **같은 전환 패턴 반복** | 단조로움 | 3가지 이상 기법 섞기 |
| **회상 중 회상(이중 플래시백)** | 독자 시간선 혼란 | 회상은 1단계까지만 |
| **모든 장면을 "아침에 눈을 뜨며" 시작** | 가장 흔한 초보 실수 | 행동/대화/감각으로 시작 |

**[비인기]** 가장 빈번한 실패 — 매 장면이 "잠에서 깨며 시작 → 이동 → 도착 → 사건" 패턴 반복

## 3. 장면 배치 전략

### 3.1 1회차 내 장면 배치

**웹소설 기준 (3,000~5,000자)**:
```
장면 수: 2~4개
구성 예시:
  장면 1 (도입/연결) — 500~800자
  장면 2 (메인 사건) — 1,500~2,500자
  장면 3 (여파/다음 훅) — 500~1,000자
```

**장편 기준 (7,000~9,000자)**:
```
장면 수: 3~6개
구성 예시:
  장면 1 (도입) — 800~1,200자
  장면 2 (전개A) — 1,500~2,000자
  장면 3 (전환/브릿지) — 500~800자
  장면 4 (전개B) — 1,500~2,000자
  장면 5 (클라이맥스/훅) — 1,000~1,500자
```

### 3.2 감정 곡선에 따른 배치

```
[고] ──╮     ╭──────╮
       │     │      │
[중] ──┤─────┤      ├──
       │     │      │
[저] ──╰─────╯      ╰── → 다음 회차
     장면1  장면2  장면3
     (연결) (충돌) (여파)
```

**[인기]** 성공 패턴:
- 1회차 내에서 최소 1번의 감정 고점이 있어야 함
- 마지막 장면은 반드시 "올라가는 중"에 끊기 (클리프행어)
- 첫 장면은 이전 회차의 감정 여운에서 출발

### 3.3 정보 페이싱

**[필수]** 세계관/캐릭터 정보 공개 속도 관리

| 정보 유형 | 공개 방식 | 주의점 |
|----------|----------|--------|
| 세계관 규칙 | 사건을 통해 자연스럽게 | 설명 덤프 절대 금지 |
| 캐릭터 배경 | 행동/대사에서 힌트 → 나중에 전체 공개 | 한 번에 다 알려주면 흥미 ↓ |
| 비밀/반전 | 복선 3단계(씨앗→물주기→개화) | 너무 빨리 드러내면 충격 ↓ |
| 관계 변화 | 미세한 변화 축적 → 결정적 순간 | 갑작스러운 변화는 비현실적 |

**1회차당 새 정보 권장량**:
- 신규 용어/개념: 1~2개
- 캐릭터 새로운 면: 1~2개
- 세계관 디테일: 1~3개
- 비밀/복선 관련: 0~1개

## 4. 시간 다루기

### 4.1 서술 시간 vs 스토리 시간

| 개념 | 설명 |
|------|------|
| **스토리 시간** | 작중 인물이 경험하는 시간 (며칠? 몇 년?) |
| **서술 시간** | 독자가 읽는 데 걸리는 시간 (몇 자? 몇 페이지?) |
| **빈도** | 같은 사건을 몇 번 서술하는가 |
| **순서** | 시간순? 역순? 뒤섞음? |

### 4.2 시간 압축과 팽창

**압축 (요약)**:
```
"그로부터 3개월이 흘렀다. 그 사이 카이는 세 번 더 측정을 받았고,
매번 결과는 같았다." → 3개월을 2문장으로
```
- 사건 없는 기간을 빠르게 넘길 때
- "~동안" 서술로 시간 경과를 압축

**팽창 (확대)**:
```
"검이 떨어지는 0.5초.
그 순간, 세 가지 생각이 머리를 스쳤다."
→ 0.5초를 반 페이지로
```
- 결정적 순간을 극도로 느리게
- 내면 독백, 감각 묘사로 시간을 늘림

**[인기]** 황금 비율: 중요한 장면은 스토리 시간 대비 서술 시간이 10~100배 팽창

### 4.3 회상(플래시백) 운용

**진입 트리거 유형**:
| 트리거 | 예시 |
|--------|------|
| 감각 자극 | 냄새, 소리, 맛이 과거를 떠올림 |
| 유사 상황 | 지금 상황이 과거와 비슷 |
| 대화 중 언급 | 누군가 과거 사건을 말함 |
| 의식적 회상 | 인물이 스스로 떠올림 (가장 약한 트리거) |

**회상의 규칙**:
1. **짧게**: 회상은 현재 스토리보다 절대 길면 안 됨
2. **목적**: 현재 장면에 직접 기여해야 함 (감정/정보/동기)
3. **복귀**: 현재로 돌아올 때 반드시 변화가 있어야 함 (깨달음/결심/감정 변화)
4. **빈도**: 1회차에 회상 1회 이하. 매 회차 회상은 금기

**[비인기]** 회상 실패 패턴:
- 회상이 현재보다 재미있음 → 현재 장면의 존재 이유 상실
- 회상에서 핵심 정보를 덤프 → 회상이 아니라 설정집
- 여러 시점의 회상이 뒤섞임 → 독자 시간선 혼란

### 4.4 시간 구조 유형

| 유형 | 설명 | 적합 장르 |
|------|------|----------|
| 순행 | 시간순으로 서술 | 성장물, 일반 판타지 |
| 역순 | 결과→원인으로 거슬러 | 미스터리, 반전물 |
| 교차 | 현재↔과거 교대 | 회고록, 비밀 공개물 |
| 병렬 | 같은 시간, 다른 공간 | 군상극, 전쟁물 |
| 비선형 | 시간이 뒤섞임 | 실험적 문학, SF |
| 루프 | 같은 시간이 반복 | 회귀물, 타임 루프 |

## 5. 장면 수준 자가 진단

### 5.1 단일 장면 체크리스트

- [ ] 이 장면의 목적(플롯/캐릭터/세계관)이 명확한가?
- [ ] 장면 전후로 무언가가 변했는가?
- [ ] 장면 내에 미시적 갈등/긴장이 있는가?
- [ ] 감각 묘사가 최소 2가지 이상 포함되었는가?
- [ ] 다음 장면으로 이어지는 동력(훅/궁금증)이 있는가?
- [ ] 이 장면을 삭제하면 이야기에 구멍이 생기는가? (아니면 삭제)

### 5.2 장면 전환 체크리스트

- [ ] 이전 장면과의 연결 고리(감각/감정/사물/대사)가 있는가?
- [ ] 전환 후 독자가 시공간을 즉시 파악할 수 있는가?
- [ ] 전환이 너무 급작스럽거나 너무 느리지 않은가?
- [ ] 같은 전환 패턴이 3회 이상 반복되지 않는가?

### 5.3 1회차 장면 구성 체크리스트

- [ ] 2~6개의 장면으로 구성되어 있는가?
- [ ] 최소 1개의 감정 고점이 존재하는가?
- [ ] 새로운 정보가 1~3개 포함되어 있는가?
- [ ] 마지막 장면이 다음 회차의 훅으로 기능하는가?
