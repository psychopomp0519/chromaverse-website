# 회차 구성 전략

> 연재 소설의 회차별 설계. 오프닝/클리프행어/훅, 연재 리듬, 아크 구성.

---

## 1. 회차 내부 구조

### 1.1 표준 구조 (3,000~5,000자)

```
▶ 오프닝 훅 (5~10%): 이전 클리프 해소 or 새로운 긴장
▶ 전개 A (30~40%): 상황 발전, 정보 전달, 캐릭터 교류
▶ 중간 전환점 (10%): 작은 반전, 새 정보, 상황 변화
▶ 전개 B (30~40%): 텐션 상승, 갈등 심화
▶ 클리프행어 (5~10%): 다음 회차로 이끄는 절단
```

### 1.2 유형별 회차 구조

**사이다 회차**: 축적된 고구마를 해소
```
이전 위기 상황 → 역전의 실마리 → 실행 → 사이다 폭발 → 여운+새 복선
```

**고구마 회차**: 텐션 축적
```
위기 접근 → 시도 → 실패 or 상황 악화 → 더 큰 위기 암시
```

**정보 회차**: 세계관/인물 심화
```
일상/대화 → 새로운 정보 공개 → 의미/영향 파악 → 행동 결정
```

**전환 회차**: 아크 간 연결
```
이전 아크 마무리 → 여유/성찰 → 새 목표/위협 감지 → 새 아크 시작
```

---

## 2. 아크 설계

### 2.1 소 아크 (5~10화)

```
화1: 목표 설정 + 훅
화2-3: 준비/탐색 + 첫 시도
화4-5: 위기 + 고구마
화6-7: 전환점 + 반격
화8-9: 클라이맥스 + 사이다
화10: 보상 + 다음 아크 연결
```

### 2.2 중 아크 (20~40화)

```
소아크 1 (도입): 새 환경/적 등장, 목표 설정
소아크 2 (심화): 주요 갈등 전개, 동맹/적대 관계 구축
소아크 3 (전환): 반전, 진짜 갈등 드러남
소아크 4 (결전): 클라이맥스, 해소, 성장 완성
```

### 2.3 대 아크/시즌 (50~100화)

```
중아크 1 (1장): 세계 소개, 첫 번째 시련, 파티 형성
중아크 2 (2장): 스케일 확대, 새 세력, 중간 보스
중아크 3 (3장): 핵심 진실 접근, 최대 위기, 결전
→ 시즌 완결 + 다음 시즌 시드
```

## 3. 회차별 훅 기법

### 3.1 오프닝 훅 유형

| 유형 | 효과 | 빈도 |
|------|------|------|
| 이전 클리프 직접 해소 | "어떻게 됐지?" 즉각 충족 | 가장 흔함 |
| 시간 점프 후 결과 | "아, 그 사이에!" | 간헐적 |
| 새 시점/장면 | 한 박자 늦춘 해소 → 서스펜스 유지 | 클리프 강할 때 |
| 과거/회상 시작 | 현재와의 대비, 정보 보충 | 드물게 |

### 3.2 클리프행어 황금 법칙

1. **매 회차 끝에 반드시 존재**
2. **같은 유형을 3회 연속 금지** (위기-위기-위기 → 예측 가능)
3. **해소 보장**: 깔아놓은 클리프는 반드시 2화 내 해소
4. **강도 변화**: 소-중-소-대-소 리듬

---

## 4. 연재 리듬 관리

### 4.1 주간 리듬 (주 5회 연재 기준)

| 요일 | 회차 유형 | 이유 |
|------|----------|------|
| 월 | 이전 아크 이어가기 or 새 시작 | 주 초 동력 |
| 화-수 | 텐션 축적, 고구마 | 주 중반 |
| 목 | 전환점, 반전 | 후반부 기대감 |
| 금 | 사이다 or 강력한 클리프 | 주말 기대/만족 |

### 4.2 장기 연재 번아웃 방지

- 20~30화마다 "숨 고르기 아크" (일상, 캐릭터 심화)
- 시즌 전환 시 2~3화 이완 구간
- 작가 피로 시 → 짧은 에피소드 아크로 전환
