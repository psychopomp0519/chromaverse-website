# 명장면/명대사 설계

> 독자의 기억에 영원히 남는 순간을 만드는 기술.
> "그 장면" 때문에 작품을 추천하게 만드는 힘.

---

## 1. 명장면의 조건

### 1.1 명장면의 3요소

| 요소 | 설명 |
|------|------|
| **축적** | 장면 이전에 충분한 감정/정보가 쌓여야 함 |
| **집중** | 장면 내에서 한 가지 감정/순간에 집중 |
| **여운** | 장면 이후에도 독자 마음에 남는 울림 |

**공식**: 명장면 = 긴 축적 + 짧은 폭발 + 깊은 여운

### 1.2 명장면이 되는 순간들

| 유형 | 설명 | 사례 |
|------|------|------|
| 각성/돌파 | 한계를 넘는 순간 | 주인공의 최종 각성 |
| 결의/선언 | 모두가 지켜보는 앞에서 선언 | "나는 해적왕이 될 남자다" |
| 희생 | 소중한 것을 내놓는 순간 | 동료를 위한 자기 희생 |
| 진실 공개 | 모든 것의 의미가 바뀌는 순간 | 숨겨진 정체/진실 폭로 |
| 재회/이별 | 감정의 정점 | 오랜 이별 후 재회 |
| 용서/화해 | 갈등의 해소 | 원수와의 화해 |
| 역전 | 절망에서 역전 | 패배 직전의 기적적 반격 |

---

## 2. 명장면 설계 기법

### 2.1 "조용한 폭발" 기법
- 가장 강렬한 순간을 **가장 절제된 표현**으로
```
100화의 축적 후:
"...고마웠어."
그것이 전부였다.
→ 단 한마디가 100화의 무게를 담고 있을 때 명장면
```

### 2.2 "시간 정지" 기법
- 결정적 순간을 극도로 느리게 묘사
```
검이 떨어지는 순간, 시간이 멈춘 것 같았다.
먼지가 천천히 내려앉았다.
누군가 숨을 삼켰다.
— 그리고 그가 손을 뻗었다.
```

### 2.3 "대비" 기법
- 장면 직전에 정반대의 분위기 배치
```
평화로운 식사 장면 → 갑작스러운 습격
(평화가 클수록 습격의 충격이 큼)

절망적 전투 → 구원의 한마디
(절망이 깊을수록 구원의 감동이 큼)
```

### 2.4 "콜백" 기법
- 초반의 사소한 장면이 후반에 거대한 의미로 돌아옴
```
1화: "언젠가 이 검을 쓸 일이 올 거야" (가벼운 농담)
200화: 같은 검으로 세계를 구하는 순간 (농담이 예언이 됨)
→ 독자가 1화를 떠올리며 소름
```

## 3. 명대사 설계

### 3.1 명대사의 조건

| 조건 | 설명 |
|------|------|
| 짧음 | 한 문장, 길어도 두 문장 |
| 캐릭터성 | 그 캐릭터만 할 수 있는 말 |
| 맥락 의존 | 해당 상황에서만 의미가 폭발 |
| 보편성 | 독자의 삶에도 적용 가능한 울림 |

### 3.2 명대사 유형

| 유형 | 예시 구조 |
|------|----------|
| 선언형 | "나는 ~다" / "~하겠다" |
| 질문형 | "넌 왜 ~하는 거야?" |
| 부정형 | "아니, ~가 아니야" |
| 역설형 | "~이기 때문에 ~다" |
| 침묵형 | "..." (말하지 않는 것이 대사) |

### 3.3 명대사의 금기

- 설교형: "인생이란 말이야..." → 독자가 공감하기 전에 결론 제시
- 과장형: 상황에 비해 과도하게 거창한 대사
- 범용형: 아무 캐릭터가 해도 어울리는 대사 → 개성 없음

---

## 4. 명장면 배치 전략

### 4.1 빈도

| 작품 규모 | 명장면 빈도 |
|----------|------------|
| 단편 | 1개 (클라이맥스) |
| 단행본 1권 | 2~3개 |
| 웹소설 50화 | 3~5개 |
| 웹소설 200화 | 10~15개 |

### 4.2 배치 위치
- 아크 클라이맥스 (매 아크 1개)
- 시리즈 전환점 (시즌 끝)
- 캐릭터 아크 완성 시점
- 작품 전체 클라이맥스
