# 독자 몰입 요소

> 독자가 "다음 페이지를 넘기게" 만드는 심리적 메커니즘.
> 사이다/고구마, 감정 동선, 페이지터너 기법.

---

## 1. 독자 심리의 핵심 동력

### 1.1 궁금증 (Curiosity)
- "다음에 뭐가 일어날까?"
- **관리법**: 떡밥 투척 → 부분 회수 → 새 떡밥 → 반복

### 1.2 감정 투자 (Emotional Investment)
- "이 캐릭터가 잘 됐으면 좋겠다"
- **관리법**: 공감→호감→응원→걱정의 단계적 구축

### 1.3 성취 대리 (Vicarious Achievement)
- "내가 한 것처럼 뿌듯하다"
- **관리법**: 도전→고전→성공의 과정을 독자와 함께

### 1.4 소속감 (Belonging)
- "이 세계에 살고 싶다"
- **관리법**: 세계관의 매력, 파티/집단의 유대감

### 1.5 우월감 (Superiority)
- "나는 이미 알고 있지" (독자가 캐릭터보다 많이 알 때)
- **관리법**: 복선 인지, 서스펜스 상황에서의 정보 우위

## 2. 사이다/고구마 시스템

### 2.1 사이다 (쾌감/해소)

**사이다 유형**:
| 유형 | 설명 | 강도 |
|------|------|------|
| 실력 인정 | 숨겨진 실력이 드러남 | 중~대 |
| 모욕 복수 | 무시한 자에게 되갚음 | 대 |
| 진실 폭로 | 숨겨진 진실이 밝혀짐 | 중~대 |
| 선제 대응 | 위기를 미리 막음 | 중 |
| 성장 완성 | 노력의 결실 | 대 |
| 관계 진전 | 고백/화해/인정 | 중~대 |

**사이다 극대화 공식**:
> 사이다 쾌감 = 고구마 축적량 × 해소의 예상 밖 정도

### 2.2 고구마 (답답함/축적)

**고구마 유형**:
| 유형 | 설명 | 한계 |
|------|------|------|
| 불의 | 부당한 대우, 억울함 | 너무 오래 → 이탈 |
| 오해 | 소통 부족에 의한 갈등 | 3~5화 이내 해소 권장 |
| 실패 | 노력했지만 안 됨 | 성장의 전조여야 함 |
| 배신 | 신뢰했던 자의 배반 | 반드시 결과가 따라야 함 |
| 숨김 | 비밀/진실을 말 못함 | 이유가 납득 가능해야 |

**[필수]** 고구마 한계선:
- 고구마가 5화 연속 → 해소 반드시 필요
- 고구마의 원인이 "캐릭터의 바보짓"이면 → 독자 이탈
- 고구마의 원인이 "상황/적의 강력함"이면 → 독자 응원

### 2.3 밸런스 관리

```
이상적 흐름 (10화 기준):
화1: 사이다 (소) — 흥미 유발
화2-3: 고구마 (축적) — 긴장 상승
화4: 사이다 (중) — 중간 해소
화5-7: 고구마 (대축적) — 최대 텐션
화8: 사이다 (대) — 카타르시스
화9: 여유 — 이완
화10: 새 고구마 암시 — 다음 아크 연결
```

## 3. 페이지터너 기법

### 3.1 마이크로 텐션 (문장 단위)

> 매 문장이 다음 문장을 읽게 만드는 힘

**기법**:
- 미완결 정보: "그 순간, 예상하지 못한 일이" → 다음 문장으로
- 감각적 미끼: "이상한 냄새가 났다." → 무슨 냄새?
- 행동의 결과 지연: "그가 버튼을 눌렀다." → 무슨 일이?

### 3.2 매크로 텐션 (회차/장 단위)

- 회차 끝 클리프행어
- 미해결 질문 유지 (항상 1개 이상)
- 새 회차 시작 시 이전 클리프 해소 + 새 텐션 도입

## 4. 독자 감정 동선 설계

### 4.1 감정 롤러코스터

```
기대→긴장→안도→놀라움→감동→불안→분노→쾌감
→ 다양한 감정을 빠르게 순환시키면 "중독성" 발생
```

### 4.2 감정의 낙차 효과

| 순서 | 효과 |
|------|------|
| 기쁨 → 슬픔 | 슬픔 극대화 |
| 공포 → 안도 | 안도감 극대화 |
| 분노 → 쾌감 | 카타르시스 극대화 |
| 일상 → 충격 | 충격 극대화 |

**[인기]** 감정의 낙차가 클수록 독자 경험이 강렬

## 5. 독자 유형별 몰입 포인트

| 독자 유형 | 주요 몰입 포인트 | 장르 선호 |
|----------|----------------|----------|
| 감정 추구형 | 로맨스, 감동, 눈물 | 로맨스, 드라마 |
| 성취 추구형 | 레벨업, 성장, 승리 | 성장물, 시스템물 |
| 지적 추구형 | 추리, 세계관, 복선 | 미스터리, SF, 하이판타지 |
| 스릴 추구형 | 긴장, 공포, 반전 | 스릴러, 호러 |
| 힐링 추구형 | 안정, 따뜻함, 소소함 | 일상물, 힐링 |
