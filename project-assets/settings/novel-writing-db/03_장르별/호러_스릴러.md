# 호러/스릴러 장르

> 공포와 긴장의 설계. 독자의 불안을 조율하는 기술.
> 공포의 유형, 서스펜스 구축, 반전 설계, 심리적 압박.

---

## 1. 하위 장르 분류

### 1.1 심리 호러
- **특징**: 외적 위협보다 내면의 공포. 불안, 광기, 인식 의심
- **핵심**: "진짜 무서운 것은 바깥이 아닌 안에 있다"
- **대표**: 셔터 아일랜드, 검은 집

### 1.2 슬래셔/서바이벌 호러
- **특징**: 살인마/괴물의 추격, 생존
- **핵심**: 위협의 물리적 존재감, 생존 전략
- **대표**: 스크림, 미스트

### 1.3 코스믹 호러
- **특징**: 이해 불가능한 존재, 우주적 공포, 인간의 무의미
- **핵심**: 알면 알수록 공포가 커지는 역설
- **대표**: 러브크래프트 계열, 어나이얼레이션

### 1.4 고딕 호러
- **특징**: 고성, 저택, 가문의 저주, 퇴폐적 분위기
- **핵심**: 과거의 비밀이 현재를 집어삼킴
- **대표**: 레베카, 스위트홈

### 1.5 미스터리/스릴러
- **특징**: 범죄/수사/추리. 퍼즐과 서스펜스
- **핵심**: 단서 배치, 논리적 추리, 반전
- **대표**: 구의 증인, 살인자의 기억법, 셜록 홈즈

### 1.6 심리 스릴러
- **특징**: 인물 간 심리전, 조종과 기만
- **핵심**: 신뢰할 수 없는 상황, 진실 vs 거짓
- **대표**: 나를 찾아줘, 넷플릭스 유

## 2. 공포 설계 기법

### 2.1 공포의 3단계 모델

```
1단계: 불안(Unease) — "뭔가 이상한데..."
   → 미세한 위화감, 일상의 균열
2단계: 공포(Dread) — "다가오고 있다"
   → 위협의 존재 인지, 그러나 정체 불명
3단계: 테러(Terror) — "지금 여기 있다"
   → 직접 대면, 도피 불가
```

**[인기]** 핵심: 1~2단계가 길수록 3단계의 충격이 큼
**[비인기]** 실패: 1단계 없이 3단계부터 시작 → 놀라움은 있지만 공포는 없음

### 2.2 미지(Unknown)의 공포

> "공포는 설명되는 순간 줄어든다"

| 단계 | 공포 수준 | 이유 |
|------|----------|------|
| 무엇인지 모름 | 최대 | 상상이 현실보다 무섭다 |
| 일부만 보임 | 높음 | 빈 부분을 독자가 채움 |
| 정체 드러남 | 중간 | 알게 되면 대응 가능 → 공포 감소 |
| 완전히 설명됨 | 최소 | 미스터리 소멸 |

**[팁]** 괴물/위협은 가능한 한 **늦게, 부분적으로** 드러내라

### 2.3 일상의 파괴

**[인기]** 가장 효과적인 공포: 익숙한 것이 이상해지는 순간
```
매일 지나가던 골목이 갑자기 낯설게 느껴진다.
늘 웃던 이웃이 미소 없이 서 있다.
자주 쓰던 물건이 원래 위치와 살짝 다르다.
```

## 3. 서스펜스 구축

### 3.1 서스펜스 vs 미스터리 vs 서프라이즈

| 유형 | 독자의 정보 | 효과 |
|------|------------|------|
| **서프라이즈** | 독자도 모름 → 갑자기 밝혀짐 | 순간 충격 (1회성) |
| **미스터리** | 독자에게 질문 던짐 → 답 찾아감 | 호기심 (지적 재미) |
| **서스펜스** | 독자가 위험을 알지만 캐릭터는 모름 | 긴장 (지속적) |

**히치콕의 법칙**: "서프라이즈는 15초, 서스펜스는 15분"
- 폭탄이 갑자기 터지면 → 15초 놀람
- 독자만 폭탄을 알고 캐릭터는 모르면 → 15분 내내 긴장

### 3.2 서스펜스 유지 기법

| 기법 | 설명 |
|------|------|
| 시간 제한 | "해가 지기 전에 찾아야 한다" |
| 정보 비대칭 | 독자 > 캐릭터 (위험을 안다) |
| 부분 공개 | 진실의 일부만 보여줌 |
| 가짜 안전 | 안전해 보이는 순간 → 위협 재등장 |
| 딜레마 | 어떤 선택도 위험 |

## 4. 반전(Twist) 설계

### 4.1 좋은 반전의 조건

| 조건 | 설명 |
|------|------|
| 사전 단서 존재 | 돌아보면 복선이 있었음 |
| 논리적 일관성 | 세계관/캐릭터 논리를 깨지 않음 |
| 감정적 충격 | 놀라움 + 감정적 파장 |
| 재독(再讀) 가치 | 다시 읽으면 새로운 의미 발견 |
| 서사적 의미 | 반전이 테마를 강화 |

### 4.2 나쁜 반전의 유형

| 유형 | 문제 |
|------|------|
| 복선 없는 반전 | "갑자기?" — 억지 느낌 |
| 규칙 파괴 반전 | 세계관/논리 자체를 깨뜨림 |
| 충격만을 위한 반전 | 서사적 의미 없음 |
| "꿈이었다" | 모든 서사 무효화 → 독자 배신감 |
| 범인이 미등장인물 | 추리할 기회 없음 → 불공정 |

## 5. 장르별 인기/비인기 종합

### 인기 패턴

| 패턴 | 설명 |
|------|------|
| 신뢰할 수 없는 화자 | 독자의 인식 자체를 뒤흔듦 |
| 밀실/고립 환경 | 탈출 불가 → 긴장 극대화 |
| 괴물의 점진적 노출 | 조금씩만 보여줌 |
| 일상 속 침투 | 안전지대가 무너지는 공포 |
| 공범 독자 | 독자만 진실을 알아 초조 |

### 비인기 패턴

| 패턴 | 문제점 |
|------|--------|
| 고어만 과다 | 공포가 아닌 불쾌감 |
| 바보 캐릭터 | "왜 거기를 들어가?" 답답함 |
| 설명 과다 | 미지가 소멸 → 공포 소멸 |
| 반복 패턴 | 매번 같은 방식 위기→탈출 |
| 구원이 없는 절망만 | 독자가 읽을 이유 상실 |
