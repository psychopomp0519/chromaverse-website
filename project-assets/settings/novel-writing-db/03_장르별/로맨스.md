# 로맨스 장르

> 관계의 발전이 서사의 중심축. 감정선 설계와 독자 감정 조율의 기술.
> 로맨스 판타지(로판), 현대 로맨스, 웹소설 특화 패턴.

---

## 1. 하위 장르 분류

### 1.1 로맨스 판타지 (로판)
- **특징**: 판타지 세계관 + 로맨스 메인. 여성향 웹소설의 주력
- **핵심 설정**: 귀족/왕족 사회, 마법, 계약/정략 결혼, 전생/회귀
- **[인기]** 핵심: 남녀 주인공의 관계 발전 + 여주의 능동적 행동
- **대표**: 이 결혼은 어차피 망합니다, 악녀는 모래시계를 되돌린다

### 1.2 현대 로맨스 (현판)
- **특징**: 현실 배경, 직장/학교/일상 속 연애
- **핵심 설정**: 직장 상사-부하, 재회, 비밀 연애, 계약 연애
- **[인기]** 핵심: 현실적 공감 + 판타지적 설렘의 균형

### 1.3 로맨스 서스펜스/스릴러
- **특징**: 미스터리/스릴러 + 로맨스 결합
- **핵심**: 위험 속 연대가 관계를 가속

## 2. 로맨스 감정선 설계

### 2.1 감정 발전 7단계

```
1. 첫 만남/첫인상 ─── 호기심 또는 반감
2. 접근/재회 ─────── 의도적/우연적 접촉 반복
3. 끌림 인지 ─────── "이 감정이 뭐지?" 자각
4. 장벽 ──────────── 내적(두려움)/외적(신분/상황) 장애
5. 밀당/시험 ─────── 가까워졌다 멀어지기 반복
6. 전환점 ─────────── 결정적 사건으로 관계 변화
7. 결합/확인 ─────── 감정 확인, 관계 확립
```

**[인기]** 핵심: 3~5단계(끌림~밀당)가 전체의 50~60%를 차지해야 함
**[비인기]** 실패: 1→7 급속 전개 ("만나자마자 사랑에 빠짐") → 감정이입 불가

### 2.2 설렘 포인트 설계

**독자의 심장이 뛰는 순간들**:

| 유형 | 설명 | 빈도 |
|------|------|------|
| 미세 접촉 | 손끝, 시선, 옷깃 스침 | 매 1~2화 |
| 돌봄/보호 | 상대를 위한 행동 (타인 모름) | 3~5화마다 |
| 시선 교차 | 의미 있는 눈 맞춤 | 매 1~2화 |
| 독점 행동 | "나한테만" 다른 태도 | 5~10화마다 |
| 고백/직설 | 감정을 직접 표현 | 핵심 전환점 |
| 위기 속 구원 | 위험에서 지켜줌 | 아크마다 1회 |
| 질투 | 제3자에 의한 감정 자극 | 적절히 사용 |

### 2.3 밀당 (Push-Pull) 역학

```
당기기(Pull): 호감 표현, 접근, 친절, 보호 → 독자 기대감 ↑
  ↕ 반복
밀기(Push): 거리두기, 오해, 장벽, 회피 → 독자 불안감 ↑
```

**[인기]** 밀당 비율: 당기기 60% / 밀기 40%
**[비인기]** 밀기 과다: "이 사람 진짜 좋아하는 거 맞아?" → 독자 짜증

## 3. 로맨스 장벽 유형

### 3.1 외적 장벽 (상황/환경)

| 장벽 | 설명 | 해소 방향 |
|------|------|----------|
| 신분 차이 | 귀족/평민, 상사/부하 | 능력 증명, 신분 상승, 사회 변화 |
| 정략 결혼/약혼 | 다른 사람과 엮임 | 파혼 사건, 약혼자의 퇴장 |
| 거리/환경 | 물리적 분리 | 재회 사건, 노력 |
| 가문/가족 반대 | 집안의 반대 | 설득, 독립, 가족 문제 해결 |
| 사회적 편견 | 금기 관계 | 편견과 싸우는 과정 자체가 서사 |

### 3.2 내적 장벽 (심리/감정)

| 장벽 | 설명 | 해소 방향 |
|------|------|----------|
| 과거 트라우마 | 이전 연애/상처 | 치유, 새로운 신뢰 |
| 자존감 부족 | "나는 자격이 없다" | 자기 수용 + 상대의 인정 |
| 감정 회피 | 사랑을 두려워함 | 감정 직면의 계기 |
| 오해 | 잘못된 정보에 의한 불신 | 진실 소통 |
| 자기 희생 | "상대를 위해 떠나야 해" | "함께하는 것이 진짜 위하는 것" 깨달음 |

**[인기]** 내적 장벽이 외적 장벽보다 감정적으로 강력
**[비인기]** 오해 장벽만 과다: "대화 한 번이면 풀릴 일을..." → 독자 답답

## 4. 로맨스 캐릭터 유형

### 4.1 여성 주인공 인기 유형 (여성향)

| 유형 | 특징 | 인기 요소 |
|------|------|----------|
| 능동적 지략형 | 머리로 상황 타개 | 답답하지 않은 전개 |
| 회귀/환생형 | 과거 실패 만회 | 성장 + 복수 + 로맨스 |
| 강인한 외유내강형 | 부드러우나 결정적 순간 강함 | 반전 매력 |
| 무심한 쿨형 | 남주에게 관심 없음 → 끌림 | "사랑에 빠지는 과정"의 재미 |

### 4.2 남성 주인공 인기 유형 (여성향)

| 유형 | 특징 | 인기 요소 |
|------|------|----------|
| 차도남(여주에게만 따뜻) | 타인에게 냉정, 여주에게 헌신 | 독점감 |
| 강하지만 결함 있는 | 능력자이나 감정적 결핍 | 치유 서사 |
| 충견형 | 여주에 절대 충성 | 안정감, 신뢰 |
| 츤데레 | 겉은 차갑고 속은 따뜻 | 갭 모에 |

### 4.3 남성향 로맨스 특성

| 특징 | 설명 |
|------|------|
| 직접적 감정 표현 | 감정 간접 표현보다 행동/선언 선호 |
| 외적 갈등 비중 높음 | 전투/모험 속 로맨스 |
| 다수 히로인 | 각기 다른 매력의 인물들 |
| 영웅-히로인 역학 | 위기에서 구하는 구조 빈번 |

## 5. 인기/비인기 패턴

### 5.1 인기 패턴

| 패턴 | 설명 |
|------|------|
| 계약 관계→진심 | 가짜 연인/결혼이 진짜가 됨 |
| 재회 로맨스 | 과거 인연이 재회, 과거 청산 |
| 원수→연인 | 적대 관계에서 사랑으로 |
| 보호자→연인 | 돌봐주다 사랑에 빠짐 |
| 첫사랑 재발견 | 성장 후 첫사랑의 의미 재해석 |

### 5.2 비인기 패턴 (주의)

| 패턴 | 문제점 | 대안 |
|------|--------|------|
| 즉석 사랑 | 감정 발전 과정 없음 | 끌림→갈등→확인의 과정 필수 |
| 삼각관계 과다 | 질질 끌면 독자 피로 | 일찍 해소하거나 양쪽에 의미 부여 |
| 소유욕=사랑 | 불건강한 관계 미화 | 존중 기반 관계 |
| 여주 수동성 | "구해지기만 기다림" | 자기 의지로 행동하는 여주 |
| 소통 부재 과다 | "한마디면 풀릴 것을" 반복 | 소통 못하는 합리적 이유 필요 |
| 연인 외 관계 부재 | 세계가 연인 중심으로만 돌아감 | 친구/가족/동료 관계 병행 |

## 6. 로맨스 묘사 기법

### 6.1 감정 묘사의 간접화

**[금기]** 직접 서술:
```
❌ 그녀는 그를 사랑했다.
❌ 심장이 두근거렸다.
```

**[인기]** 간접 묘사:
```
✅ 그가 고개를 돌린 순간, 그녀는 자신이 숨을 멈추고 있었다는 걸 깨달았다.
✅ 멀어지는 그의 등을 보며, 손이 저절로 소매 끝을 움켜쥐었다.
✅ 그의 이름을 부르려다 입술만 달싹였다. 목소리가 나오지 않았다.
```

### 6.2 시선 묘사 기법

```
"눈이 마주쳤다" — 가장 기본적이지만 여전히 효과적
"시선이 닿았다" — 부드러운 접촉감
"눈을 피했다" — 의식하고 있음의 반증
"오래 바라보았다" — 감정의 깊이
"시선을 거두지 못했다" — 끌림의 강도
```

### 6.3 물리적 거리 묘사

```
거리 축소: "어느새 한 뼘 거리" → "숨결이 닿을 만큼" → "온기가 느껴지는"
거리 확대: "한 걸음 물러섰다" → "등을 돌렸다" → "문이 닫혔다"
→ 물리적 거리 = 감정적 거리의 시각화
```

## 7. 로맨스 체크리스트

- [ ] 두 사람이 끌리는 이유가 명확한가? (외모 외 요소)
- [ ] 감정 발전의 각 단계가 자연스럽게 연결되는가?
- [ ] 장벽이 합리적이고, 해소 시 성장이 동반되는가?
- [ ] 로맨스 외 서브플롯(성장/사회/모험)이 있는가?
- [ ] 밀당이 적절하고, 독자를 지치게 하지 않는가?
- [ ] 양쪽 캐릭터 모두 독립적 인격과 목표가 있는가?
- [ ] 감정 묘사가 간접적(행동/감각)으로 이루어지는가?
- [ ] 관계가 건강한가? (소유욕/통제를 미화하지 않는가?)
