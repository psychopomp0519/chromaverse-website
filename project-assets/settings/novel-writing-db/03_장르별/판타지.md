# 판타지 장르

> 가장 넓은 스펙트럼을 가진 장르. 하위 장르별 특성, 세계관 구축, 마법 체계, 인기/비인기 패턴 정리.

---

## 1. 하위 장르 분류

### 1.1 하이 판타지 (High Fantasy)
- **특징**: 완전히 창작된 세계, 선악 대립, 서사시적 스케일
- **대표**: 반지의 제왕, 얼음과 불의 노래, 눈물을 마시는 새
- **필수 요소**: 독자적 세계관, 역사/지리/문화, 마법 체계, 종족
- **[인기]** 핵심: 세계관이 "살아 숨 쉬는" 느낌 — 주인공이 없어도 세계가 굴러가는 밀도
- **[비인기]** 실패: 세계관 설명에 치중 → 이야기가 진행 안 됨

### 1.2 로우 판타지 (Low Fantasy)
- **특징**: 현실 세계에 마법적 요소 침투
- **대표**: 해리포터, 어반 판타지 계열
- **필수 요소**: 현실과 판타지의 경계, 비밀 사회, 일상↔비일상의 대비
- **[인기]** 핵심: "내 옆집에도 마법사가 살 수 있다"는 친밀감

### 1.3 다크 판타지 (Dark Fantasy)
- **특징**: 어둡고 잔혹한 세계, 도덕적 모호성, 비극적 요소
- **대표**: 베르세르크, 다크소울, 흑집사
- **필수 요소**: 절망적 세계관, 안티히어로, 희생과 대가, 호러 요소
- **[인기]** 핵심: 어둠 속에서 빛나는 인간성의 찰나

### 1.4 동양 판타지 / 무협 판타지
- **특징**: 동양 신화/무술/기(氣) 체계 기반
- **대표**: 화산귀환, 나혼자만 레벨업(동양풍), 중국 선협소설
- **필수 요소**: 무공/기 체계, 문파/종파, 사제 관계, 강호(江湖) 사회
- **[인기]** 핵심: 수련→돌파의 성장 쾌감 + 의리와 배신의 관계극

### 1.5 이세계 판타지 (Isekai)
- **특징**: 현대인이 다른 세계로 이동
- **대표**: 무직전생, 오버로드, 이 멋진 세계에 축복을
- **필수 요소**: 소환/전이 메커니즘, 현대 지식 활용, 문화 충격
- **[인기]** 핵심: 현대 지식/감각으로 이세계를 바라보는 "신선한 시점"
- **[비인기]** 실패: 이세계가 "게임 같은 세계"로 얕게 설정

## 2. 세계관 구축 (Worldbuilding)

### 2.1 세계관 5대 축

| 축 | 핵심 질문 | 서사 연결 |
|----|----------|----------|
| **지리** | 세계의 물리적 구조? | 여정의 장애물, 전쟁의 원인 |
| **역사** | 어떤 사건이 현재를 만들었나? | 복선, 갈등의 뿌리 |
| **사회** | 누가 권력을 가졌나? 계층은? | 사회적 갈등, 캐릭터 동기 |
| **문화** | 종교, 관습, 가치관은? | 캐릭터 행동 규범, 문화 충돌 |
| **규칙** | 마법/기술의 법칙은? | 문제 해결과 제약의 기반 |

### 2.2 빙산 이론 (Iceberg Theory)

```
독자에게 보이는 부분 (10~20%)
━━━━━━━━━━━━━━━━━━━━━━━━━
작가가 알고 있는 부분 (80~90%)
→ 보이는 10%가 자연스러워지려면 보이지 않는 90%가 필요
→ 그러나 90%를 다 보여주면 안 됨
```

**[인기]** 세계관 노출 원칙:
- "필요할 때, 필요한 만큼" — 장면에 유기적으로 녹임
- 캐릭터의 행동/대화/반응으로 세계관을 "보여주기"
- 직접 설명은 전체의 20% 미만

**[비인기]** 세계관 과다 노출:
- 첫 3화가 세계관 설명 → 독자 이탈
- 정보 덤프(Info Dump) — 캐릭터가 멈추고 세계관 강의
- "세계관은 대단한데 이야기가 없어"

### 2.3 세계관 도입 스케줄

| 단계 | 타이밍 | 도입 방식 |
|------|--------|----------|
| 1단계 | 1~5화 | 일상 속 자연 노출 (세계의 "표면") |
| 2단계 | 6~20화 | 사건을 통한 심화 (세계의 "규칙") |
| 3단계 | 21~50화 | 역사/비밀 드러남 (세계의 "이면") |
| 4단계 | 51화~ | 세계의 근본에 닿음 (세계의 "진실") |

## 3. 마법/힘의 체계 (Magic System)

### 3.1 경성 마법 vs 연성 마법

| 구분 | 경성 (Hard) | 연성 (Soft) |
|------|------------|------------|
| **규칙** | 명확한 규칙과 제약 | 모호하고 신비로움 |
| **해결** | 마법으로 문제 해결 가능 | 마법은 분위기/위협용 |
| **대표** | 브랜든 샌더슨, 나루토 | 반지의 제왕, 지브리 |
| **장점** | 퍼즐적 재미, 전략적 전투 | 경이로움, 서정미 |
| **위험** | 게임적/기계적 느낌 | 데우스 엑스 마키나 위험 |

### 3.2 샌더슨의 법칙 3가지

**제1법칙**: "작가가 마법 체계를 독자에게 잘 설명할수록, 마법으로 갈등을 해결해도 괜찮다"
- 설명 안 된 마법으로 문제 해결 = 억지
- 잘 설명된 마법으로 문제 해결 = 카타르시스

**제2법칙**: "제한/약점/대가가 힘 자체보다 중요하다"
- 독자가 관심 갖는 것: "얼마나 강한가"보다 "무엇을 못 하는가"
- 제약이 갈등을 만들고, 갈등이 이야기를 만듦

**제3법칙**: "새로운 마법을 추가하기 전에 기존 마법을 확장하라"
- 능력 인플레이션 방지
- 기존 능력의 새로운 활용이 더 인상적

### 3.3 마법 체계 설계 체크리스트

- [ ] 마법의 원천/에너지원은 무엇인가?
- [ ] 마법의 대가/제약은 무엇인가?
- [ ] 마법의 한계(못 하는 것)는 명확한가?
- [ ] 마법이 사회/경제/정치에 미치는 영향을 고려했는가?
- [ ] 마법 간 상성/상호작용이 있는가?
- [ ] 마법의 규칙이 이야기 전체에서 일관되는가?

## 4. 판타지 인기/비인기 패턴

### 4.1 인기 패턴

| 패턴 | 설명 | 대표작 |
|------|------|--------|
| 약자 시작 성장 | 바닥에서 시작, 단계적 강해짐 | 나홀렙, 무직전생 |
| 숨겨진 세계의 비밀 | 세계가 알려진 것과 다름 | 전독시, 진격의 거인 |
| 복수와 재기 | 배신/몰락 후 재기 | 화산귀환, 몬테크리스토 |
| 선택받지 못한 영웅 | "선택받은 자"를 뒤집음 | 약먹는 천재마법사 |
| 파티 케미 | 동료 간 유대와 성장 | 던전밥, 프리렌 |

### 4.2 비인기 패턴 (피해야 할 것)

| 패턴 | 문제점 | 대안 |
|------|--------|------|
| 먼치킨 무쌍 | 도전/긴장 없음 | 강함에 대가/제약 부여 |
| 하렘 수집 | 캐릭터 깊이 없음 | 소수의 깊은 관계 |
| 세계관 설명 폭격 | 이야기 진행 없음 | 사건 속 자연 노출 |
| 클리셰 종족(엘프/드워프) 복붙 | 독창성 없음 | 종족 특성에 반전 부여 |
| 능력 인플레이션 | 이전 위기감 무력화 | 질적 변화 > 양적 강화 |
| 설정만 거창 | 이야기가 설정을 못 따라감 | 설정은 이야기를 위한 도구 |

## 5. 판타지 전투 장면 설계

### 5.1 전투의 3층 구조
```
전략층: 왜 싸우는가? (갈등/동기)
전술층: 어떻게 싸우는가? (마법/기술 활용)
감정층: 싸우면서 무엇을 느끼는가? (두려움/분노/결의)
```

**[필수]** 3층이 모두 존재해야 명장면. 전술만 있으면 스포츠 중계.

### 5.2 전투 실패 패턴

| 실패 | 원인 | 해결 |
|------|------|------|
| 전투가 지루 | 감정층 부재 | 개인적 스테이크 부여 |
| 전투가 난잡 | 전략/전술 불명확 | 목표와 장애물 명확히 |
| 전투가 뻔함 | 항상 주인공 승리 | 패배/피해/대가 도입 |
| 파워 밸런스 붕괴 | 능력 인플레이션 | 제약/상성 체계 활용 |

## 6. 판타지 집필 체크리스트

- [ ] 세계관이 이야기에 봉사하는가? (세계관 ≠ 이야기)
- [ ] 마법 체계에 명확한 제약이 있는가?
- [ ] 세계관 정보가 장면에 자연스럽게 녹아 있는가?
- [ ] "왜 이 세계여야 하는가?" — 현실에서 불가능한 이야기인가?
- [ ] 종족/문화가 단순 클리셰를 넘어서는가?
- [ ] 전투가 전략+전술+감정의 3층 구조를 갖추었는가?
- [ ] 능력 성장에 대가와 제약이 있는가?
