# 퓨전/크로스 장르

> 장르 혼합의 기술. 이세계/회귀/환생, 시스템물, 장르 결합 패턴.
> 한국 웹소설 시장의 주력 장르이자 가장 빠르게 진화하는 분야.

---

## 1. 주요 퓨전 유형

### 1.1 이세계물 (Isekai/Transmigration)
- **구분**: 전이(다른 세계로 이동) / 빙의(기존 인물 몸에 깨어남)
- **[인기]** 핵심: 현대인의 시점으로 이세계를 "신선하게" 바라봄
- **주의**: 이세계가 단순 게임 세계면 깊이 부족

### 1.2 회귀물
- **핵심**: 미래 지식을 가지고 과거로. 복수/재기 서사
- **[인기]** 패턴: 지식의 한계(나비효과) + 감정적 대가 + 관계 재구성
- **[비인기]** 패턴: 모든 것을 아는 전지적 주인공 → 긴장 0

### 1.3 시스템물
- **핵심**: 게임적 시스템(레벨, 스킬, 퀘스트)이 현실에 도입
- **[인기]** 조건: 시스템이 "왜 존재하는가"에 대한 답이 있을 때
- **[비인기]** 조건: 시스템이 단순 치트키 역할만

### 1.4 아포칼립스/게이트물
- **핵심**: 갑작스러운 세계 변화(던전/게이트/재난) 후 적응
- **[인기]** 요소: 기존 사회 구조 + 새 질서의 충돌

### 1.5 장르 결합 유형

| 결합 | 특징 | 대표 |
|------|------|------|
| 판타지 + 로맨스 | 로판 | 이혼 못합니다류 |
| 판타지 + 게임 | 시스템물 | 나홀렙, 템빨 |
| 무협 + 회귀 | 회귀무협 | 화산귀환 |
| SF + 호러 | 우주 호러 | 에일리언 |
| 일상 + 판타지 | 어반판타지 | 드래곤라자 |
| 로맨스 + 스릴러 | 로맨스 서스펜스 | 나를 찾아줘 |

## 2. 퓨전 장르의 핵심 원칙

### 2.1 "주 장르 + 부 장르" 구조

**[필수]** 두 장르의 비율을 명확히 설정:
```
주 장르 (70%): 이야기의 근간, 독자 유입 이유
부 장르 (30%): 차별화 요소, 깊이 추가
```

**예시**:
- 회귀 무협: 무협(주 70%) + 회귀(부 30%)
- 로판: 로맨스(주 60%) + 판타지(부 40%)

**[비인기]** 실패: 주 장르 불명확 → "이게 뭔 장르야?" → 독자 혼란

### 2.2 장르 문법 충족

**[필수]** 주 장르의 핵심 문법을 반드시 이행:
- 로맨스 → 설렘 포인트 필수
- 액션 → 전투 긴장감 필수
- 미스터리 → 단서와 추리 필수
- 성장물 → 성장 과정 필수

**[비인기]** 장르 문법 불이행:
- "로맨스인데 설레는 장면이 없어"
- "전투물인데 전투가 재미없어"

### 2.3 고유 시스템 설계 (시스템물)

**시스템의 5가지 질문**:
1. 왜 존재하는가? (기원/목적)
2. 어떻게 작동하는가? (규칙)
3. 한계는 무엇인가? (제약)
4. 사회에 어떤 영향을 미치는가? (파급)
5. 주인공에게만 특별한 점은? (차별화)

## 3. 클리셰 관리

### 3.1 웹소설 퓨전 장르 주요 클리셰

| 클리셰 | 빈도 | 처방 |
|--------|------|------|
| 회귀 후 주식/부동산으로 부자 | 극높음 | 재산이 목적이 아닌 수단으로 |
| 전생 기억으로 만렙 지식 | 극높음 | 기억의 한계/오류 설정 |
| 악역 영애가 실은 선인 | 높음 | 진짜 악한 면도 가진 복합성 |
| 히든 보스가 주인공 | 높음 | 히든 보스임의 대가/부담 |
| 각성/강림/시스템 창 | 높음 | 시스템의 목적과 의미 부여 |

### 3.2 클리셰 활용법

**[팁]** 클리셰 자체가 나쁜 것이 아님. 처리 방식이 문제:

| 접근법 | 설명 |
|--------|------|
| 직행 | 클리셰를 그대로 사용하되 실행 품질로 승부 |
| 변주 | 클리셰의 예상을 한 단계만 비틂 |
| 해체 | 클리셰가 왜 작동하는지/못하는지 자체를 탐구 |
| 결합 | 두 클리셰를 합쳐 새로운 것 생성 |

## 4. 인기/비인기 종합

### 인기 패턴

| 패턴 | 핵심 |
|------|------|
| 정보 우위의 합리적 활용 | 회귀 지식을 논리적으로 사용 |
| 성장에 의미 있는 대가 | 강해지는 것이 "좋기만 한" 게 아님 |
| 사회/관계의 변화 | 주인공의 행동이 세계를 바꿈 |
| 장르 기대의 신선한 변주 | "뻔한 거 아니었어?" 순간 |

### 비인기 패턴

| 패턴 | 문제점 |
|------|--------|
| 치트키 남발 | 위기감 없음 |
| 줄거리 = 스킬 획득 목록 | 이야기가 아닌 카탈로그 |
| 주변 인물 바보화 | 주인공 돋보이려고 세계가 멍청해짐 |
| 여러 장르 반반씩 | 어느 쪽도 만족 못함 |
| 세계관 설정 미비 | 시스템만 있고 세계가 없음 |
