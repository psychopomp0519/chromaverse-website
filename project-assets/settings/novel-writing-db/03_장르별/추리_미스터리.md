# 추리/미스터리 장르

> 독자와의 지적 게임. "범인은 누구인가?"를 넘어 "진실은 무엇인가?".
> 단서, 추리, 반전 — 논리와 서프라이즈의 균형.

---

## 1. 하위 장르 분류

### 1.1 본격 추리 (본격 미스터리)
- **특징**: 논리적 추리, 페어플레이 원칙, 독자도 풀 수 있는 퍼즐
- **핵심**: "독자에게 모든 단서를 제공하되, 답은 보여주지 않는다"
- **대표**: 셜록 홈즈, 명탐정 코난, 금전차, 엘러리 퀸

### 1.2 사회파 미스터리
- **특징**: 범죄를 통해 사회 문제 탐구
- **핵심**: "왜 범죄가 일어났는가"가 "누가 했는가"보다 중요
- **대표**: 구의 증인, 살인자의 기억법, 범죄의 재구성

### 1.3 서스펜스/스릴러
- **특징**: 독자가 범인/위협을 알고 있는 상태에서의 긴장감
- **핵심**: "어떻게 벗어나는가", "잡히기 전에 막을 수 있는가"
- **대표**: 양의 침묵, 올드보이, 파피용의 성

### 1.4 코지 미스터리
- **특징**: 잔인하지 않은 미스터리, 일상+추리 결합
- **핵심**: 아늑한 배경(마을/가게)+가벼운 살인 사건
- **대표**: 비블리아 고서당, 카모메 식당 시리즈

### 1.5 법정 미스터리
- **특징**: 재판 과정이 서사의 중심
- **핵심**: 증거/증언의 반전, 법정 드라마
- **대표**: 역전재판, 소송, 링컨 변호사

## 2. 추리 설계의 핵심 원칙

### 2.1 페어플레이 (Fair Play) 원칙

**녹스의 10계명** (현대적 재해석):

| 원칙 | 현대적 의미 |
|------|-----------|
| 범인은 초반에 등장해야 한다 | 막판에 새 인물이 범인이면 안 됨 |
| 초자연적 해결 금지 | SF/판타지면 그 세계의 규칙 내에서 |
| 비밀 통로/장치는 사전 암시 | 트릭은 독자도 알 수 있어야 |
| 미발견 독약/장치 금지 | 독자에게 공정한 단서 제공 |
| 단서는 모두 작중에 존재 | 독자가 다시 읽으면 찾을 수 있어야 |

### 2.2 단서 배치 기법

**단서의 3층 구조**:

| 층 | 설명 | 비율 |
|----|------|------|
| **표면 단서** | 직접적으로 제시, 독자가 쉽게 발견 | 30% |
| **매몰 단서** | 다른 정보 사이에 묻힘, 주의 깊은 독자만 발견 | 50% |
| **역방향 단서** | 오해를 유도하는 미스디렉션, 나중에 진짜 의미 드러남 | 20% |

**단서 매몰 기법**:
| 기법 | 설명 |
|------|------|
| **목록 삽입** | 여러 정보를 나열하며 단서를 하나 섞음 |
| **감정 전이** | 강한 감정 장면 직후에 단서 배치 → 독자 주의 분산 |
| **일상 위장** | 단서를 평범한 일상 묘사로 위장 |
| **대화 삽입** | 중요 단서를 일상 대화 속에 자연스럽게 |
| **시점 제한** | 화자가 단서를 보고도 의미를 모르는 채 서술 |

### 2.3 트릭과 반전

**좋은 반전의 4조건**:
1. **놀라움**: "설마!" → 독자의 예상을 벗어남
2. **납득**: "그래서 그랬구나!" → 돌아보면 모든 단서가 맞음
3. **감정**: 논리적 정답 + 감정적 충격
4. **의미**: 반전이 플롯/캐릭터에 깊이를 더함

**반전의 유형**:
| 유형 | 설명 | 효과 |
|------|------|------|
| 범인 반전 | 예상 밖의 인물이 범인 | 놀라움 |
| 동기 반전 | 범인은 알지만 이유가 충격 | 감정적 충격 |
| 피해자 반전 | 피해자가 사실 가해자였음 | 도덕적 혼란 |
| 탐정 반전 | 추리하는 자가 관련자 | 서사적 복잡도 |
| 구조 반전 | 이야기 자체의 전제가 뒤집힘 | 최대 충격 |

## 3. 추리 구조 설계

### 3.1 기본 구조

```
[사건 발생] → [수사/탐문] → [단서 수집] → [추리] → [반전/진실] → [해결]

웹소설 적용:
1~2화: 사건 발생 + 탐정 역할 확립
3~6화: 수사, 용의자 탐문, 단서 수집
7~8화: 중간 반전 (첫 추리가 틀림)
9~10화: 진짜 단서 발견, 최종 추리
결말화: 범인 대면, 진실 공개, 여파
```

### 3.2 다중 수수께끼 구조 (장편)

```
사건 A (소규모) ───해결───→
사건 B (중규모) ─────────해결──→
사건 C (대규모/연쇄) ──────────────해결→
                                    ↑
                               사건 A, B가
                               사건 C의 단서
```

**[인기]** 소규모 사건을 해결하며 큰 사건의 단서를 축적 = 독자 만족도 최대

### 3.3 정보 비대칭 설계

| 유형 | 설명 | 효과 |
|------|------|------|
| 독자 > 탐정 | 독자가 먼저 알아챔 | 서스펜스 ("빨리 알아차려!") |
| 탐정 > 독자 | 탐정이 뭔가 알지만 안 말함 | 미스터리 ("뭘 알고 있는 거야?") |
| 독자 = 탐정 | 함께 추리 | 참여감 |
| 모두 모름 | 모든 단서가 부족 | 공포/불안 |

## 4. 추리와 다른 장르의 결합

| 결합 | 핵심 | 예시 |
|------|------|------|
| 추리+판타지 | 마법 세계의 규칙 내에서 추리 | 마법 트릭, 능력 기반 알리바이 |
| 추리+로맨스 | 사건 해결 과정에서 관계 발전 | 파트너 탐정 |
| 추리+호러 | 범인이 초자연적 존재일 수 있다는 공포 | 공포+논리의 줄다리기 |
| 추리+역사 | 과거 사건의 진실 추적 | 고증+추리 |
| 추리+SF | 미래 기술 기반의 새로운 범죄/트릭 | 기술적 추리 |

## 5. 인기/비인기 패턴

### 5.1 인기 패턴

| 패턴 | 핵심 |
|------|------|
| 매력적 탐정 | 특이한 능력/성격의 탐정 캐릭터 |
| 페어플레이 | 독자가 다시 읽으면 "아!" 하는 단서 |
| 다층 반전 | 1차 반전 후 2차 반전 (더 깊은 진실) |
| 인간적 동기 | 범죄의 이유가 공감 가능 |
| 도덕적 딜레마 | 진실을 밝히는 것이 정의인가? |

### 5.2 비인기 패턴

| 패턴 | 문제점 |
|------|--------|
| 불공정한 반전 | 독자에게 단서를 주지 않고 반전 → 사기 느낌 |
| 범인의 동기 빈약 | "그냥 미쳤으니까" → 설득력 없음 |
| 트릭 과시 | 이야기보다 트릭 자체에 집착 |
| 수사 과정 지루 | 단서만 나열, 캐릭터/감정 없음 |
| 우연의 남발 | "마침 그때 ~가" → 논리적 추리 파괴 |
