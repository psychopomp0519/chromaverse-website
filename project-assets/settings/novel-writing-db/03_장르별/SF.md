# SF (Science Fiction) 장르

> 과학적 상상력으로 인간의 본질을 탐구하는 장르.
> "만약 ~라면?"(What if?) 질문에서 시작하는 사고 실험.

---

## 1. 하위 장르 분류

### 1.1 하드 SF
- **특징**: 과학적 정확성 중시, 기술적 디테일
- **핵심**: 과학 법칙 내에서의 상상력
- **대표**: 마션, 세 몸 문제, 인터스텔라

### 1.2 소프트 SF
- **특징**: 사회과학/인문학적 상상력, 기술은 배경
- **핵심**: 기술이 인간/사회에 미치는 영향
- **대표**: 시녀 이야기, 멋진 신세계, 1984

### 1.3 사이버펑크
- **특징**: 고도 기술 + 저급 삶, 기업 지배, 해커
- **핵심**: 기술 발전과 사회적 불평등의 모순
- **대표**: 뉴로맨서, 블레이드 러너, 공각기동대

### 1.4 스페이스 오페라
- **특징**: 은하 규모 서사, 우주 전쟁, 문명 충돌
- **핵심**: 웅장한 스케일 + 인간 드라마
- **대표**: 듄, 은하영웅전설, 스타워즈

### 1.5 포스트 아포칼립스
- **특징**: 문명 붕괴 후 세계, 생존
- **핵심**: 극한 상황에서의 인간성 탐구
- **대표**: 더 로드, 스테이션 일레븐, 매드맥스

### 1.6 타임 SF
- **특징**: 시간 여행, 시간 조작
- **핵심**: 인과율, 패러독스, 선택의 의미
- **대표**: 당신 인생의 이야기, 다크, 타임루프물

## 2. SF의 핵심 질문 설계

### 2.1 "What if?" 프레임워크

**1단계**: 기술/과학적 가정 설정
```
"만약 인간의 기억을 복사할 수 있다면?"
```

**2단계**: 사회적 파급 추론
```
"기억 시장이 형성된다", "죽은 사람의 기억으로 범죄 수사"
"기억이 다른 두 '나'의 정체성 문제"
```

**3단계**: 인간적 딜레마 도출
```
"복사된 기억의 '나'는 진짜 나인가?"
"사랑하는 사람의 기억을 사서 보는 것은 도덕적인가?"
```

**[인기]** 핵심: 기술 자체보다 기술이 만드는 **인간적 질문**이 중심

### 2.2 SF적 세계관의 일관성

**[필수]** 기술의 영향을 끝까지 추적:
- 전송 기술이 있다면 → 물류/교통/전쟁/범죄 모두 변함
- 불멸 기술이 있다면 → 인구/경제/종교/윤리 모두 변함
- **일부만 변하고 나머지는 현대와 같으면 → 설득력 상실**

## 3. SF 인기/비인기 패턴

### 3.1 인기 패턴

| 패턴 | 핵심 |
|------|------|
| 기술의 양면성 | 유토피아적 기술이 디스토피아를 만듦 |
| 인간성의 경계 | AI/복제인간/개조인간 — "인간이란 무엇인가" |
| 문명 접촉 | 이질적 문명 간 소통/충돌 |
| 극한 생존 | 우주/재난에서의 과학적 문제 해결 |
| 시간의 아이러니 | 바꾸려 할수록 꼬이는 시간 |

### 3.2 비인기 패턴

| 패턴 | 문제점 |
|------|--------|
| 기술 나열 | 이야기 없이 설정만 나열 |
| 사회 풍자만 | 캐릭터/서사 없이 메시지만 |
| 과학적 모순 | 자체 설정 내에서도 일관성 부재 |
| 인간 부재 | 기술은 흥미롭지만 감정이입할 인물 없음 |

## 4. SF 세계관 심화

### 4.1 기술의 사회적 파급 설계

**[필수]** 하나의 기술을 도입하면 7가지 영역의 변화를 추적:

| 영역 | 질문 |
|------|------|
| 경제 | 누가 이익을 보고 누가 손해를 보는가? |
| 정치 | 권력 구조가 어떻게 변하는가? |
| 군사 | 전쟁/안보에 어떤 영향? |
| 문화 | 일상 생활/관습이 어떻게 바뀌는가? |
| 윤리 | 어떤 새로운 도덕적 질문이 생기는가? |
| 법률 | 어떤 새로운 법/규제가 필요한가? |
| 심리 | 인간의 자아/관계가 어떻게 변하는가? |

### 4.2 AI/로봇 서사 설계

현대 SF의 핵심 소재:

| 주제 | 핵심 질문 | 서사 접근 |
|------|----------|----------|
| AI 의식 | "기계가 자아를 가질 수 있는가?" | 튜링 테스트, 감정 시뮬레이션 |
| 인간-AI 관계 | "AI를 사랑할 수 있는가?" | 감정의 진정성 탐구 |
| AI 위협 | "AI가 인간을 대체하는가?" | 공존 vs 지배 |
| 디지털 존재 | "업로드된 의식은 '나'인가?" | 정체성, 연속성 |
| AI 윤리 | "AI의 행동에 누가 책임지는가?" | 제작자/사용자/AI 자신 |

### 4.3 과학적 그럴듯함 확보법

**하드 SF를 위한 고증 전략**:
1. 현재 과학의 한계를 정확히 파악 (논문/과학 매체)
2. "한 가지"만 현실과 다르게 가정 (그 외는 현실 법칙 준수)
3. 가정의 논리적 귀결을 끝까지 추적
4. 전문가 자문 또는 전문 포럼 검증

**소프트 SF의 "그럴듯함"**:
- 기술 원리 대신 **사회적 영향**에 초점
- "어떻게 작동하는가"보다 "어떤 세계를 만드는가"
- 내적 일관성만 유지하면 과학적 세부는 생략 가능

### 4.4 SF 신흥 하위 장르

| 장르 | 특징 | 핵심 |
|------|------|------|
| **솔라펑크** | 친환경 기술 + 낙관적 미래 | 기술로 자연과 공존 |
| **바이오펑크** | 유전 공학 + 생명 조작 | 생명의 정의와 경계 |
| **사이언스 판타지** | SF + 마법/초자연 결합 | 장르 경계 해체 |
| **CLI-Fi** | 기후 변화 중심 SF | 환경 재앙과 적응 |
| **프로그레션 SF** | SF 세계에서의 능력 성장 | 웹소설 시스템물 + SF |

## 5. SF 특유의 기법

### 5.1 네오로지즘 (신조어) 설계

SF 세계의 고유 용어 만들기:

| 전략 | 예시 | 효과 |
|------|------|------|
| 합성어 | 사이버스페이스, 다크웹 | 직관적 이해 |
| 약어 | AI, VR, AR | 기술적 느낌 |
| 기존어 전용 | "리부트" (인간 재생을 지칭) | 익숙함+낯설음 |
| 완전 신조어 | 앤시블(ansible) | 세계 고유의 느낌 |

**[팁]** 한 작품에 완전 신조어는 3~5개 이하. 초과 시 독자 부담.

### 5.2 SF 묘사의 특징

```
나쁜 예: "그는 우주선을 타고 화성에 갔다."
→ SF적 감각 없음

좋은 예: "가속이 시작되자 좌석이 등을 밀었다.
창 밖으로 지구가 줄어드는 동안, 작별 인사를 할 시간도 없이
화성의 붉은 점이 커지기 시작했다."
→ 감각 + 감정 + SF적 환경
```
