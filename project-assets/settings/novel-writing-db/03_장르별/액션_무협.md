# 액션/무협 장르

> 전투와 대결의 미학. 물리적 긴장감과 정신적 대립의 조화.
> 전투 설계, 파워 밸런스, 무협 특화 패턴, 인기/비인기 분석.

---

## 1. 하위 장르 분류

### 1.1 전통 무협
- **특징**: 중국/동양 무술 세계, 강호(江湖), 문파 체계
- **필수**: 무공 체계, 내공/외공, 무림 맹주, 사제 관계
- **대표**: 김용 작품군, 화산귀환

### 1.2 현대 액션
- **특징**: 현실 배경, 총기/격투/첩보
- **필수**: 리얼리즘(무기/전술), 긴장감, 시간 압박
- **대표**: 잭 리처, 본 시리즈

### 1.3 능력 배틀
- **특징**: 초능력/마법 기반 전투, 능력 상성
- **필수**: 능력 체계, 전략적 전투, 능력 활용의 창의성
- **대표**: 헌터x헌터, 주술회전, 나루토

### 1.4 던전/헌터물
- **특징**: 던전 탐사, 몬스터 전투, 시스템
- **필수**: 레벨/등급 체계, 보스전, 보상 시스템
- **대표**: 나혼자만 레벨업, 튜토리얼이 너무 어렵다

### 1.5 밀리터리/전쟁물
- **특징**: 전쟁/군사 작전, 부대 규모 전투
- **필수**: 전략/전술, 지휘 체계, 전장의 혼란
- **대표**: 은하영웅전설, 킹덤

## 2. 전투 장면 설계

### 2.1 전투의 4층 구조

| 층 | 내용 | 역할 |
|----|------|------|
| **동기층** | 왜 싸우는가 | 감정적 투자 |
| **전략층** | 어떤 계획으로 | 지적 재미 |
| **전술층** | 어떻게 싸우는가 | 시각적 흥분 |
| **감정층** | 싸우면서 무엇을 느끼는가 | 감정적 공명 |

**[필수]** 4층 모두 존재 = 명장면. 전술층만 = 게임 실황.

### 2.2 전투 리듬

**전투 전**:
```
긴장 고조 (느린 페이싱)
→ 적 등장/대치 (정적)
→ 전투 준비/각오 (내면 묘사)
```

**전투 중**:
```
초반 교전 (빠름) → 형세 분석 (중간) → 위기 (매우 빠름)
→ 전환점 (느림→빠름) → 결정타 (극한 가속) → 결말 (정지)
```

**전투 후**:
```
결과 확인 (느림) → 피해 점검 → 감정 정리 → 의미 부여
```

### 2.3 전투 묘사 기법

**가속 기법**:
- 짧은 문장, 3~5어절
- 의성어/의태어 활용
- 문장 간 인과 생략 → 독자가 보충

```
검이 스쳤다. 피가 튀었다. 뒤로 물러났다.
→ 어디를 베었는지, 얼마나 깊은지 안 말함 — 독자가 상상
```

**슬로모션 기법**:
- 결정적 순간을 느리게 확대
- 감각 묘사 삽입

```
시간이 느려진 것 같았다. 칼끝이 다가오는 것이 보였다.
빛이 칼날을 타고 흘렀다. 그는 목을 틀었다. 머리카락 한 올이 잘렸다.
```

**시점 제한 기법**:
- 전투 참가자의 제한된 시야로 묘사
- 안 보이는 곳에서 일어나는 일 = 소리/진동으로만 전달
- 혼란과 긴장감 극대화

## 3. 파워 밸런스 설계

### 3.1 파워 체계의 필수 요소

| 요소 | 역할 | 미비 시 문제 |
|------|------|------------|
| 등급/계층 | 강함의 기준 명시 | "얼마나 강한지 모르겠다" |
| 상성 | 상대적 유불리 | "주인공이 항상 이김" |
| 제약/대가 | 능력 사용의 비용 | "왜 안 쓰지?" 의문 |
| 천장 | 최강의 기준 | 능력 인플레 |
| 성장 경로 | 강해지는 방법 | "갑자기 강해짐" |

### 3.2 파워 인플레이션 방지

**[비인기]** 인플레 패턴:
```
1화: 도적 무리 → 10화: 기사단 → 30화: 드래곤 → 50화: 신
→ "다음 적은 뭐야? 우주?" → 긴장감 소실
```

**[인기]** 인플레 방지법:

| 방법 | 설명 |
|------|------|
| 질적 변화 | 양적 강화 대신 능력의 "종류"가 달라짐 |
| 제약 강화 | 강해질수록 대가가 커짐 |
| 상성 활용 | 강한 적도 특정 약점 존재 |
| 전장 변화 | 개인전→집단전→전략전으로 스케일 변화 |
| 인간적 위기 | 물리적으로 강해져도 감정적/도덕적 위기는 별개 |

### 3.3 전투 밸런스 황금 비율

| 결과 | 비율 | 효과 |
|------|------|------|
| 주인공 승리 | 40% | 사이다, 카타르시스 |
| 고전 끝 승리 | 30% | 긴장 + 성취 |
| 패배/후퇴 | 15% | 성장 동기, 스테이크 상승 |
| 무승부/중단 | 10% | 다음 대결 기대 |
| 제3자 개입 | 5% | 세계관 확장, 새 변수 |

## 4. 무협 특화 요소

### 4.1 무공 체계 설계

**기본 구조**:
```
내공(기의 총량/질) → 무공(기의 활용법) → 경지(경지에 의한 질적 변화)
```

**경지 체계 예시**:
```
무인 → 일류 → 초일류 → 화경 → 강호 초월 → 전설/신화급
```

**[인기]** 핵심: 경지 돌파가 단순 수치 상승이 아닌 "깨달음"과 연동

### 4.2 강호(江湖) 사회 설계

| 요소 | 설명 | 서사적 역할 |
|------|------|------------|
| 문파 | 무술 전승 집단 | 소속감, 충성 vs 자유 |
| 무림맹 | 정파 연합 | 권력 구조, 정치적 갈등 |
| 사파 | 비정파 세력 | 대립 구도, 도덕적 모호성 |
| 강호 규율 | 불문율 | 행동 제약, 갈등 원천 |
| 비급/보물 | 무공 전서, 신병기 | 갈등의 맥거핀 |

### 4.3 무협 인기/비인기 패턴

**[인기]**:
- 몰락 문파 재건 (화산귀환형)
- 숨겨진 고수의 세상 나들이
- 사제 간 갈등과 초월
- 강호의 이면/비밀 파헤치기

**[비인기]**:
- 맞기만 해도 내공 상승 → 논리 부재
- 무한 비급 수집 → 독창성 없음
- 여성 캐릭터가 보상/트로피 역할만

## 5. 액션 장면 인기/비인기 종합

### 5.1 인기 패턴

| 패턴 | 설명 |
|------|------|
| 전략적 승리 | 힘이 아닌 두뇌로 역전 |
| 합동 전투 | 파티원 각자의 역할 연계 |
| 의미 있는 패배 | 지지만 성장/각오의 계기 |
| 능력의 창의적 활용 | 기존 능력의 새로운 조합 |
| 대가 있는 승리 | 이기지만 무언가를 잃음 |

### 5.2 비인기 패턴

| 패턴 | 문제점 | 대안 |
|------|--------|------|
| 무쌍(원사이드) | 긴장 없음 | 리스크/대가 부여 |
| 설명 전투 | "이 기술은 ~이며..." 전투 중 설명 | 행동으로 보여주기 |
| 반복 패턴 | 매번 같은 방식 승리 | 상황/적 변화 |
| 주변인물 무력화 | 주인공만 활약 | 팀 전투/역할 분배 |
| 플롯 아머 | 죽을 상황에서 항상 살아남 | 패배/부상/퇴각 도입 |
