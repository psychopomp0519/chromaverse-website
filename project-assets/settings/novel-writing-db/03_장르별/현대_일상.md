# 현대/일상 장르

> 비범한 것 없이 독자를 붙잡는 기술. 일상의 디테일에서 의미를 발견하는 서사.
> 일상물, 직업물, 힐링물, 생활밀착형 드라마.

---

## 1. 하위 장르 분류

### 1.1 일상물/힐링물
- **특징**: 거대한 갈등 없이, 소소한 일상의 따뜻함
- **핵심**: 분위기, 관계, 작은 성취감
- **대표**: 은수저, 바라카몬, 논논비요리

### 1.2 직업물
- **특징**: 특정 직업의 세계를 깊이 파고드는 서사
- **핵심**: 전문성 + 인간 드라마 + 직업의 의미
- **대표**: 미생, 슬기로운 의사생활, 그랑블루

### 1.3 학원물
- **특징**: 학교 배경, 성장/우정/연애
- **핵심**: 청춘의 불확실성, 관계의 시작과 끝

### 1.4 가족 드라마
- **특징**: 가족 내 갈등과 화해
- **핵심**: 세대 간 차이, 사랑의 다양한 형태

### 1.5 생활 밀착 드라마
- **특징**: 사회 문제(취업, 주거, 돌봄 등)를 개인 서사로
- **핵심**: 공감 + 사회적 메시지

## 2. 일상물의 몰입 장치

### 2.1 큰 갈등 없이 독자를 붙잡는 법

**원리**: 거대한 위기 대신 **미세한 감정 변화와 관계 역학**으로 텐션 유지

| 장치 | 설명 | 예시 |
|------|------|------|
| 관계의 미세 변화 | 어제와 오늘의 사이가 미묘하게 다름 | 호칭이 바뀌는 순간 |
| 일상의 의식(儀式) | 반복되는 행위에 의미 부여 | 매일 아침 같은 카페 |
| 소소한 도전 | 작지만 의미 있는 목표 | 요리 성공, 발표 완수 |
| 계절감 | 시간의 흐름으로 변화 체감 | 벚꽃→여름→단풍→눈 |
| 공간의 변화 | 장소가 기분/관계를 반영 | 익숙한 곳이 달라 보이는 순간 |
| 우연의 발견 | 일상 속 작은 놀라움 | 몰랐던 동네 가게, 새로운 사람 |

### 2.2 디테일의 힘

**[인기]** 일상물의 경쟁력 = 디테일의 정확성

```
✅ "그는 편의점 도시락의 비닐 뚜껑을 절반만 벗기고 전자레인지에 넣었다.
   누르는 시간은 늘 2분 30초. 1초라도 더하면 밥이 질퍽해진다는 걸 알기 때문이다."
→ 이 묘사 하나로 캐릭터의 생활/성격/습관이 전달됨

❌ "그는 편의점에서 밥을 사 먹었다."
→ 아무 정보도, 감정도 없음
```

**디테일을 살려야 할 영역**:
- 음식 (맛, 온도, 식감)
- 공간 (소리, 빛, 냄새)
- 습관 (반복 행동의 의미)
- 계절 (온도, 옷차림, 자연)
- 감각 (촉감, 온기, 피로)

## 3. 직업물 설계

### 3.1 직업물의 3대 매력

| 매력 | 설명 |
|------|------|
| 전문성 | 몰랐던 세계를 알아가는 재미 |
| 성장 | 초보→전문가의 과정 |
| 철학 | "왜 이 일을 하는가"에 대한 탐구 |

### 3.2 직업물 집필 시 주의사항

**[필수]** 정확한 취재/고증:
- 직업 종사자의 일상 루틴
- 전문 용어의 적절한 사용 (과하지 않게)
- 업계의 관행, 암묵적 규칙
- 실제 직업의 고충과 보람

**[비인기]** 실패:
- 직업이 배경일 뿐 이야기와 무관 → "이 설정이 왜 필요해?"
- 전문성 과시형 나열 → 독자 이탈
- 비현실적 미화 → 종사자 독자가 이탈

## 4. 인기/비인기 패턴

### 4.1 인기 패턴

| 패턴 | 설명 |
|------|------|
| 성장 실감 | 작은 성취의 누적이 큰 변화 |
| 관계의 깊어짐 | 시간이 지나면서 자연스럽게 |
| 일상 속 비범 | 평범한 상황에서 특별한 의미 발견 |
| 공감의 디테일 | "나도 저래본 적 있어" 순간 |
| 시간의 무게 | 계절/세월의 흐름이 주는 감동 |

### 4.2 비인기 패턴

| 패턴 | 문제점 | 대안 |
|------|--------|------|
| 사건 없음 | 일상이 정말로 아무 일 없음 | 미세한 갈등/변화라도 배치 |
| 주인공 수동성 | 흘러가는 대로 살아감 | 작은 의지/선택이라도 |
| 설교적 메시지 | "삶이란 이런 것이다" 직접 서술 | 장면으로 보여주기 |
| 감정 과잉 | 사소한 일에 대한 과도한 감정 | 절제된 표현이 더 깊음 |

## 5. 하위 장르 심화

### 5.1 학원물

**핵심 매력**: 한정된 공간과 시간 안에서의 관계 밀도

| 요소 | 설계 |
|------|------|
| 공간 | 교실, 운동장, 동아리방 — 장소마다 다른 관계 역학 |
| 시간 | 학기/방학/졸업 — 자연스러운 시간 구조 |
| 이벤트 | 시험, 축제, 수학여행 — 서사 촉진 장치 |
| 성장 | 자기 발견, 꿈 탐색, 관계 형성/해체 |

**[인기]** 학원물의 감정적 강점: 졸업 = 자연스러운 이별/성장의 완결

### 5.2 가족 드라마

**핵심 매력**: 독자 누구나 "가족"이 있다 → 보편적 공감

| 테마 | 갈등 구조 |
|------|----------|
| 세대 갈등 | 부모의 기대 vs 자식의 꿈 |
| 형제 경쟁 | 비교/질투 → 이해/화해 |
| 돌봄 | 노부모/환자 간병의 고단함과 사랑 |
| 비밀 | 가족 간 감춰진 진실 → 폭로의 순간 |

### 5.3 사회파 소설

**핵심 매력**: 사회 문제를 개인 서사로 체험

**[필수]** 균형 원칙:
- 메시지가 캐릭터보다 앞서면 안 됨
- 사회 문제는 "배경"이 아니라 "캐릭터의 삶에 직접 영향"
- 독자가 문제의 당사자 시점을 경험하게 만드는 것이 핵심

| 소재 | 서사 접근 |
|------|----------|
| 취업난 | 구직자의 일상과 자존감 |
| 주거 문제 | 집을 찾는 과정이 곧 서사 |
| 차별 | 당사자의 일상적 경험을 섬세하게 |
| 돌봄 노동 | 감정 노동의 구체적 묘사 |

### 5.4 직업물 심화 — 직업별 가이드

| 직업 | 매력 포인트 | 고증 핵심 |
|------|-----------|----------|
| 의료 | 생사의 경계, 윤리적 딜레마 | 수술/진단 과정, 의료 용어 |
| 법조 | 진실 vs 법, 정의의 의미 | 재판 절차, 법 논리 |
| 요리 | 오감 자극, 창작의 기쁨 | 조리 과정, 식재료 지식 |
| 예술 | 자기 표현, 상업 vs 예술 | 창작 과정, 업계 관행 |
| IT/개발 | 문제 해결, 창조의 쾌감 | 기술 용어, 개발 문화 |
| 교육 | 성장의 보람, 교실의 현실 | 교육 현장의 일상 |

## 6. 현대물의 대화체

### 6.1 자연스러운 현대 한국어 대화

| 요소 | 설명 |
|------|------|
| 줄임말 | 세대/관계에 맞는 약어 사용 |
| 불완전한 문장 | 실제 대화는 완전한 문장이 아님 |
| 중첩/끊기 | 한 명이 말하는 도중 다른 사람이 끼어듦 |
| 침묵 | "..." 이 가장 강력한 대사일 때 |
| 비언어 | 톤, 속도, 시선이 대사의 의미를 바꿈 |

### 6.2 메신저/SNS 대화 표현

현대물에서 필수적인 디지털 소통 묘사:

```
표현 방법:
1. 채팅 형식으로 직접 표현
   [카이] 지금 어디야
   [코다] 집
   [코다] 왜
   [카이] 아무것도 아니야
   (읽음)

2. 서술 속에 녹이기
   코다에게 메시지를 보냈다. "지금 어디야." 답은 빠르게 돌아왔다.
   "집." 그리고 "왜." 두 글자짜리 답이 두 개. 별 의미 없다는 뜻이다.
```

## 7. 시간 경과와 계절감

### 7.1 시간의 흐름 표현

| 기법 | 예시 |
|------|------|
| 계절 변화 | "벚꽃이 질 때 만났던 사람과, 단풍이 물들 때 다시 마주쳤다" |
| 습관의 변화 | "언젠가부터 아침에 커피 대신 차를 마시게 되었다" |
| 공간의 변화 | "자주 가던 카페가 문을 닫았다" |
| 나이의 체감 | "작년에는 안 아프던 무릎이 시큰거렸다" |

### 7.2 계절별 묘사 소재

| 계절 | 시각 | 촉각 | 후각 | 청각 |
|------|------|------|------|------|
| 봄 | 벚꽃, 새싹, 황사 | 따스한 바람 | 꽃내음, 흙냄새 | 새소리 |
| 여름 | 입도선, 그늘, 소나기 | 습기, 땀 | 풀냄새, 아스팔트 | 매미, 천둥 |
| 가을 | 단풍, 은행잎, 짧은 해 | 서늘한 공기 | 볶은 밤, 낙엽 | 바스락 |
| 겨울 | 눈, 서리, 김 | 찬 공기, 핫팩 | 군밤, 귤 | 발소리 |

## 8. 체크리스트

- [ ] 디테일이 충분히 구체적인가? (감각, 습관, 공간)
- [ ] 미세한 갈등/변화가 매 회차 존재하는가?
- [ ] 캐릭터가 시간의 흐름 속에서 변화하는가?
- [ ] 직업/일상의 고증이 정확한가?
- [ ] "왜 이 이야기를 읽어야 하는가"에 대한 답이 있는가?
- [ ] 현대 대화체가 자연스러운가? (줄임말, 불완전 문장)
- [ ] 계절감/시간 경과가 서사에 녹아 있는가?
- [ ] 사회적 소재를 다룬다면 메시지가 캐릭터보다 앞서지 않는가?
