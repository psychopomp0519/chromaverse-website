# 장르 이탈 (드리프트)

> 웹소설 최대 논란 중 하나. 독자가 기대한 장르에서 이탈하면 신뢰가 무너진다.
> 장르 이탈의 유형, 원인, 방지법.

---

## 1. 장르 이탈이란?

**정의**: 작품의 내용 성격이 연재 도중 기존과 달라지는 현상.

웹소설은 제목/태그/시놉시스/초반 전개로 독자와 **암묵적 계약**을 맺음:
- "이 작품은 판타지 성장물입니다"
- "이 작품은 로맨스 판타지입니다"

→ 이 계약을 깨면 독자가 배신감을 느끼고 이탈

---

## 2. 장르 이탈 유형

### 2.1 소프트 드리프트 (점진적)
- 서서히 장르 비중이 변화
- 예: 판타지 액션물 → 점점 정치극/로맨스 비중 증가
- **독자 반응**: "이거 원래 이런 소설이었나?"

### 2.2 하드 드리프트 (급격한)
- 갑자기 다른 장르가 됨
- 예: 일상 힐링물 → 갑자기 배틀로얄/데스게임
- **독자 반응**: "이게 무슨 소설이야?"

### 2.3 톤 드리프트
- 장르는 같지만 분위기/톤이 급변
- 예: 밝은 모험물 → 갑자기 다크하고 잔혹
- **독자 반응**: "분위기가 왜 이래?"

### 2.4 센서티브 드리프트
- 민감한 요소가 갑자기 등장
- 예: 일반 로맨스 → 갑자기 NTR/불륜/고어
- **독자 반응**: 가장 극단적 반발, 별점 테러

---

## 3. 장르 이탈의 원인

| 원인 | 설명 |
|------|------|
| 작가 취향 변화 | 연재 중 쓰고 싶은 것이 변함 |
| 소재 고갈 | 초기 소재가 바닥나 다른 소재 도입 |
| 수익 추구 | 유행 장르로 급선회 |
| 역량 부족 | 해당 장르를 끝까지 유지할 실력 부족 |
| 세계 확장 | 세계관이 커지면서 자연스럽게 장르 범위 확대 |

---

## 4. 장르 이탈 방지법

### 4.1 사전 설계
- 연재 전 전체 아크 설계 → 장르 일관성 사전 확보
- 시놉시스/기획 단계에서 "이 작품은 끝까지 ○○ 장르"임을 확정

### 4.2 장르 문법 유지
- **[필수]** 주 장르의 핵심 요소가 매 10화마다 최소 1번 등장:
  - 액션물 → 전투
  - 로맨스 → 설렘/관계 진전
  - 미스터리 → 단서/추리
  - 성장물 → 성장 실감

### 4.3 부장르 비중 관리
```
주 장르 60~70% / 부 장르 30~40%
→ 이 비율이 역전되면 장르 이탈
```

### 4.4 변화가 필요할 때의 안전한 방법
1. **점진적 전환**: 20화 이상에 걸쳐 서서히
2. **사전 복선**: 다른 장르 요소를 미리 조금씩 노출
3. **주 장르 유지**: 다른 요소가 추가되더라도 주 장르는 유지
4. **독자 소통**: 작가의 말/태그로 사전 안내

---

## 5. 장르 이탈 vs 장르 확장

| 이탈 (나쁨) | 확장 (좋음) |
|------------|------------|
| 기존 장르 요소가 사라짐 | 기존 장르 요소 유지 + 새 요소 추가 |
| 갑작스러운 변화 | 점진적, 복선 있는 변화 |
| 독자 기대 배신 | 독자 기대를 넘어서는 확장 |
| 주 장르 비율 역전 | 주 장르 비율 유지 |

---

## 6. 웹소설 특화 민감 요소

### 6.1 남성향 웹소설의 주요 드리프트 지뢰

| 요소 | 독자 반응 |
|------|----------|
| 갑작스러운 로맨스 과다 | "이거 로맨스 소설 아니었는데?" |
| NTR/네토라레 | 극단적 반발, 별점 테러 |
| 먼치킨 → 패배주의 | "왜 갑자기 약해짐?" |
| 히로인 비중 급감 | "어디 갔어?" |

### 6.2 여성향 웹소설의 주요 드리프트 지뢰

| 요소 | 독자 반응 |
|------|----------|
| 남주 교체/삼각관계 장기화 | "누구 편이야?" |
| 갑작스런 잔혹/고어 | "이런 소설이 아니었는데" |
| 로맨스 실종 → 정치극만 | "연애는 어디 갔어?" |
| 여주의 능동성 상실 | "왜 갑자기 수동적?" |
