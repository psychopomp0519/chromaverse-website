# 독자 이탈 요인

> 독자가 작품을 버리는 결정적 순간들. 이탈 시점별/원인별 분석.

---

## 1. 이탈 시점별 분석

### 1.1 즉시 이탈 (1화 이내)

| 원인 | 빈도 | 대응 |
|------|------|------|
| 설정 폭격으로 시작 | 극높음 | 사건/행동으로 시작 |
| 주인공이 무개성 | 높음 | 1화에 강렬한 개성 제시 |
| 문장이 읽기 어려움 | 높음 | 가독성 검토, 맞춤법 |
| "뻔하다"는 첫인상 | 중간 | 장르 문법 이행하되 변주 |
| 오프닝이 지루 | 중간 | 3줄 훅 삽입 |

### 1.2 초반 이탈 (2~10화)

| 원인 | 빈도 | 대응 |
|------|------|------|
| 이야기 진행이 없음 | 극높음 | 매 회차 변화/진전 |
| 캐릭터에 공감 불가 | 높음 | 결핍/노력/유머로 공감 구축 |
| 세계관 이해 불가 | 중간 | 정보를 단계적으로 |
| 첫 사이다 부재 | 중간 | 3~5화 내 첫 카타르시스 |

### 1.3 중반 이탈 (20~80화)

| 원인 | 빈도 | 대응 |
|------|------|------|
| 반복적 전개 | 극높음 | 새로운 변수/스케일 변화 |
| 고구마만 축적 | 높음 | 10화 이내 사이다 보장 |
| 캐릭터 정체 | 높음 | 새 성장 아크 시작 |
| 관심 없는 서브플롯 과다 | 중간 | 서브플롯 정리/통합 |
| 파워 밸런스 붕괴 | 중간 | 새 유형의 위협 도입 |

### 1.4 후반 이탈 (80화+)

| 원인 | 빈도 | 대응 |
|------|------|------|
| 늘어지는 전개 | 극높음 | 완결을 향한 가속 |
| 떡밥 미회수 | 높음 | 회수 일정 관리 |
| 캐릭터 퇴화 | 중간 | 성장 일관성 유지 |
| 새 캐릭터 과다 투입 | 중간 | 기존 캐릭터 활용 |
| 완결 품질 저하 | 중간 | 완결부 사전 설계 |

## 2. 원인별 심층 분석

### 2.1 "답답하다" (고구마 과다)

**독자 심리**: "왜 안 말해?" "왜 안 해?" "언제까지?"

**발생 조건**:
- 오해 기반 갈등이 5화 이상 지속
- 해결할 수 있는데 안 하는 것처럼 보임
- 주인공이 수동적 (당하기만)

**해결**:
- 고구마에 "합리적 이유" 부여 (말하면 더 위험한 상황)
- 고구마 중에도 작은 진전 보여주기
- 5화 이내 최소한의 해소

### 2.2 "뻔하다" (예측 가능)

**독자 심리**: "다음에 뭐 일어날지 다 알겠어"

**발생 조건**:
- 장르 클리셰를 무변주 사용
- 복선이 너무 노골적
- 같은 패턴 반복 (도전→위기→극복→보상)

**해결**:
- 예상 가능한 전개에 예상 못한 방식 결합
- 독자 예측을 역이용 (일부러 예측하게 → 반전)
- 패턴 변형 (매번 다른 방식의 극복)

### 2.3 "이해가 안 된다" (논리 파탄)

**독자 심리**: "왜 갑자기?" "앞에서 이랬잖아" "말이 안 돼"

**발생 조건**:
- 캐릭터 행동이 기존 성격과 불일치
- 설정 간 모순
- 복선 없는 반전

**해결**:
- 설정집 교차 검증
- 캐릭터 행동의 동기 명시
- 모든 반전에 사전 복선 최소 2개

### 2.4 "재미없다" (텐션 부재)

**독자 심리**: "아무 일도 안 일어나" "그래서?"

**발생 조건**:
- 갈등이 없거나 스테이크가 낮음
- 정보/설명만 있고 사건이 없음
- 캐릭터 간 상호작용이 단조로움

**해결**:
- 매 장면에 최소 하나의 갈등/질문
- 정보는 갈등 속에서 전달
- 대화에 서브텍스트와 긴장 삽입

## 3. 이탈 방지 핵심 원칙

### 3.1 "3화 생존" 법칙
- 독자의 70%가 3화 이내에 계속/중단 결정
- 1화: 훅 + 주인공 매력 + 목표 제시
- 2화: 갈등 심화 + 세계 확장
- 3화: 첫 카타르시스 + 장기 목표 암시

### 3.2 "10화 약속" 법칙
- 10화까지 작품의 핵심 매력을 모두 보여줘야 함
- 전투물이면 최고의 전투 하나
- 로맨스면 강력한 설렘 장면 하나
- 미스터리면 놀라운 단서/반전 하나

### 3.3 "50화 갱신" 법칙
- 50화마다 새로운 요소 투입 (적/동료/목표/세계)
- 같은 패턴이 50화 이상 지속되면 독자 이탈 급증
- 스케일/상황/관계의 질적 변화 필수
