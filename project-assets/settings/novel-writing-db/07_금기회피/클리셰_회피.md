# 클리셰 회피

> 클리셰 자체가 적이 아니라, 무사유(無思惟)한 사용이 적이다.
> 장르별 주요 클리셰, 대안, 역이용법.

---

## 1. 범장르 클리셰

### 1.1 캐릭터 클리셰

| 클리셰 | 문제점 | 대안/변주 |
|--------|--------|----------|
| 완벽한 주인공 | 긴장감 없음, 공감 불가 | 결함+대가 부여 |
| "평범한 학생"이 사실은 천재 | 과용됨 | 진짜 평범하되 특수한 상황에서 빛남 |
| 부모 부재 (고아/사망) | 너무 흔함 | 부모가 있되 복잡한 관계 |
| 순수한 소녀/소년 | 비현실적 | 순수하되 경험에서 오는 지혜 |
| 미스터리한 전학생 | 과용됨 | 전학생이 미스터리가 아닌 평범, 미스터리는 다른 곳에 |

### 1.2 플롯 클리셰

| 클리셰 | 문제점 | 대안/변주 |
|--------|--------|----------|
| 예언/선택받은 자 | 뻔한 운명 | 예언이 틀림/다중 해석 가능 |
| 트레이닝 몽타주 | 성장 과정 생략 | 훈련 자체에 갈등/사건 배치 |
| "사실 알고 보니" 반전 | 과용됨 | 단서를 충분히 깔고 공정한 반전 |
| 친구의 배신 | 예측 가능 | 배신자가 예상 밖 인물, 또는 배신이 아닌 오해 |
| 최종 보스 2형태 | 게임적, 진부 | 최종 대결이 물리전이 아닌 다른 형태 |

### 1.3 설정 클리셰

| 클리셰 | 문제점 | 대안/변주 |
|--------|--------|----------|
| 중세 유럽풍 세계 | 너무 흔함 | 다른 시대/문화 기반 |
| 4원소 마법 체계 | 새로움 없음 | 독자적 마법 원리 |
| 엘프/드워프/오크 | 톨킨 복사 | 종족 특성에 반전 부여 |
| 모험자 길드 | 게임적 | 길드의 사회적 기능/문제점 탐구 |
| 마왕 | 최종 보스=악 | 마왕의 사정/동기 깊이 파기 |

## 2. 장르별 클리셰

### 2.1 판타지

| 클리셰 | 역이용법 |
|--------|---------|
| 신비로운 검/무기 획득 | 무기가 저주를 품고 있거나 대가를 요구 |
| 왕국의 위기 | 왕국이 이미 망한 후에서 시작 |
| 마법학교 | 학교가 사기/사교 집단인 반전 |
| 드래곤 퇴치 | 드래곤이 사실 피해자 |

### 2.2 로맨스

| 클리셰 | 역이용법 |
|--------|---------|
| 차도남이 여주에게만 따뜻 | 따뜻한 남주가 여주에게만 벽을 세움 |
| 삼각관계 | 삼각이 자연스럽게 해소 (깨끗하게) |
| 정략결혼→진심 | 이미 사랑하지만 정략이 방해 |
| 구원→사랑 | 구원받는 쪽이 먼저 좋아함 |

### 2.3 시스템/회귀물

| 클리셰 | 역이용법 |
|--------|---------|
| 최약 → 최강 | 시작이 강하고 점점 약해지는 구조 |
| 숨겨진 던전/스킬 | 획득한 스킬이 저주/양날의 검 |
| 미래 지식으로 투자 | 투자가 나비효과로 예상 밖 결과 |
| 시스템 창 | 시스템이 거짓말을 하거나 목적이 악의적 |

## 3. 클리셰 관리 원칙

### 3.1 클리셰 사용의 3단계 판단

```
1단계: "이 클리셰가 이야기에 필수적인가?"
   → 아니면 삭제

2단계: "독자가 예상하는 방향은?"
   → 그 방향을 한 단계만 비틀기

3단계: "같은 효과를 내는 다른 방법은?"
   → 있으면 교체, 없으면 실행 품질로 승부
```

### 3.2 클리셰를 살리는 법

| 방법 | 설명 |
|------|------|
| 자각 | 캐릭터가 클리셰를 인지 ("이건 소설이면 함정인데") |
| 결과 변형 | 같은 시작, 예상 밖 결과 |
| 맥락 변형 | 같은 패턴, 다른 문화/배경 |
| 감정 변형 | 같은 사건, 다른 감정적 경험 |
