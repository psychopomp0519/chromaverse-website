# 작가 생존과 수익 구조

> 좋은 소설을 쓰는 것만으로는 생존할 수 없다.
> 작가로서 지속 가능한 경력을 만드는 실무 지식.

---

## 1. 수익 모델 이해

### 1.1 웹소설 수익 구조

| 모델 | 설명 | 작가 수익 |
|------|------|----------|
| **편당 결제** | 독자가 1화당 결제 | 판매가의 30~70% (플랫폼별 상이) |
| **기다리면 무료** | 일정 기간 후 무료 전환 | 초기 유입 후 편당 결제 전환 |
| **미리보기** | 최신 N화만 유료 | 연재 속도가 수익에 직결 |
| **구독제** | 월정액으로 무제한 읽기 | 조회수/완독률 기반 정산 |
| **MG (선급금)** | 계약 시 선지급 | 보장 수익, 추가 정산은 MG 초과분 |

### 1.2 수익 규모 현실

| 단계 | 월 수익 (추정) | 비고 |
|------|-------------|------|
| 입문 (연재 시작) | 0~10만원 | 대부분 무수익 |
| 하위권 | 10~50만원 | 부업 수준 |
| 중위권 | 50~300만원 | 전업 가능 경계 |
| 상위권 | 300~1,000만원 | 안정적 전업 |
| 최상위 | 1,000만원+ | 극소수 |

**[필수]** 현실 직시: 웹소설 작가의 80%는 최저임금 이하의 수익

### 1.3 단행본/문학 수익 구조

| 모델 | 설명 |
|------|------|
| **인세** | 판매가의 8~15% (저자) |
| **선인세 (어드밴스)** | 출판 전 선지급, 인세에서 차감 |
| **판권료** | 영상화/웹툰화 시 별도 계약 |
| **공모전 상금** | 1회성 수입 |
| **강연/칼럼** | 부가 수입원 |

## 2. 계약 기초

### 2.1 웹소설 계약 유형

| 유형 | 특징 | 장점 | 단점 |
|------|------|------|------|
| **전속 계약** | 해당 플랫폼에만 연재 | MG 보장, 프로모션 지원 | 다른 플랫폼 사용 불가 |
| **비전속** | 여러 플랫폼 동시 연재 가능 | 자유도 | MG 없음, 프로모션 약함 |
| **CP 계약** | 콘텐츠 프로바이더를 통해 | 복수 플랫폼 유통 대행 | 수수료 발생 |

### 2.2 계약 시 확인 사항

| 항목 | 확인 포인트 |
|------|-----------|
| **저작권 귀속** | 1차 저작권은 반드시 작가에게 |
| **2차 저작물 권리** | 웹툰화/영상화 수익 배분 비율 |
| **계약 기간** | 자동 갱신 조건, 해지 조건 |
| **MG 조건** | 지급 시점, 미달 시 반환 여부 |
| **독점 범위** | 어떤 플랫폼/매체까지 독점인지 |
| **완결 의무** | 연재 중단 시 페널티 |

**[필수]** 계약서는 반드시 꼼꼼히 읽고, 가능하면 전문가 자문

### 2.3 2차 저작물 (IP 확장)

| 유형 | 일반적 조건 |
|------|-----------|
| 웹툰화 | 원작료 + 매출 배분 (보통 원작 5~15%) |
| 드라마/영화화 | 판권 계약 (1,000만~수억, 작품별 상이) |
| 게임화 | 판권 + 감수 참여 |
| 번역 출판 | 해외 인세 (보통 국내와 유사 구조) |

## 3. 지속 가능한 작가 생활

### 3.1 전업 작가 전환 조건

| 조건 | 기준 |
|------|------|
| 안정 수입 | 최소 6개월 연속 생활비 이상 수익 |
| 버퍼 | 생활비 6개월분 저축 |
| 작품 파이프라인 | 현재작 + 다음작 기획 완료 |
| 건강 관리 | 번아웃 대비 루틴 확립 |

### 3.2 번아웃 예방

| 전략 | 실행 |
|------|------|
| **분량 관리** | 무리한 연재 주기 피하기 (주 3~5회) |
| **휴식 계획** | 아크 사이 1~2주 계획된 휴재 |
| **운동/건강** | 앉아서 작업하는 직업 → 의식적 운동 |
| **사회적 연결** | 동료 작가/커뮤니티 교류 |
| **창작 외 시간** | 독서, 여행, 경험 → 소재의 원천 |

### 3.3 커리어 다각화

| 활동 | 효과 |
|------|------|
| 복수 작품 연재 | 수익 분산, 한 작품 부진 시 대비 |
| 출판 + 웹소설 병행 | 다른 독자층 확보 |
| 강의/멘토링 | 부가 수입 + 네트워킹 |
| IP 확장 참여 | 웹툰/드라마 감수 → 원작료 외 수입 |
| 해외 시장 진출 | 번역 출판, 해외 플랫폼 |

## 4. 작가 커뮤니티와 네트워킹

### 4.1 커뮤니티 활용

| 커뮤니티 | 특성 |
|----------|------|
| 웹소설 연재 갤러리 | 현직 작가 정보 교류 (주의: 분위기 험악) |
| 작가 카페/디스코드 | 동료 작가 간 피드백/응원 |
| 출판사 워크숍 | 편집자와의 접점 |
| 문학 교실/강좌 | 체계적 학습 + 네트워크 |

### 4.2 네트워킹의 가치

- 같은 장르 작가와의 교류 → 시장 정보 공유
- 편집자/PD와의 인맥 → 기회 확대
- 선배 작가의 조언 → 시행착오 감소
- 동료 작가의 베타리딩 → 작품 품질 향상
