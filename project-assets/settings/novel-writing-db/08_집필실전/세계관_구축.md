# 세계관 구축 (범용 Worldbuilding)

> 세계관은 "설정의 양"이 아니라 "규칙의 일관성"이다.
> 독자가 "이 세계에 살고 싶다/두렵다"고 느끼면 성공.

---

## 1. 세계관의 5대 축

### 1.1 물리/자연 환경

| 요소 | 설계 질문 |
|------|----------|
| 지형/지리 | 어떤 대륙/도시/지역이 있는가? 이동 수단은? |
| 기후/계절 | 현실과 같은가? 다른 규칙이 있는가? |
| 자원 | 무엇이 풍부하고 무엇이 희귀한가? |
| 생태계 | 어떤 동식물이 있는가? 인간과의 관계는? |
| 물리 법칙 | 현실과 같은가? 마법/기술로 변형되는가? |

### 1.2 사회/문화 체계

| 요소 | 설계 질문 |
|------|----------|
| 정치 | 어떤 권력 구조? 누가 통치? 갈등은? |
| 경제 | 화폐 체계? 교역? 빈부격차? |
| 계층/계급 | 어떤 기준으로 나뉘는가? 이동 가능한가? |
| 교육 | 어떻게 지식을 전달? 문맹률? 학교? |
| 풍습/일상 | 인사법, 식사, 의복, 주거, 여가 |
| 언어 | 공용어? 방언? 금기어? 호칭 체계? |
| 예술/문화 | 음악, 문학, 건축의 특성 |

### 1.3 힘의 체계 (마법/기술/초능력)

| 요소 | 설계 질문 |
|------|----------|
| 원천 | 힘의 근원은 무엇인가? |
| 규칙 | 무엇이 가능하고 불가능한가? |
| 대가 | 힘을 쓰면 무엇을 잃는가? |
| 계층 | 힘의 등급/수준 구분은? |
| 분포 | 누가 힘을 가지는가? 희귀한가? |
| 사회적 영향 | 힘이 사회 구조에 어떤 영향? |

**샌더슨의 3법칙** (모든 힘의 체계에 적용):
1. **1법칙**: 마법으로 문제를 해결하는 능력은, 독자가 그 마법을 이해하는 정도에 비례
2. **2법칙**: 한계와 약점이 힘 자체보다 흥미로움
3. **3법칙**: 새로운 마법을 추가하기 전에 기존 마법을 확장하라

### 1.4 역사/신화

| 요소 | 설계 질문 |
|------|----------|
| 창세 | 이 세계는 어떻게 시작되었는가? |
| 전환점 | 역사를 바꾼 결정적 사건들은? |
| 전쟁/갈등 | 과거의 대립과 그 유산은? |
| 인물 | 전설적 인물들과 그 영향은? |
| 잃어버린 것 | 잊힌 기술/문명/진실은? |

### 1.5 종교/신앙/철학

| 요소 | 설계 질문 |
|------|----------|
| 신/초월적 존재 | 실재하는가? 영향력은? |
| 종교 조직 | 구조, 권력, 사회적 역할 |
| 신앙과 현실 | 신앙이 일상에 미치는 영향 |
| 이단/갈등 | 종교 간 갈등? 세속과의 갈등? |
| 사후/내세 | 죽음에 대한 관념? |

## 2. 세계관 구축 방법론

### 2.1 빙산 이론

```
수면 위 (독자에게 보이는 것) — 10~20%
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
수면 아래 (작가만 아는 것) — 80~90%
```

**[필수]** 작가는 100을 알고 10~20만 보여준다
- 수면 아래 설정이 있어야 보이는 부분이 "깊이 있게" 느껴짐
- 수면 아래가 없으면 독자가 "세계가 얕다"고 느낌

### 2.2 세계관 도입 전략

**[금기]** 절대 하면 안 되는 것:
```
"이 세계는 크로마계라 불리는 곳으로, 빛과 색의 에너지인
프리즈마가 만물을 구성하는 세계이다. 프리즈마는 크게
루미나스와 쿠로겐으로 나뉘며..." → 설정 덤프
```

**[인기]** 올바른 도입법:

| 방법 | 설명 | 예시 |
|------|------|------|
| **일상 노출** | 인물의 일상 행동에 설정을 녹임 | "아침 측정소에 줄을 선다" → 측정이 일상 |
| **사건 체험** | 사건을 통해 규칙을 보여줌 | 마법 사고가 나서 규칙을 체감 |
| **대화 삽입** | 인물 간 대화에 자연스럽게 | "오늘 크로마 수치가 높대" |
| **갈등 발생** | 규칙 위반/한계로 갈등이 생김 | 규칙 때문에 원하는 것을 못함 |
| **신참자 시점** | 세계를 모르는 인물이 배워감 | 이세계물의 기본 장치 |

**용어 도입 속도**:
| 시점 | 도입량 | 방식 |
|------|--------|------|
| 1~5화 | 핵심 3~5개 | 일상 노출, 반복 |
| 6~20화 | 10~15개 추가 | 사건/대화 |
| 21~50화 | 필요 시 추가 | 갈등/발견 |

### 2.3 장르별 세계관 핵심

| 장르 | 세계관의 핵심 | 필수 설정 |
|------|-------------|----------|
| 판타지 | 마법 체계 + 종족 | 힘의 규칙, 대가, 계층 |
| SF | 기술 + 사회 변화 | "What if?" 답변, 기술의 한계 |
| 현대물 | 실재하는 세계의 디테일 | 직업, 장소, 사회 규범 |
| 역사물 | 시대 고증 + 허구 | 실재/허구의 경계 명시 |
| 호러 | 일상의 균열 | 공포의 규칙 (알려진/미지의) |
| 이세계물 | 현실↔이세계 차이 | 이세계의 독자적 논리 |

## 3. 세계관 일관성 관리

### 3.1 일관성의 3원칙

1. **내적 일관성**: 세계 내부의 규칙이 모순 없이 작동
2. **감정 일관성**: 세계의 분위기/톤이 일관적
3. **논리 일관성**: 원인→결과의 사슬이 끊기지 않음

### 3.2 일관성 깨지는 순간들

| 문제 | 원인 | 예방 |
|------|------|------|
| 마법이 때에 따라 되고 안 됨 | 규칙 미정의 | 사전에 가능/불가능 명시 |
| 거리/시간 불일치 | 지리 설정 부재 | 지도 + 이동 시간표 작성 |
| 인물 능력 불일치 | 파워 밸런스 미설계 | 능력치 + 성장 곡선표 |
| 사회 제도와 행동 불일치 | 문화 설정 부족 | 풍습/법률 기본안 |
| 경제적 모순 | 경제 체계 미고려 | 화폐/물가/직업 기본안 |

### 3.3 세계관 관리 도구

**설정집 구조** (권장):
```
1. 개요 (1페이지) — 한눈에 보는 세계
2. 지리/환경 — 지도, 지역 설명
3. 힘의 체계 — 규칙, 등급, 한계
4. 사회/정치 — 국가, 조직, 계층
5. 역사 — 연표, 핵심 사건
6. 종교/문화 — 신앙, 풍습
7. 경제 — 화폐, 교역, 자원
8. 용어 사전 — 확정 용어 + 금지 용어
부록: 캐릭터 관련 세계관 연결표
```

**연표 작성법**:
```
| 시점 | 사건 | 영향 | 작중 언급 |
|------|------|------|----------|
| 0년 | 창세 | 세계 형성 | 신화로 |
| 500년 | 대전쟁 | 국경 변화 | 역사서 |
| 780년 | 소설 시점 | 현재 | 직접 |
```

## 4. 세계관 깊이의 레벨

| 레벨 | 설명 | 적합 작품 |
|------|------|----------|
| **Lv1. 배경** | 이야기가 벌어지는 장소 설정만 | 단편, 일상물 |
| **Lv2. 규칙** | 세계의 고유한 규칙이 플롯에 영향 | 대부분의 장르소설 |
| **Lv3. 생태계** | 규칙이 사회/문화/경제까지 영향 | 중장편 판타지/SF |
| **Lv4. 역사** | 과거가 현재를 설명하고, 현재가 미래를 암시 | 장편, 시리즈물 |
| **Lv5. 살아있는 세계** | 주인공 없이도 세계가 돌아감 | 대하 서사, 서사시 |

**[팁]** 대부분의 웹소설은 Lv2~3이면 충분. Lv4~5는 대장편에만 필요.

## 5. 체크리스트

### 5.1 기본 체크

- [ ] 5대 축(환경/사회/힘/역사/종교) 중 최소 3개가 설정되었는가?
- [ ] 힘의 체계에 명확한 규칙과 대가가 있는가?
- [ ] 세계관 도입이 설정 덤프 없이 자연스러운가?
- [ ] 설정집 또는 용어 사전이 관리되고 있는가?
- [ ] 연표가 작성되어 있는가?

### 5.2 일관성 체크

- [ ] 마법/기술의 가능/불가능이 일관되는가?
- [ ] 지리적 거리와 이동 시간이 일관되는가?
- [ ] 사회 제도와 인물 행동이 일관되는가?
- [ ] 경제적 설정(물가, 보상, 비용)이 현실적인가?
- [ ] 용어가 전체 원고에서 통일되어 있는가?
