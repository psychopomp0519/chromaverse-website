# 퇴고와 편집

> "초고의 의무는 존재하는 것이고, 퇴고의 의무는 빛나게 하는 것이다."
> 퇴고 능력 = 작가의 진짜 실력.

---

## 1. 퇴고의 3회전 시스템 (심화)

### 1.1 1회전: 구조 퇴고 (거시적)

**읽기 방식**: 전체를 한 번에 통독. 세부 문장은 무시.

| 점검 항목 | 질문 |
|----------|------|
| 플롯 구멍 | 설명 안 된 사건, 잊힌 복선이 있는가? |
| 페이싱 | 지루한 구간이 3화 이상 계속되는가? |
| 캐릭터 아크 | 주인공이 시작과 끝에서 달라졌는가? |
| 불필요한 장면 | 삭제해도 이야기가 성립하는 장면은? |
| 일관성 | 설정/시간선/능력에 모순이 있는가? |

**구조 퇴고에서 할 수 있는 것**:
- 장면 순서 재배치
- 불필요한 회차 삭제/통합
- 아크 길이 조정 (늘리기/줄이기)
- 서브플롯 추가/제거

### 1.2 2회전: 장면 퇴고 (중시적)

**읽기 방식**: 1회차씩 꼼꼼히.

| 점검 항목 | 질문 |
|----------|------|
| 장면 목적 | 이 장면이 플롯/캐릭터/세계관에 기여하는가? |
| 장면 전환 | 이전/이후 장면과의 연결이 자연스러운가? |
| 감정선 | 감정의 변화가 자연스럽고 축적되는가? |
| 대화 기능 | 대화가 정보 전달/관계 변화/갈등에 기여하는가? |
| Show vs Tell | 감정을 직접 서술하지 않고 행동/대화로 보여주는가? |
| 정보 밀도 | 한 장면에 정보가 너무 많거나 적지 않은가? |

**Before/After 예시**:
```
Before (Tell):
"카이는 화가 났다. 이 상황이 너무 불공평하게 느껴졌다."

After (Show):
"카이의 손이 떨렸다. 주먹을 쥐자 손톱이 손바닥에 파고들었지만,
그 통증조차 지금은 아무렇지 않았다."
```

### 1.3 3회전: 문장 퇴고 (미시적)

**읽기 방식**: 한 문장씩, 소리 내어 읽기.

**삭제 후보**:
| 대상 | 이유 | 대안 |
|------|------|------|
| "매우/정말/너무" | 약한 강조, 대체 가능 | 구체적 행동으로 강도 표현 |
| "~것 같다" 남발 | 확신 부족, 문장 약화 | 단정으로 변경 가능 여부 확인 |
| "~하기 시작했다" | 불필요한 시작 동작 | "~했다"로 직접 진행 |
| 중복 정보 | 같은 말 2번 | 1번으로 축소 |
| 불필요한 수식어 | 문장 속도 저하 | 삭제 후 의미 변화 확인 |

**강화 후보**:
| 대상 | 방법 |
|------|------|
| 약한 동사 | "걸었다" → "성큼거렸다/비틀거렸다/서성였다" |
| 추상적 묘사 | "아름다웠다" → 구체적 감각으로 |
| 빈 대화 | "네" → 행동+대사 결합 |

**Before/After**:
```
Before:
"그는 매우 빠르게 뛰기 시작했다. 정말 다리가 아팠지만 멈출 수가 없었다."

After:
"뛰었다. 다리가 불탔다. 멈출 수 없었다."
→ 불필요한 수식어 제거, 짧은 호흡으로 긴박감 강화
```

## 2. 한국어 퇴고 체크포인트

### 2.1 문법/표기 체크

| 항목 | 흔한 실수 | 올바른 표현 |
|------|----------|-----------|
| 띄어쓰기 | "할수있다" | "할 수 있다" |
| 사이시옷 | "횟수" vs "회수" | 맥락에 따라 다름 |
| 피동/사동 | "보여지다" (이중 피동) | "보이다" |
| 외래어 | "메세지" | "메시지" |
| 접속 부사 | "그리고" 남발 | 삭제하거나 다른 연결어 |

### 2.2 종결어미 다양성 체크

```
한 페이지 내 종결어미 목록 추출 → 3회 이상 연속 반복이 없는지 확인

도구: 마지막 글자만 추출하여 패턴 확인
"~다. ~다. ~다." → 수정 필요
"~다. ~까. ~다." → OK
```

### 2.3 호흡/리듬 체크

**소리 내어 읽기 테스트**:
- 숨이 차는 문장 = 너무 긴 → 분리
- 뚝뚝 끊기는 문장 연속 = 너무 짧은 것이 과다 → 1~2개 합치기
- 자연스럽게 읽히는 = 한국어 호흡에 맞음

## 3. 퇴고 방법론

### 3.1 시간 두기 (숙성)

| 퇴고 회전 | 권장 숙성 기간 |
|----------|--------------|
| 1회전 (구조) | 초고 완성 후 1~2주 |
| 2회전 (장면) | 1회전 후 3~5일 |
| 3회전 (문장) | 2회전 후 1~3일 |
| 최종 확인 | 3회전 후 1일 (프린트 출력 권장) |

**[팁]** 웹소설 연재 시에는 이 기간을 지키기 어려움 → 버퍼의 중요성

### 3.2 역순 읽기

- 마지막 회차부터 첫 회차까지 거꾸로 읽기
- 스토리에 빠지지 않고 문장 자체를 객관적으로 볼 수 있음
- 3회전(문장 퇴고) 시 효과적

### 3.3 역할 바꿔 읽기

| 역할 | 초점 | 질문 |
|------|------|------|
| 독자로 읽기 | 재미, 몰입 | "계속 읽고 싶은가?" |
| 편집자로 읽기 | 구조, 논리 | "빠진 것은 없는가?" |
| 비평가로 읽기 | 주제, 문체 | "이 작품이 말하고자 하는 것은?" |
| 적대적 독자로 읽기 | 약점, 허점 | "이 장면을 공격할 수 있는가?" |

## 4. 베타리딩과 편집 협업

### 4.1 베타리더 활용

**좋은 베타리더의 조건**:
- 타겟 독자층과 유사한 독서 취향
- 솔직하게 말할 수 있는 관계
- 감정적 반응과 논리적 지적을 모두 할 수 있는 사람

**베타리더에게 요청할 질문**:
1. "어디서 지루했는가?"
2. "이해 안 되는 부분이 있었는가?"
3. "캐릭터 중 누가 가장 좋았고, 누가 가장 안 좋았는가?"
4. "읽다가 멈춘 적이 있는가? 어디서?"
5. "결말에 만족했는가?"

### 4.2 편집자와의 협업

| 단계 | 내용 |
|------|------|
| 기획 편집 | 플롯/구조 논의, 방향성 조율 |
| 원고 편집 | 구조적 수정 제안, 장면 재배치 |
| 교정 편집 | 문장 다듬기, 오류 수정 |
| 교열 | 맞춤법, 표기법, 사실 확인 |

**편집 피드백 수용 원칙**:
1. 첫 반응으로 반발하지 않기 (24시간 숙성)
2. "무엇이 문제인지"는 수용, "어떻게 고치라"는 참고만
3. 핵심 서사/테마에 대한 수정은 충분히 논의 후 결정
4. 모든 수정에 동의할 필요 없음 — 최종 결정은 작가
