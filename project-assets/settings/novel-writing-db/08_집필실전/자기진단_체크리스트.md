# 자기 진단 종합 체크리스트

> 작가의 가장 강력한 무기는 "자기 작품을 객관적으로 보는 능력"이다.
> 이 체크리스트는 집필 전/중/후 모든 단계에서 사용 가능한 종합 진단 도구.

---

## 1. 기획 단계 진단

### 1.1 아이디어 검증

- [ ] 이 아이디어를 한 줄(로그라인)로 설명할 수 있는가?
- [ ] 이 아이디어로 예정 분량을 채울 수 있는 서사적 잠재력이 있는가?
- [ ] 기존 작품과 차별화되는 고유한 매력이 있는가?
- [ ] 타겟 독자층이 명확한가?
- [ ] 이 아이디어에 대한 열정이 장기 연재를 버틸 만큼 강한가?

### 1.2 플롯 설계 검증

- [ ] 전체 아크 구조가 설계되어 있는가? (소/중/대)
- [ ] 각 아크의 클라이맥스가 명확한가?
- [ ] 주인공의 목표와 장애물이 명확한가?
- [ ] 갈등이 에스컬레이션 구조를 가지고 있는가?
- [ ] 결말(또는 방향)이 설정되어 있는가?

### 1.3 캐릭터 설계 검증

- [ ] 주인공에게 명확한 동기/결핍/목표가 있는가?
- [ ] 주인공에게 성장의 여지가 있는가?
- [ ] 악역/적대자에게 납득 가능한 동기가 있는가?
- [ ] 주요 인물 3~5명의 캐릭터 시트가 있는가?
- [ ] 인물 간 관계가 갈등/화학작용을 만들어내는가?

### 1.4 세계관 검증

- [ ] 핵심 설정(힘의 체계/사회 구조)이 정리되어 있는가?
- [ ] 설정에 내적 모순이 없는가?
- [ ] 세계관이 플롯에 직접 영향을 미치는가?
- [ ] 용어가 통일되어 있는가?

---

## 2. 회차 단위 진단

### 2.1 구조 진단

- [ ] 이 회차에 명확한 목적(플롯 진행/캐릭터 변화/세계관 심화)이 있는가?
- [ ] 오프닝(첫 3줄)이 독자를 잡는가?
- [ ] 2~4개의 장면으로 구성되어 있는가?
- [ ] 최소 1개의 감정 고점이 있는가?
- [ ] 엔딩이 다음 회차의 훅으로 기능하는가?
- [ ] 이 회차를 삭제하면 전체 서사에 구멍이 생기는가?

### 2.2 캐릭터 진단

- [ ] 인물의 행동이 설정된 성격과 일관되는가?
- [ ] 대사가 캐릭터 고유의 말투를 유지하는가?
- [ ] 캐릭터 간 호칭이 관계에 맞는가?
- [ ] 이 회차에서 캐릭터에 대한 새로운 면이 1개 이상 드러나는가?

### 2.3 문체 진단

- [ ] 문장 길이에 변화가 있는가? (긴 문장만 또는 짧은 문장만 아닌가?)
- [ ] 종결어미가 3회 이상 연속 반복되지 않는가?
- [ ] Show가 Tell보다 우세한가? (감정 직접 서술 vs 행동으로 보여주기)
- [ ] 불필요한 수식어가 없는가? ("매우", "정말", "너무")
- [ ] 같은 단어/표현이 가까운 거리에서 반복되지 않는가?

### 2.4 정보 진단

- [ ] 신규 용어/개념이 1~2개 이하인가?
- [ ] 세계관 설명이 설정 덤프가 아닌 자연스러운 도입인가?
- [ ] 독자가 이해하기에 충분한 정보가 제공되었는가?
- [ ] 정보 과잉으로 읽기 부담을 주지 않는가?

### 2.5 분량/가독성 진단

- [ ] 목표 분량을 충족하는가? (웹소설: 3,000~5,000자 / 장편: 7,000~9,000자)
- [ ] 대화 비중이 40%+ 인가? (웹소설 기준)
- [ ] 문단 길이가 2~4문장인가? (모바일 가독성)
- [ ] 시각적 여백이 충분한가? (빽빽하지 않은가?)

---

## 3. 아크/권 단위 진단

### 3.1 서사 구조

- [ ] 이 아크에 명확한 시작-중간-끝이 있는가?
- [ ] 주인공의 목표가 이 아크에서 변화/발전했는가?
- [ ] 갈등이 에스컬레이션 되었는가?
- [ ] 클라이맥스가 충분한 카타르시스를 제공하는가?
- [ ] 다음 아크로의 연결이 자연스러운가?

### 3.2 캐릭터 성장

- [ ] 주인공이 아크 시작과 끝에서 달라졌는가?
- [ ] 성장이 급작스럽지 않고 축적된 결과인가?
- [ ] 조연들에게도 변화가 있었는가?
- [ ] 새로 도입된 캐릭터가 서사에 기여하고 있는가?

### 3.3 떡밥/복선 관리

- [ ] 이 아크에서 설치한 떡밥/복선이 기록되었는가?
- [ ] 이전 아크의 단기 떡밥이 회수되었는가?
- [ ] 장기 떡밥에 대한 리마인드가 있었는가?
- [ ] 떡밥 부채가 누적되고 있지 않은가?

### 3.4 페이싱

- [ ] 텐션 곡선이 파형(고↔저)을 그리는가?
- [ ] 고텐션 구간이 3화 이상 연속하지 않는가?
- [ ] 이완 구간에서도 서브플롯/캐릭터 발전이 있는가?
- [ ] 아크 내 "가장 지루한 구간"이 3화를 넘지 않는가?

---

## 4. 전체 작품 진단

### 4.1 일관성 검증

- [ ] 세계관 규칙이 전체에서 일관되는가?
- [ ] 캐릭터 성격/능력이 설정과 일치하는가?
- [ ] 시간선에 모순이 없는가?
- [ ] 용어가 통일되어 있는가?
- [ ] 힘의 체계(파워 밸런스)가 일관되는가?

### 4.2 테마/주제

- [ ] 작품의 테마가 회차 전반에 걸쳐 일관되는가?
- [ ] 테마가 설교가 아닌 서사를 통해 전달되는가?
- [ ] 주인공의 성장 아크가 테마를 반영하는가?

### 4.3 완결성

- [ ] 모든 떡밥/복선이 회수되었거나 계획되어 있는가?
- [ ] 주인공의 핵심 결핍이 해소되는 방향인가?
- [ ] 결말이 서사적으로 납득 가능한가?
- [ ] 독자에게 충분한 여운을 남기는가?

---

## 5. 독자 관점 진단

### 5.1 몰입 테스트 (자가)

다음 질문에 "아니오"가 3개 이상이면 수정 필요:

1. 내가 독자라면 1화를 끝까지 읽겠는가?
2. 3화까지 읽은 독자가 4화를 클릭하겠는가?
3. 주인공에게 감정 이입이 되는가?
4. "다음에 무슨 일이 벌어질까" 궁금한가?
5. 악역/적대자가 위협적으로 느껴지는가?
6. 이 작품만의 "이것이 재밌다"를 설명할 수 있는가?
7. 50화까지 읽을 독자가 완결까지 갈 이유가 있는가?

### 5.2 신규 독자 접근성

- [ ] 1화만 읽어도 작품의 매력을 알 수 있는가?
- [ ] 세계관을 모르는 독자도 초반 5화를 이해할 수 있는가?
- [ ] 시놉시스/태그가 작품 내용과 일치하는가?

### 5.3 장르 계약 확인

- [ ] 태그/장르에 약속한 요소가 실제로 등장하는가?
- [ ] 장르 드리프트가 발생하고 있지 않은가?
- [ ] 독자 기대를 의도적으로 배반할 때 더 큰 만족을 주는가?

---

## 6. 사용법

### 6.1 진단 타이밍

| 타이밍 | 사용 섹션 | 목적 |
|--------|----------|------|
| 집필 전 | 1. 기획 단계 | 설계 검증 |
| 매 회차 | 2. 회차 단위 | 품질 관리 |
| 아크 완료 시 | 3. 아크/권 단위 | 구조 검증 |
| 50/100/200화 | 4. 전체 작품 | 방향성 점검 |
| 독자 반응 이상 시 | 5. 독자 관점 | 문제 진단 |

### 6.2 진단 기록

```
날짜:
진단 범위: [회차/아크/전체]
총 체크 항목: __개
통과: __개 / 미통과: __개
핵심 문제:
  1.
  2.
  3.
조치 계획:
  1.
  2.
  3.
```
