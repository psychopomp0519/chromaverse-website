# AI 보조 집필 워크플로우

> AI는 작가를 대체하지 않는다. 작가의 도구를 확장한다.
> 핵심은 "AI가 무엇을 잘하고, 무엇을 못하는가"를 아는 것.

---

## 1. AI의 강점과 한계

### 1.1 AI가 잘하는 것

| 영역 | 활용법 |
|------|--------|
| **아이디어 브레인스토밍** | "만약 ~라면?" 질문에 대한 다양한 답변 생성 |
| **세계관 설정 초안** | 지리/사회/종교 등 설정 골격 빠르게 구축 |
| **캐릭터 시트 보조** | 외형/성격/배경 초안 생성 → 작가가 다듬기 |
| **플롯 변형** | "이 전개를 다른 방식으로" → 대안 탐색 |
| **묘사 확장** | 감각 묘사, 환경 묘사의 초안 제안 |
| **문법/맞춤법 교정** | 기계적 오류 탐지 |
| **번역체 탐지** | 번역투 문장 식별 및 대안 제시 |
| **일관성 체크** | 설정/시간선/호칭 모순 탐지 |
| **요약/정리** | 긴 원고의 요약, 떡밥/복선 목록화 |

### 1.2 AI가 못하는 것

| 영역 | 이유 |
|------|------|
| **작가만의 목소리** | AI는 "평균적" 문체를 생성 → 개성 부재 |
| **감정의 진정성** | 실제 감정을 느끼지 않으므로 깊이에 한계 |
| **독창적 테마** | 기존 패턴의 조합일 뿐, 새로운 질문 생성 어려움 |
| **독자 감수성** | "이 장면에서 독자가 울까?"를 판단 못함 |
| **장기 서사 일관성** | 수백 화에 걸친 떡밥/성장의 통합적 관리 |
| **문화적 뉘앙스** | 한국어 특유의 존비어/호칭/세대별 말투의 미묘함 |

### 1.3 핵심 원칙

**[필수]** AI는 "초안 생성기"이지 "작가"가 아니다.
- AI 출력물은 반드시 작가가 검토/수정/재창작
- AI가 제안한 것을 그대로 쓰면 "AI 냄새"가 남 → 독자가 감지
- AI의 역할은 "빈 페이지의 공포"를 없애는 것

## 2. 단계별 AI 활용 워크플로우

### 2.1 기획 단계

| 작업 | AI 활용법 | 작가의 역할 |
|------|----------|-----------|
| 아이디어 | "~장르에서 신선한 소재 10가지" | 선별 + 독창성 부여 |
| 로그라인 | AI 초안 → 작가 다듬기 | 핵심 매력 결정 |
| 캐릭터 시트 | 기본 골격 생성 | 내면/비밀/독자적 특성 추가 |
| 세계관 | 5대 축 초안 생성 | 고유한 규칙/미학 부여 |
| 플롯 구조 | 대안적 전개 탐색 | 최종 선택 + 세부 설계 |

### 2.2 집필 단계

| 작업 | AI 활용법 | 주의점 |
|------|----------|--------|
| 묘사 확장 | "이 장면의 오감 묘사" | 작가 문체로 재작성 필수 |
| 대사 변형 | "이 대사를 다른 톤으로" | 캐릭터 일관성 확인 |
| 막힘 해소 | "다음 장면 전개 3가지 옵션" | 가장 자연스러운 것 선택 |
| 연결 문장 | 장면 전환 문장 제안 | 리듬/톤 일치 확인 |

### 2.3 퇴고 단계

| 작업 | AI 활용법 | 주의점 |
|------|----------|--------|
| 문법/맞춤법 | 자동 교정 | 의도적 파격 표현 보존 |
| 번역체 탐지 | 번역투 문장 식별 | 대안 중 작가 문체에 맞는 것 선택 |
| 일관성 | 설정/호칭/시간선 체크 | AI가 모든 맥락을 파악하진 못함 |
| 종결어미 반복 | 패턴 분석 | 수정은 작가가 직접 |

---

## 3. AI 활용의 금기

| 금기 | 이유 |
|------|------|
| AI 출력을 그대로 게재 | "AI 냄새" — 독자가 감지, 신뢰 상실 |
| AI에게 핵심 대사를 맡기기 | 캐릭터의 고유한 목소리가 사라짐 |
| AI 의존으로 자기 문체 포기 | 장기적으로 작가로서 성장 멈춤 |
| AI에게 테마 결정을 맡기기 | 작가의 핵심 역할 — 반드시 직접 |
| 기밀/미공개 원고 무분별 입력 | 정보 유출 위험 → 서비스 약관 확인 |

---

## 4. AI 도구별 적합 용도

| 도구 유형 | 적합 용도 | 부적합 용도 |
|----------|----------|-----------|
| 범용 대화형 AI | 브레인스토밍, 설정 토론, 퇴고 보조 | 긴 원고 직접 생성 |
| 문법 교정 AI | 맞춤법, 문법, 띄어쓰기 | 문체 판단 |
| 요약/분석 AI | 긴 원고 요약, 구조 분석 | 감정/미학 판단 |
| 이미지 생성 AI | 캐릭터/장소 시각화 (참고용) | 최종 삽화 |
