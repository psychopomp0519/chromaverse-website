# 소설 작문 종합 데이터베이스

> **목적**: 모든 장르, 모든 매체(웹소설/단행본/단편/장편)를 포괄하는 소설 작문 레퍼런스.
> 인기작의 성공 패턴과 비인기작의 실패 요인을 교차 분석하여 실전 집필에 활용.
>
> **버전**: 2.1.0
> **생성일**: 2026-03-18
> **최종 업데이트**: 2026-03-18 (3차 자체 평가 기반 고급 보완 완료)

## 사용법

### 집필 전 참조 순서
1. **집필실전** (08) → 전체 프로세스 파악, 기획 단계 시작
2. **장르별** (03) → 해당 장르의 필수 요소와 금기 확인
3. **매체별** (04) → 대상 매체의 구조적 제약과 전략 확인
4. **서사구조** (01) → 플롯/갈등/페이싱/테마 설계
5. **캐릭터** (02) → 인물 설계와 관계 구축
6. **문체기법** (05) → 실제 집필 시 문장/묘사/시점/가독성 참조
7. **독자심리** (06) → 몰입 장치와 회차 전략 점검
8. **금기회피** (07) → 최종 점검 체크리스트 (장르 이탈 포함)

### 상황별 빠른 참조표 (교차 참조)

| "지금 이런 상황인데..." | 참조할 파일 |
|------------------------|-----------|
| 전투 장면을 써야 한다 | `전투_액션묘사` → `페이싱_리듬` → `감정전달` → `액션_무협` |
| 캐릭터 간 대화가 부자연스럽다 | `대사설계` → `한국어_특화기법` → `관계역학` |
| 장면 전환이 어색하다 | `장면설계_전환` → `페이싱_리듬` |
| 악역이 매력이 없다 | `악역_적대자설계` → `캐릭터_아키타입` → `갈등설계` |
| 독자가 떠나고 있다 | `독자이탈_요인` → `독자소통_피드백` → `회차구성_전략` |
| 세계관이 얕아 보인다 | `세계관_구축` → `판타지(또는 해당 장르)` |
| 서브플롯을 넣고 싶다 | `서브플롯_다선구조` → `페이싱_리듬` |
| 복선/떡밥 관리가 안 된다 | `복선_떡밥_기법` → `자기진단_체크리스트` |
| 퇴고를 어떻게 해야 할지 모르겠다 | `퇴고_편집` → `한국어_특화기법` → `가독성_대화비중` |
| 연재 버퍼가 바닥이다 | `연재관리_운영` → `집필_프로세스` → `독자소통_피드백` |
| 유료 전환 시점이 고민된다 | `연재관리_운영` → `플랫폼별_전략` → `회차구성_전략` |
| 반전을 넣고 싶다 | `추리_미스터리` → `복선_떡밥_기법` → `명장면_설계` |
| 문학 공모전에 내고 싶다 | `순문학_문학소설` → `시놉시스_피칭` → `퇴고_편집` |
| 장면이 지루하다 | `독자몰입_요소` → `장면설계_전환` → `유머_코미디` |
| 명장면을 만들고 싶다 | `명장면_설계` → `감정전달` → `전투_액션묘사` |

### 크로마버스 프로젝트 연동
- 이 데이터베이스는 **범용 레퍼런스**로, 크로마버스 전용 설정(`크로마버스_작문_원칙_통합.md`)과 병행 참조
- 크로마버스 특화 규칙이 범용 규칙과 충돌 시: **크로마버스 마스터 문서 우선**
- 기존 `fantasy_writing_reference.json`은 판타지 장면 패턴 특화 → 이 DB의 보조 자료로 유지

## 분석 기반

### 인기작 분석 소스 (장르별 대표작)
- **판타지**: 달빛조각사, 전지적 독자 시점, 오버로드, 무직전생, 반지의 제왕
- **로맨스**: 이 결혼은 어차피 망합니다, 악녀는 모래시계를 되돌린다, 그 기사가 레이디를 고른 사연
- **액션/무협**: 나 혼자만 레벨업, 화산귀환, 튜토리얼이 너무 어렵다
- **현대/일상**: 미생, 나의 아저씨, 치즈 인 더 트랩
- **SF**: 인터스텔라(소설), 세 몸 문제, 안드로이드는 전기양의 꿈을 꾸는가
- **호러/스릴러**: 구의 증인, 살인자의 기억법, 파피용의 성
- **추리/미스터리**: 셜록 홈즈, 금전차, 역전재판, 양의 침묵
- **퓨전**: 템빨, 약먹는 천재마법사, 회귀자 사용설명서
- **단행본/문학**: 소년이 온다, 채식주의자, 살인자의 기억법

### 비인기작 분석 소스
- 문피아/카카오페이지/네이버시리즈 하위권 작품군 공통 패턴
- 독자 리뷰 이탈 요인 분석 (별점 3 이하 작품 공통점)
- 조기 완결/연재 중단 작품의 구조적 문제점
