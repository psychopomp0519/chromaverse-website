# 단행본

> 권 단위 완결성을 갖춘 서사 설계. 출판 시장의 특성과 전략.

---

## 1. 단행본의 구조적 특성

| 특성 | 웹소설과의 차이 |
|------|---------------|
| 한 권이 완결적 | 회차별 훅보다 전체 아크 중시 |
| 일괄 소비 | 중간 이탈 적음, 여운이 중요 |
| 편집 과정 존재 | 퀄리티 높음, 수정 기회 |
| 서점 노출 | 표지/제목/소개문이 핵심 |
| 분량 고정 | 200~400쪽 (장르별 상이) |

---

## 2. 권 단위 설계

### 2.1 1권의 구조 (시리즈물)

```
도입 (10~15%): 세계/인물 소개 + 일상 → 촉발 사건
전개 (30~40%): 새 세계 진입, 시련, 성장
전환 (10~15%): 중간점 반전, 게임 체인저
위기 (20~25%): 최고 긴장, 가장 어두운 순간
절정+해소 (10~15%): 클라이맥스 + 1권 마무리 + 다음 권 암시
```

### 2.2 시리즈 설계

**각 권의 이중 구조**:
- **독립적 갈등**: 해당 권에서 시작되고 해결됨 (읽는 만족)
- **연속적 갈등**: 시리즈 전체를 관통 (다음 권 기대)

**권별 역할 예시 (3권 기준)**:
| 권 | 독립 갈등 | 시리즈 갈등 | 감정 |
|----|----------|------------|------|
| 1권 | 첫 번째 시련 해결 | 큰 위협 인지 | 기대+불안 |
| 2권 | 두 번째 시련 + 배신 | 위협 접근, 동맹 구축 | 절망+결의 |
| 3권 | 최종 대결 | 시리즈 갈등 해소 | 카타르시스 |

---

## 3. 단행본 vs 웹소설 집필 차이

| 요소 | 웹소설 | 단행본 |
|------|--------|--------|
| 오프닝 | 3줄 훅 필수 | 1장 전체가 훅 역할 |
| 문장 | 빠른 호흡, 짧은 문장 | 다양한 호흡 허용 |
| 묘사 | 최소화 경향 | 풍부한 묘사 가능 |
| 복선 | 근거리 위주 | 장거리 가능 |
| 분량 | 회차별 균일 | 장별 가변 가능 |
| 여운 | 클리프행어 | 의미 있는 마무리 |

## 4. 단행본 체크리스트

- [ ] 한 권 안에서 시작-전개-해결이 완결적인가?
- [ ] 시리즈물이라면 독립 갈등 + 연속 갈등이 모두 존재하는가?
- [ ] 첫 장이 독자를 끌어들이는가?
- [ ] 마지막 장에 여운이 있는가?
- [ ] 전체 분량이 적절한가? (장르/독자층 기준)
