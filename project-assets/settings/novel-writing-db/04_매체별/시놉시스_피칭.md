# 시놉시스와 피칭

> 작품을 "팔기 위한 글쓰기". 공모전/투고/플랫폼 제출에 필수.
> 시놉시스, 로그라인, 기획서, 피칭 프레젠테이션.

---

## 1. 로그라인 (1문장 요약)

### 1.1 로그라인 공식

```
[특성/결핍을 가진 주인공]이 [촉발 사건]에 의해
[목표]를 추구하지만 [장애물/대립]에 직면하는 이야기
```

### 1.2 좋은 로그라인의 조건

| 조건 | 설명 |
|------|------|
| 아이러니 | 설정 자체에 모순/반전 ("죽음을 두려워하는 사신") |
| 갈등 명시 | 무엇과 싸우는지 |
| 스테이크 명시 | 실패하면 뭘 잃는지 |
| 차별점 | 왜 이 이야기가 특별한지 |

### 1.3 예시

```
좋은 예:
"세계 최강의 마법사가 가장 약한 종족으로 환생하여, 마법 없이 세계를 구해야 한다."
→ 아이러니(최강→최약), 갈등(마법 없이), 스테이크(세계)

나쁜 예:
"주인공이 모험을 떠나 강해지는 이야기."
→ 아이러니 없음, 갈등 불명확, 차별점 없음
```

## 2. 시놉시스 (작품 요약)

### 2.1 시놉시스 유형

| 유형 | 분량 | 용도 |
|------|------|------|
| 숏 시놉시스 | 200~500자 | 플랫폼 소개, 1차 심사 |
| 풀 시놉시스 | 1,000~3,000자 | 공모전 제출, 편집자 검토 |
| 확장 시놉시스 | 5,000~10,000자 | 출판 기획서, 시리즈 계획 |

### 2.2 숏 시놉시스 구조 (500자)

```
[도입] 주인공 소개 + 일상/결핍 (100자)
[촉발] 모든 것을 바꾸는 사건 (50자)
[전개] 목표 추구 + 주요 갈등 (150자)
[전환] 중간점 반전/진짜 갈등 드러남 (100자)
[결말 암시] 스테이크 제시 + 기대감 (100자)
```

**[필수]** 시놉시스에서 **결말을 밝히는가?**:
- 공모전 제출용: **결말 포함** (심사위원은 완결성을 봄)
- 플랫폼 연재용: **결말 미포함** (독자 호기심 유발)

### 2.3 시놉시스에서 해야 할 것 / 하면 안 될 것

| 해야 할 것 | 하면 안 될 것 |
|-----------|-------------|
| 핵심 갈등 명시 | 모든 서브플롯 나열 |
| 캐릭터의 동기/결핍 | 캐릭터 외형 묘사 |
| 세계관의 핵심 1개 | 세계관 전체 설명 |
| 차별점 강조 | "재밌습니다" 자화자찬 |
| 구체적 사건 언급 | 추상적 설명 ("성장하는 이야기") |

## 3. 제목 설계

### 3.1 웹소설 제목 패턴 (인기 패턴)

| 패턴 | 예시 | 효과 |
|------|------|------|
| 역설/반전형 | "약먹는 천재마법사" | 아이러니로 클릭 유도 |
| 목표 선언형 | "나 혼자만 레벨업" | 목표 명확, 장르 즉각 파악 |
| 비밀/미스터리형 | "전지적 독자 시점" | "뭐지?" 호기심 |
| 상태 제시형 | "이 결혼은 어차피 망합니다" | 상황 즉각 전달 |
| 행동 선언형 | "화산귀환" | 간결, 임팩트 |

### 3.2 나쁜 제목 패턴

| 패턴 | 문제 |
|------|------|
| 너무 추상적 | "빛의 여정" → 무슨 이야기? |
| 너무 김 | "어느 날 눈을 떠 보니 세계 최강이 되어 있었는데 사실 그게 저주였던 이야기" |
| 장르 불일치 | 판타지인데 로맨스 같은 제목 |
| 검색 불가 | 일반 단어만으로 구성 → 검색 묻힘 |

## 4. 태그/키워드 전략

### 4.1 필수 태그 구성

```
장르 태그: 판타지, 무협, 로맨스 등
소재 태그: 회귀, 성장, 복수, 시스템 등
톤 태그: 다크, 코미디, 힐링 등
특징 태그: 먼치킨, 지략, 비밀병기 등
```

### 4.2 태그의 역할
- 태그 = 독자와의 약속 (이 태그의 내용이 나옵니다)
- 잘못된 태그 = 장르 이탈과 동일한 효과
- **[필수]** 태그에 쓴 내용은 작품에 반드시 존재해야 함

---

## 5. 공모전 기획서 구성

```
1. 작품 개요 (1페이지)
   - 제목, 장르, 분량 계획, 타겟 독자
   - 로그라인
   - 핵심 차별점

2. 시놉시스 (1~2페이지)
   - 전체 줄거리 (결말 포함)

3. 캐릭터 소개 (1페이지)
   - 주인공 + 핵심 조연 2~3명
   - 각 인물의 동기/갈등/역할

4. 세계관 요약 (선택, 0.5~1페이지)
   - 핵심 설정만 간략히

5. 연재 계획 (0.5페이지)
   - 예상 분량, 연재 주기, 아크 구성

6. 원고 샘플 (3~5화)
```
