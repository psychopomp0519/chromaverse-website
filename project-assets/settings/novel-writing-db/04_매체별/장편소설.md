# 장편소설

> 시리즈/대하 소설의 설계. 다선 서사, 세계관 밀도, 장기 연재의 기술.

---

## 1. 장편의 구조적 특성

### 1.1 규모별 분류

| 규모 | 분량 | 특성 |
|------|------|------|
| 단권 장편 | 300~500쪽 | 완결적, 단일 아크 |
| 시리즈 (3~5권) | 900~1,500쪽 | 권별 완결+시리즈 아크 |
| 대하 소설 (10권+) | 3,000쪽+ | 다선 서사, 세계관 밀도 |
| 웹소설 장편 (200화+) | 60만자+ | 연재 특화, 루프 구조 |

---

## 2. 장편 설계의 핵심

### 2.1 다선(多線) 서사 관리

**메인 플롯 + 서브 플롯 구조**:
```
메인 플롯: 전체를 관통하는 핵심 갈등/목표
  ├── 서브 플롯 A: 캐릭터 관계 (로맨스/우정)
  ├── 서브 플롯 B: 세계관 미스터리
  ├── 서브 플롯 C: 개인적 성장/과거
  └── 서브 플롯 D: 정치/사회적 배경
```

**관리 원칙**:
- 서브 플롯은 메인 플롯과 연동 (독립적 서브 = 산만)
- 서브 플롯도 시작-전개-해소 존재 (방치 금지)
- 한 장면에서 최소 2개 플롯이 동시 진행 = 효율적

### 2.2 세계관 밀도 관리

| 단계 | 비율 | 방법 |
|------|------|------|
| 표면 세계관 | 30% | 자연스러운 배경 묘사 |
| 사건 연동 세계관 | 50% | 갈등/사건과 연결된 설정 |
| 심층 세계관 | 20% | 독자가 추론해야 하는 부분 |

### 2.3 장기 텐션 관리

**시즌 구조** (대하소설/장기 연재):
```
시즌 1 (1~50화): 세계 소개, 첫 번째 대적, 기반 구축
시즌 2 (51~100화): 스케일 확장, 새 세력, 큰 위기
시즌 3 (101~150화): 핵심 진실 접근, 최대 위기
시즌 4 (151~200화): 최종 결전, 해소, 여운
```

각 시즌은 독자적 아크를 가지되 전체 스토리에 수렴

## 3. 장편 특유의 기법

### 3.1 시간 도약 (Time Skip)
- 성장/변화를 압축적으로 보여줌
- **[주의]** 도약 후 "달라진 것"을 바로 보여줘야 효과적
- 도약 중 무슨 일이 있었는지는 점진적 회상으로

### 3.2 시점 확장
- 주인공 외 다른 인물의 시점 삽입 (장편만의 특권)
- **[인기]** 악역/조연의 시점이 서사 깊이를 더함
- **[비인기]** 시점이 너무 많으면 산만 → 3~4개 이내 권장

### 3.3 테마의 변주
- 같은 테마를 다른 상황/인물로 반복 탐구
- 시리즈가 길어질수록 테마가 깊어져야 함 (반복이 아닌 심화)

## 4. 완결 설계

### 4.1 만족스러운 완결의 조건

| 조건 | 설명 |
|------|------|
| 약속 이행 | 서사 초반의 약속(갈등/질문)에 답함 |
| 캐릭터 아크 완성 | 주요 인물의 변화가 완결됨 |
| 복선 회수 | 깔아둔 것들을 거둬들임 |
| 감정적 해소 | 독자가 축적한 감정이 해소됨 |
| 여운 | 끝나고도 생각나는 무언가 |

### 4.2 완결의 금기

| 금기 | 이유 |
|------|------|
| 급마무리 | "갑자기 끝났어" → 배신감 |
| 미해결 투성이 | "그래서 뭐?" → 허탈 |
| 신캐릭터가 해결 | 기존 인물의 서사 무의미화 |
| 과도한 에필로그 | 여운 대신 사족 |
| "사실 꿈이었다" | 전체 서사 부정 |
