# 플랫폼별 연재 전략

> 같은 작품이라도 플랫폼에 따라 성패가 갈린다.
> 한국 주요 웹소설 플랫폼별 특성, 독자층, 전략.

## 2. 카카오페이지 전략

### 2.1 핵심 특성
- "기다리면 무료(기다무)" 시스템 → 초반 유입 극대화 가능
- 웹툰/웹소설 통합 플랫폼 → 교차 유입
- AI 추천 알고리즘 → 초반 데이터(클릭률/구매전환)가 핵심

### 2.2 전략
- **1화 극강 임팩트**: 기다무로 1화 무료 → 1화가 마케팅 그 자체
- **짧은 회차(3,000~5,000자)**: 빠른 소비, 모바일 최적화
- **매일 연재**: 알고리즘 노출 극대화
- **판현무(판타지/현대판타지/무협)** 장르가 매출 최상위
- **밀리언페이지 달성** = 성공 지표 (누적 조회 100만)

### 2.3 인기작 사례 분석
- **괴담출근**: 현대판타지, 3개월 만에 1억 조회, 밀리언페이지 최단 달성
  - 성공 요인: 독특한 소재(괴담+직장), 빠른 전개, 사이다 밀도 높음

## 3. 네이버시리즈 전략

### 3.1 핵심 특성
- 무협 장르 강세 (카카오와 차별점)
- 네이버 포털 연동 → 검색 유입 가능
- 평점/리뷰 시스템이 작품 생사 결정

### 3.2 전략
- **무협 장르 적합**: 네이버에서 무협 독자 기반 탄탄
- **중간 분량(4,000~6,000자)**: 카카오보다 여유 있는 호흡 가능
- **평점 관리**: 초반 리뷰가 장기 노출에 영향
- **주 5회 이상 연재**: 꾸준함이 랭킹 유지 핵심

---

## 4. 문피아 전략

### 4.1 핵심 특성
- 남성향 매니아 커뮤니티
- "베스트" 게시판의 영향력 → 입소문 형성
- 긴 호흡의 서술 수용 (다른 플랫폼보다 긴 분량 가능)
- 자유연재 → 유료전환 경로

### 4.2 전략
- **자유연재로 독자 확보 후 프리미엄 전환**: 50~100화 자유연재 → 유료
- **긴 분량(5,000~8,000자) 가능**: 독자가 긴 호흡 수용
- **커뮤니티 소통**: 댓글/작가의 말이 독자 유지에 핵심
- **판타지/시스템물/무협** 강세

## 5. 공모전/독점 계약 전략

### 5.1 주요 공모전

| 공모전 | 플랫폼 | 특징 |
|--------|--------|------|
| 카카오페이지 스테이지 | 카카오 | 최대 상금, 독점 계약 기회 |
| 문피아 대상 | 문피아 | 남성향 특화, 권위 |
| 네이버 공모전 | 네이버 | 포털 노출 기회 |
| 노벨피아 공모전 | 노벨피아 | 진입 장벽 낮음 |

### 5.2 시놉시스 작성법

**[필수]** 시놉시스 = 작품의 영업 자료

**구성**:
```
1. 한 줄 로그라인 (1문장): 핵심 매력 압축
2. 장르/태그 (1줄): 독자 타겟 명시
3. 줄거리 요약 (300~500자): 시작→갈등→전환점→결말 암시
4. 캐릭터 소개 (200자): 주인공+핵심 조연 2~3명
5. 차별점 (100자): "왜 이 작품을 읽어야 하는가"
```

**로그라인 공식**:
```
[주인공]이 [촉발 사건]으로 인해 [목표]를 위해 [장애물]에 맞서는 [장르] 이야기
```

**예시**:
```
버림받은 검의 성자가 300년 전으로 회귀하여, 무너질 왕국을 재건하며
자신을 배신한 자들에게 복수하는 전략 회귀 판타지
```

| 지표 | 설명 | 목표치 |
|------|------|--------|
| 클릭률(CTR) | 목록에서 클릭하는 비율 | 5%+ |
| 구매전환율 | 무료→유료 전환 | 10%+ |
| 리텐션 | 다음 화 계속 읽는 비율 | 70%+ |
| 일평균 조회 | 하루 조회수 | 성장 추세 유지 |
