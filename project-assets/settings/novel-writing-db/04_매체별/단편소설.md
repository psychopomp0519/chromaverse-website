# 단편소설

> 모든 글쓰기의 정수. 한 편의 단편에서 배울 수 있는 것이 장편 10권보다 많다.
> 압축, 반전, 여운, 상징의 기술.

---

## 1. 단편의 본질

### 1.1 핵심 원칙
> "단편소설은 하나의 효과를 위한 모든 것" — 에드거 앨런 포

| 원칙 | 설명 |
|------|------|
| 단일 효과 | 하나의 감정/인식/충격을 전달 |
| 경제성 | 모든 문장이 목적을 가짐. 불필요한 것 = 삭제 |
| 집중 | 인물 최소, 시간/공간 제한, 갈등 하나 |
| 여운 | 끝나고 나서 독자의 머릿속에 남는 것이 본문보다 큼 |

### 1.2 단편 vs 장편의 차이

| 요소 | 단편 | 장편 |
|------|------|------|
| 인물 수 | 1~3명 | 다수 |
| 시간 범위 | 짧은 순간~며칠 | 주~년 |
| 갈등 | 1개 핵심 갈등 | 다층 갈등 |
| 세계관 | 암시, 최소 설명 | 구축, 상세 설명 |
| 성장 | 인식 변화 (순간적) | 점진적 성장 |
| 결말 | 열린 결말/반전/여운 | 해소/완결 |

## 2. 단편 구조 패턴

### 2.1 반전형
- 마지막에 모든 것의 의미가 바뀜
- **핵심**: 다시 읽으면 처음부터 단서가 있었음
- 적합: 미스터리, 호러, 실험적 소설

### 2.2 에피파니형 (깨달음)
- 인물이 어떤 진실을 깨닫는 순간으로 끝남
- **핵심**: 독자도 함께 "아" 하는 순간
- 적합: 문학, 일상, 성장

### 2.3 순환형
- 시작과 끝이 같은 이미지/상황이지만, 의미가 달라짐
- **핵심**: 변화의 유무가 테마를 전달

### 2.4 분위기형
- 명확한 서사보다 분위기/감정을 전달
- **핵심**: 언어의 음악성, 이미지의 힘
- 적합: 시적 소설, 감각적 호러

## 3. 단편 집필 기법

### 3.1 시작: in medias res
- 사건의 한가운데서 시작
- 배경 설명 최소화 → 독자가 맥락을 추론하게

### 3.2 상징 활용
- 단편에서 상징은 장편의 서브플롯 역할
- 반복되는 이미지/물건/색이 의미를 축적

### 3.3 생략의 기술
- 말하지 않는 것이 말한 것보다 강력
- 행간에 의미를 숨기고 독자가 발견하게

### 3.4 마지막 문장의 힘
**[필수]** 단편의 마지막 문장 = 이야기의 심장

```
좋은 예: 전체를 관통하는 한 문장으로 여운
나쁜 예: 설명적 마무리 ("그래서 그는 행복해졌다")
```

## 4. 인기/비인기 패턴

### 인기
| 패턴 | 핵심 |
|------|------|
| 강력한 마지막 한 줄 | 기억에 남는 여운 |
| 경제적 서술 | 군더더기 없음 |
| 보편적 테마 | 사랑, 상실, 정체성, 죽음 |
| 독특한 시점 | 예상 밖의 화자/시점 |

### 비인기
| 패턴 | 문제 |
|------|------|
| 장편을 압축한 느낌 | "더 긴 이야기를 억지로 줄인" 느낌 |
| 결말 설명 과다 | 여운 없이 전부 설명 |
| 테마 불분명 | "뭘 말하려는 거야?" |
| 인물 과다 | 단편에 5명 이상 → 혼란 |

## 5. 공모전 실전 가이드

### 5.1 공모전 유형

| 유형 | 특성 | 분량 | 주요 공모전 |
|------|------|------|-----------|
| 문예지 신인상 | 문학성 중심, 순문학 | 원고지 80~120매 | 문학과지성, 창작과비평, 현대문학 |
| 장르 공모전 | 장르적 재미+완성도 | 200~400매 | 대산창작기금, 한겨레문학상 |
| 플랫폼 공모전 | 시장성+독자 반응 | 5~30화 (회차 단위) | 문피아, 카카오, 네이버 공모전 |
| 테마 공모전 | 특정 주제 제한 | 다양 | 각종 기관/기업 공모 |

### 5.2 심사위원 관점 시뮬레이션

심사위원은 수십~수백 편을 읽는다. **첫 3페이지에서 판가름**:

| 탈락 사유 (공통) | 비율 | 대응 |
|----------------|------|------|
| 진부한 소재/전개 | 30% | 첫 문장부터 차별화 |
| 문장력 부족 | 25% | 퇴고 3회전 이상 |
| 테마 불분명 | 20% | 핵심 질문 명확히 |
| 구조적 미완성 | 15% | 결말까지 설계 후 집필 |
| 분량 초과/미달 | 10% | 규정 엄수 |

**[팁]** 심사위원이 좋아하는 것:
- "이건 처음 보는 이야기다" → 소재의 신선함 또는 시점의 독창성
- "문장이 좋다" → 첫 3줄의 문장력으로 판단
- "무엇을 말하고자 하는지 명확하다" → 테마의 선명함

### 5.3 원고 분량 관리

| 단위 | 글자 수 (공백 포함) |
|------|-------------------|
| 원고지 1매 | 200자 |
| 원고지 80매 | 16,000자 |
| 원고지 100매 | 20,000자 |
| A4 1페이지 (12pt) | 약 800~1,000자 |

**[필수]** 공모전 규정 분량의 90~100%를 채울 것. 80% 미만이면 "미완성" 인상.

### 5.4 공모전 투고 체크리스트

- [ ] 응모 규정(분량, 형식, 기한) 확인
- [ ] 원고 규격(글꼴, 행간, 여백) 준수
- [ ] 작가 정보와 원고를 분리 (익명 심사인 경우)
- [ ] 맞춤법/오탈자 최종 검수
- [ ] 타 공모전 중복 응모 규정 확인

## 6. 초단편/플래시 픽션

### 6.1 정의

| 유형 | 분량 | 특징 |
|------|------|------|
| 초단편 | 1,000~3,000자 | 장면 1~2개, 핵심만 |
| 플래시 픽션 | 500자 이하 | 한 장면, 한 순간 |
| 마이크로 픽션 | 100자 이하 | 한 문장~한 문단 |

### 6.2 초단편의 핵심

- **1개의 이미지**: 전체가 하나의 강렬한 이미지
- **역전의 힘**: 마지막 한 줄이 모든 것을 뒤집음
- **암시의 극대화**: 말한 것보다 말하지 않은 것이 10배

- [ ] 마지막 문장/장면에 여운이 있는가?
- [ ] 설명보다 암시가 많은가?
- [ ] 다시 읽으면 새로운 의미가 보이는가?
- [ ] 공모전이라면 규정 분량을 준수하는가?
- [ ] 첫 3줄이 독자(심사위원)를 잡는가?
