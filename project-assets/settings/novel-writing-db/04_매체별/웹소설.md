# 웹소설

> 연재 매체의 특성을 이해하고 활용하는 전략.
> 회차 구조, 플랫폼별 특성, 연재 전략, 수익화.

---

## 1. 웹소설의 구조적 특성

### 1.1 회차 단위 소비

| 특성 | 영향 | 대응 |
|------|------|------|
| 1회차 = 독립적 소비 단위 | 매 회차에 "읽을 이유" 필요 | 훅+클리프행어 필수 |
| 유료 결제 기반 | 다음 회차를 "사게" 만들어야 | 떡밥/기대감 관리 |
| 댓글/반응 실시간 | 독자 피드백 즉시 반영 가능 | 독자 심리 읽기 |
| 경쟁작 다수 | 이탈 장벽 극히 낮음 | 초반 3화 생존 필수 |

### 1.2 회차 분량

| 플랫폼 | 권장 분량 | 특이사항 |
|--------|----------|----------|
| 카카오페이지 | 3,000~5,000자 | 짧은 호흡, 클리프행어 중시 |
| 네이버시리즈 | 4,000~6,000자 | 중간 분량, 균형형 |
| 문피아 | 5,000~8,000자 | 긴 호흡 가능, 남성향 중심 |
| 노벨피아 | 4,000~7,000자 | 다양한 장르 |
| 일본 나로우/카쿠요무 | 2,000~4,000자 | 매우 짧은 호흡 |

## 2. 회차 설계 전략

### 2.1 오프닝 훅 (첫 3줄)

**[필수]** 첫 3줄 안에 독자를 붙잡아야 함:

**효과적인 훅 유형**:
| 유형 | 예시 |
|------|------|
| 행동 개시 | "칼이 목에 닿았을 때, 나는 웃었다." |
| 미스터리 | "그 날, 마을에서 세 번째 아이가 사라졌다." |
| 대사 | "네가 왕이 되는 건 불가능해." |
| 감각 충격 | "피 냄새. 그것이 내가 깨어나며 가장 먼저 느낀 것이었다." |
| 내면 선언 | "이번 생에는 죽지 않는다." |

**[비인기]** 피해야 할 오프닝:
- 풍경 묘사로 시작 (1화가 아니면 가능)
- 설정 설명으로 시작
- "나는 ○○이다" 자기소개

### 2.2 클리프행어 (회차 끝)

**[필수]** 매 회차 끝에 "다음 회차를 읽어야 하는 이유" 배치:

| 유형 | 효과 |
|------|------|
| 위기 절단 | 위험한 순간에서 끊음 → "어떻게 돼?" |
| 정보 직전 절단 | 비밀이 밝혀지기 직전 → "뭔데?!" |
| 반전 절단 | 예상 밖 상황 발생 → "이게 뭐야?" |
| 결심 절단 | 주인공이 결정적 결심 → "뭘 할 거야?" |
| 등장 절단 | 새 인물/위협 등장 → "누구야?" |

**[비인기]** 나쁜 회차 끝:
- 모든 것이 해결된 채 끝남 → 다음 회차 클릭 동기 없음
- 매번 똑같은 패턴의 클리프행어 → 예측 가능, 효과 감소

### 2.3 초반 3화 생존 전략

**1화**: 세계관의 핵심 + 주인공의 매력 + 강력한 갈등/훅
**2화**: 1화의 갈등 해소/발전 + 더 큰 문제의 시작
**3화**: 주인공의 방향성 확립 + 독자의 감정 투자 확보

**[인기]** 1화에서 해야 할 것:
- 주인공의 목표/결핍 제시
- 세계관의 "맛보기" (전부가 아닌 핵심 1개)
- 사이다 1개 (작지만 만족스러운 순간)
- 클리프행어

**[비인기]** 1화에서 하면 안 되는 것:
- 세계관 설정 나열
- 평범한 일상만 그리기
- 캐릭터 소개 카탈로그
- "이 세계는 ~로 이루어져 있다" 설명

## 3. 연재 전략

### 3.1 연재 주기

| 주기 | 장점 | 단점 |
|------|------|------|
| 매일 연재 | 플랫폼 알고리즘 유리, 독자 습관 형성 | 퀄리티 유지 어려움 |
| 주 3~5회 | 품질 유지 가능, 여전히 높은 노출 | 매일 연재작에 밀릴 수 있음 |
| 주 1~2회 | 높은 품질 가능 | 독자 이탈 위험, 알고리즘 불리 |

### 3.2 분량 축적 전략

**[인기]** 베스트 프랙티스:
1. 최소 20~30화 사전 집필 후 연재 시작
2. 연재 시작 후에도 항상 10화 이상 버퍼 유지
3. 버퍼로 퀄리티 관리 + 독자 반응 대응 여유

### 3.3 아크 설계 (연재 기준)

```
소 아크 (5~10화): 하나의 사건/갈등 해결 단위
  → 각 소 아크에 사이다 1개 이상

중 아크 (20~40화): 하나의 목표/성장 단위
  → 큰 반전 또는 주요 갈등 해소

대 아크 (50~100화): 시즌 단위
  → 주요 악역/문제 해결, 새 국면 시작
```

## 4. 독자 관리

### 4.1 독자 유형별 대응

| 유형 | 특징 | 대응 |
|------|------|------|
| 정주행러 | 무료 분량 폭식 | 초반 퀄리티 최고로 |
| 연재 추적러 | 매 회차 결제 | 클리프행어/떡밥 관리 |
| 완결 후 독자 | 완결 확인 후 시작 | 완결 퀄리티 중요 |
| 리뷰어/추천인 | 입소문 생산 | 1~3화 임팩트가 핵심 |

### 4.2 사이다/고구마 밸런스

```
사이다(쾌감): 주인공이 상대를 이기거나, 인정받거나, 반전 성공
고구마(답답): 오해, 불의, 위기, 패배, 억울한 상황

이상적 비율: 사이다 60% / 고구마 40%
```

**[인기]** 핵심: 고구마를 쌓은 만큼 사이다의 쾌감이 큼
**[비인기]** 고구마만 계속 → "답답해서 못 읽겠다"
