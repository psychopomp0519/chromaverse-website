# 캐릭터 아키타입

> 인물 유형의 원형(Archetype)과 변형. 원형을 알아야 독창적 변주가 가능하다.
> 장르별 주요 아키타입, 성공 패턴, 실패 패턴 정리.

---

## 1. 주인공 아키타입

### 1.1 영웅형 (Hero)
- **핵심**: 위기에 맞서 행동하는 인물. 용기와 희생이 동력.
- **변형**: 마지못한 영웅 / 안티히어로 / 타락한 영웅 / 복귀하는 영웅
- **[인기]** 결점 있는 영웅: 완벽하지 않아서 공감 가능. 약점이 성장의 재료.
- **[비인기]** 먼치킨 영웅: 처음부터 강하고, 고민 없고, 실패 없음 → 긴장 없음.
- **적합 장르**: 판타지, 액션, 모험, 성장물

### 1.2 평범인형 (Everyman/Everywoman)
- **핵심**: 독자 자신을 투영할 수 있는 보통 사람. 비범하지 않아서 공감 높음.
- **변형**: 평범한 학생 / 회사원 / 이세계 소환된 보통 사람
- **[인기]** 성장 아크와 결합: 평범 → 비범의 여정이 카타르시스
- **[비인기]** 평범한 채로 유지: 성장이 없으면 "왜 얘가 주인공이지?"
- **적합 장르**: 이세계물, 현대물, 일상물, 호러

### 1.3 반영웅형 (Anti-Hero)
- **핵심**: 전통적 영웅 미덕이 부족하지만 독자가 응원하게 되는 인물.
- **유형**: 도덕적 회색 / 냉소적 / 이기적이지만 선을 넘지 않음 / 과거에 상처
- **[인기]** 핵심: 악행에도 "이유"가 있고, "선을 넘지 않는 선" 존재.
- **[비인기]** 실패: 너무 악하면 응원 불가, 너무 착하면 반영웅이 아님.
- **적합 장르**: 느와르, 다크 판타지, 스릴러, 복수극

### 1.4 트릭스터형 (Trickster)
- **핵심**: 지혜/꾀/말로 상황을 돌파. 물리적 힘이 아닌 두뇌 플레이.
- **변형**: 사기꾼 / 해커 / 전략가 / 협상가 / 약자의 지혜
- **[인기]** 매력: 독자가 "어떻게 빠져나갈까" 기대하는 재미.
- **[비인기]** 실패: 속임수가 비현실적이거나, 주변 인물이 바보 → "이걸 왜 못 알아채?"
- **적합 장르**: 미스터리, 사기극, 정치극, 코미디

### 1.5 탐구자형 (Seeker/Explorer)
- **핵심**: 진실/지식/세계를 탐구. 호기심이 동력.
- **변형**: 학자 / 탐험가 / 기자 / 고고학자 / 관찰자
- **[인기]** 독자와 동일 시점: 탐구자가 알아가는 속도 = 독자가 알아가는 속도.
- **적합 장르**: 미스터리, 탐험, 세계관 중심 판타지, SF

### 1.6 치유자형 (Healer/Caregiver)
- **핵심**: 타인을 돌보고 치유하는 것이 동기. 부드러운 힘.
- **변형**: 의사/치료사 / 보호자 / 희생적 리더
- **[인기]** 핵심: 치유 능력이 "약점이자 강점"일 때 (남을 치유하면서 자신은 상처).
- **적합 장르**: 힐링, 로맨스, 일상, 의학물

### 1.7 회귀/환생자형 (Regressor)
- **핵심**: 미래 지식을 가지고 과거로 돌아온 인물. 웹소설 특화.
- **[인기]** 핵심: 미래 지식의 한계(나비효과, 기억 불완전, 감정적 대가) 존재.
- **[비인기]** 실패: 모든 것을 아는 전지적 주인공 → 긴장 0.
- **적합 장르**: 회귀물, 판타지, 로맨스, 무협

## 2. 조연 아키타입

### 2.1 멘토형 (Mentor/Wise Old Man)
- **역할**: 지혜 전수, 세계관 설명, 정신적 지주
- **[인기]** 패턴: 멘토가 불완전하거나, 비밀을 가지고 있거나, 퇴장(죽음/이별)
- **[비인기]** 패턴: 만능 해결사 → 멘토가 다 해결하면 주인공 존재 의미 없음
- **변형**: 타락한 멘토 / 떠나는 멘토 / 라이벌이 멘토 / 어린 멘토

### 2.2 동료형 (Companion/Sidekick)
- **역할**: 주인공의 부족한 면 보완, 감정적 지지, 유머 담당
- **[인기]** 핵심: 동료 자체의 목표/갈등이 있을 때 → 입체적 인물
- **[비인기]** 패턴: "예스맨" — 주인공에게 동의만 하는 인물 → "왜 있는 거야"
- **유형**: 대비형(성격 정반대) / 거울형(유사하지만 다른 선택) / 보완형(능력 보완)

### 2.3 연인형 (Love Interest)
- **역할**: 주인공의 감정선, 동기 부여, 약점이자 강점
- **[인기]** 핵심: 연인이 독립적 인격과 목표를 가짐 → "히로인이 아닌 인물"
- **[비인기]** 패턴: 트로피 와이프/허즈번드 — 존재 이유가 "예쁘니까/잘생겼으니까"
- **[금기]** 연인을 "보상"으로 취급 — 퀘스트 클리어 상품이 아님

### 2.4 적대자/악역형 (Antagonist/Villain)
- **역할**: 주인공의 대립면. 갈등의 원천. 테마의 반증.
- **[인기]** 6대 악역 유형:

| 유형 | 특징 | 매력 포인트 |
|------|------|------------|
| 신념형 | 자신이 옳다고 확신 | "그의 말도 일리가 있어" |
| 거울형 | 주인공의 다른 가능성 | "주인공이 저럴 수도 있었어" |
| 카리스마형 | 압도적 존재감 | 적인데 좋아하게 됨 |
| 비극형 | 과거의 상처가 동기 | 미워할 수 없는 악역 |
| 시스템형 | 개인이 아닌 체제 | 한 명 없앤다고 해결 안 됨 |
| 내면형 | 주인공 자신의 어둠 | 가장 극복하기 어려운 적 |

- **[비인기]** 실패 악역: 동기 없는 악 ("세상을 멸망시키겠다. 왜? 나쁘니까")
- **[필수]** 악역의 동기는 독자가 "이해는 되지만 동의할 수 없는" 수준

### 2.5 조력자형 (Helper/Ally)
- **역할**: 특정 국면에서 도움 → 주인공 성장의 디딤돌
- **[팁]** 도움에 대가가 있으면 서사가 풍부해짐 (호의가 아닌 거래/빚)
- **변형**: 정보상 / 대장장이 / 정치적 후원자 / 숨은 보호자

### 2.6 문지기형 (Threshold Guardian)
- **역할**: 다음 단계로 넘어가기 위해 통과해야 할 장벽 인물
- **패턴**: 처음에 적대적 → 극복 후 동료 또는 존중
- **예시**: 시험관, 까다로운 상사, 관문의 수호자

## 3. 캐릭터 조합의 화학작용

### 3.1 대비 원칙 (Contrast Principle)
- 성격이 대조되는 인물 배치 → 각자의 특성이 부각
- 예: 낙관적 vs 비관적 / 충동적 vs 신중 / 솔직 vs 은밀

### 3.2 파티 구성법 (앙상블 설계)

**이상적 파티 4~6인 구성**:

| 역할 | 기능 | 서사적 역할 |
|------|------|------------|
| 리더 | 방향 제시, 결정 | 주 갈등의 중심 |
| 두뇌 | 분석, 전략 | 세계관/정보 전달 |
| 전투력 | 물리적 해결 | 액션 장면의 핵심 |
| 감정 | 공감, 도덕적 나침반 | 감정선의 중심 |
| 조커 | 유머, 예측 불가 | 텐션 완화, 돌발 변수 |
| 미스터리 | 비밀/과거 보유 | 떡밥/복선의 보고 |

**[인기]** 핵심: 각 멤버가 "이 사람이 빠지면 파티가 무너진다"는 필요성 보유

### 3.3 관계 네트워크 설계

```
    갈등
A ←──── B
│╲      ↑
│ ╲신뢰  │보호
│  ╲    │
↓   ╲→  C
D ────→ E
  라이벌
```

**[필수]** 모든 주요 인물 쌍에 고유한 관계 정의 (A-B, A-C, B-C... 각각 다른 역학)

## 4. 캐릭터 차별화 요소

### 4.1 개성 구축의 7요소

| 요소 | 설명 | 예시 |
|------|------|------|
| 말투 | 어휘, 억양, 말버릇 | "~지 않나" 습관적 질문형 |
| 행동 습관 | 반복되는 동작 | 생각할 때 귀를 만지는 버릇 |
| 가치관 | 무엇을 중시하는가 | "약속은 반드시 지킨다" |
| 결핍 | 무엇이 부족한가 | 인정받지 못한 과거 |
| 모순 | 말과 행동의 불일치 | 냉정한 말, 따뜻한 행동 |
| 비밀 | 숨기고 있는 것 | 진짜 정체, 과거의 죄 |
| 변화 욕구 | 무엇이 되고 싶은가 | 자유로운 삶, 인정, 속죄 |

**[필수]** 최소 5개 이상은 설정해야 입체적 인물

### 4.2 캐릭터 음성 (Voice) 차별화

**[인기]** 대사만으로 누가 말하는지 알 수 있어야 함:
- A: "분석 결과로 판단하면 확률은 23%야."
- B: "에이, 될 거야 될 거! 가보자고!"
- C: "...별 상관없어."

**차별화 도구**:
| 도구 | 설명 |
|------|------|
| 어휘 수준 | 학자 vs 시골 청년 vs 아이 |
| 문장 길이 | 수다쟁이(장문) vs 과묵(단문) |
| 말버릇 | 특정 단어/표현 반복 |
| 태도 | 공손/반말/존댓말/비꼼 |
| 정보량 | 많이 말하는 사람 vs 핵심만 |
| 감정 표현 | 직접적 vs 돌려 말하기 vs 침묵 |

## 5. 아키타입 실패 패턴 (모든 장르 공통)

| 실패 유형 | 설명 | 해결책 |
|----------|------|--------|
| 메리 수/게리 스튜 | 결점 없는 완벽 캐릭터 | 의미 있는 약점 부여 |
| 종이 인형 | 설정만 있고 행동에 반영 안 됨 | 설정을 행동으로 보여주기 |
| 클리셰 껍데기 | 아키타입을 그대로 복사 | 예상을 뒤집는 특성 1개 추가 |
| 동기 부재 | 왜 행동하는지 불명확 | 목표+결핍+두려움 설정 |
| 정적(靜的) 인물 | 처음과 끝이 같음 | 변화 아크(Arc) 설계 |
| 독자 대리 인형 | 개성 없이 독자 투영용 | 고유한 관점/반응 부여 |

## 6. 장르별 주요 아키타입 조합

| 장르 | 주인공 | 핵심 조연 | 악역 |
|------|--------|----------|------|
| 하이 판타지 | 영웅/평범인 | 멘토+파티 | 신념형/거울형 |
| 로판 | 재기형/치유자 | 연인+적대자 | 사회/가문 |
| 무협 | 영웅/반영웅 | 사형제+멘토 | 카리스마형 |
| 미스터리 | 탐구자/트릭스터 | 동료(왓슨형) | 거울형/지능형 |
| 호러 | 평범인/생존자 | 희생자들 | 미지형/내면형 |
| 현대물 | 평범인 | 동료+연인 | 사회/자기자신 |
| 회귀물 | 회귀자(반영웅) | 미래의 적→동료 | 거울형/시스템형 |
