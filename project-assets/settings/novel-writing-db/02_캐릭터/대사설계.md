# 대사 설계

> 좋은 대사는 캐릭터를 살리고, 나쁜 대사는 몰입을 깬다.
> 캐릭터성 반영, 대화 리듬, 정보 전달, 감정 표현의 기술.

---

## 1. 대사의 4대 기능

| 기능 | 설명 | 비율 (이상적) |
|------|------|-------------|
| **캐릭터 표현** | 인물의 성격/배경/감정을 드러냄 | 40% |
| **서사 추진** | 갈등 유발, 결정, 행동 촉발 | 30% |
| **정보 전달** | 세계관/상황/배경 정보 전달 | 20% |
| **분위기 조성** | 유머, 긴장, 여운 만들기 | 10% |

**[비인기]** 실패: 대사의 90%가 정보 전달 → "설명 대사" → 독자 이탈

## 2. 캐릭터별 대사 차별화

### 2.1 음성(Voice) 설계 변수

| 변수 | 범위 | 예시 |
|------|------|------|
| 어휘 수준 | 학술적 ↔ 구어적 | "전략적 후퇴가 필요하다" vs "그냥 빠지자" |
| 문장 길이 | 장문형 ↔ 단문형 | 수다쟁이 vs 과묵형 |
| 직접성 | 직설적 ↔ 우회적 | "싫어" vs "생각해볼게" |
| 감정 표현 | 풍부 ↔ 절제 | "너무 좋아 정말!" vs "...괜찮네" |
| 격식 수준 | 격식체 ↔ 비격식 | "보고하겠습니다" vs "들어봐" |
| 고유 어구 | 말버릇, 감탄사 | "흥미롭군" / "아이고야" / "쯧" |
| 화제 선호 | 주로 말하는 주제 | 전략가→분석, 전사→행동, 학자→지식 |

### 2.2 대사만으로 캐릭터 구분 테스트

**[필수]** 지문(tag)을 지우고 대사만 읽었을 때 누가 말하는지 알 수 있어야 함.

**좋은 예**:
```
"확률적으로 이 문을 여는 것은 자살 행위야. 데이터를 더 모으자."
"데이터고 뭐고, 열면 알잖아. 비켜."
"...둘 다 일리가 있어. 하지만 안에 누가 있을지 몰라. 조심하면서 열자."
```
→ 각 대사의 주인이 명확

**나쁜 예**:
```
"우리 저기 가자."
"그래, 가자."
"좋아, 가보자."
```
→ 누가 누군지 구분 불가

## 3. 대화 리듬 설계

### 3.1 대화의 템포

**빠른 대화** (갈등/논쟁/긴박):
```
"그건 네 판단이야."
"아냐, 사실이야."
"사실이면 왜 숨겼어?"
"숨긴 게 아니라—"
"아니, 숨겼어."
```
→ 짧은 대사 교환, 지문 최소, 정보량 높음

**느린 대화** (감정/고백/회상):
```
"...언제부터였어?"

오랜 침묵이 흘렀다. 그가 창밖을 보며 말했다.

"아마... 처음 만난 날부터."
```
→ 대사 사이에 지문/묘사 삽입, 여백이 감정을 키움

### 3.2 침묵의 대사

**[인기]** 말하지 않는 것이 대사보다 강력할 때:
```
"미안해."
그는 대답하지 않았다. 다만 등을 돌렸다.
```

**침묵의 유형**:
| 유형 | 의미 | 서사적 효과 |
|------|------|------------|
| 분노의 침묵 | 할 말이 너무 많아서 못 함 | 긴장 고조 |
| 슬픔의 침묵 | 말로는 표현 불가 | 감정 깊이 |
| 동의의 침묵 | 말하지 않아도 통함 | 관계 깊이 |
| 거부의 침묵 | 대답할 가치 없음 | 경멸/거리감 |
| 생각의 침묵 | 판단 중 | 지적 무게 |

### 3.3 대사와 지문의 비율

| 장면 유형 | 대사 : 지문 | 이유 |
|----------|------------|------|
| 일상 대화 | 7 : 3 | 자연스러운 흐름 |
| 감정적 대화 | 4 : 6 | 지문으로 감정 표현 |
| 논쟁/갈등 | 8 : 2 | 빠른 리듬 |
| 정보 전달 대화 | 5 : 5 | 설명 대사 방지 위해 반응 삽입 |
| 고백/고해 | 3 : 7 | 내면 묘사가 핵심 |

## 4. 정보 전달 대사의 기술

### 4.1 설명 대사(Exposition)의 금기

**[금기]** "너도 알다시피" 증후군:
```
❌ "너도 알다시피, 이 왕국은 300년 전 대전쟁에서..."
→ 알고 있는 사람에게 설명할 이유 없음
```

**[금기]** 독백 강연:
```
❌ "이 마법은 세 가지 원리로 작동해. 첫째는... 둘째는... 셋째는..."
→ 강의실이 아님
```

### 4.2 자연스러운 정보 전달 기법

**기법 1: 갈등 속 정보**
```
"왜 그 길로 못 가냐고? 뇌지가 깔려 있거든. 전쟁 때 매설된 거야."
→ 세계관 정보(전쟁 역사)가 갈등(길 선택) 속에서 자연스럽게
```

**기법 2: 무지한 인물 활용**
```
"프리즈마폴이 뭐예요?"
"넌 학교에서 뭘 배운 거야... 쉽게 말하면 에너지 연결 통로야."
→ 초보 인물의 질문이 독자 대리
```

**기법 3: 반박/질문 삽입**
```
"이 지역은 안전해."
"안전? 지난달에 크로마스톰이 지나간 곳이잖아."
"그래서 지금은 안전하다는 거야. 한번 지나간 곳은 당분간..."
→ 정보가 대화의 공방 속에서 전달
```

**기법 4: 감정과 결합**
```
"또 그 이야기야? 대전쟁, 대전쟁... 할아버지가 그 전쟁에서 돌아오지 못했거든."
→ 세계관 정보 + 캐릭터 감정이 동시에 전달
```

## 5. 대사의 서브텍스트 (Subtext)

### 5.1 정의
> 말하는 내용(Text) 아래에 숨겨진 진짜 의미(Subtext)

**좋은 대사 = 겉으로 말하는 것 ≠ 진짜 의미하는 것**

```
겉(Text): "배고프지 않아."
속(Subtext): "네가 걱정하는 거 알아. 하지만 나는 괜찮아."

겉: "내일 뭐 해?"
속: "너를 만나고 싶어."

겉: "네가 알아서 해."
속: "실망했어. 더 이상 기대 안 해."
```

### 5.2 서브텍스트 생성법

| 기법 | 설명 | 예시 |
|------|------|------|
| 회피 | 핵심 주제를 피하며 주변을 돌림 | 이별 화제 대신 날씨 이야기 |
| 과잉 반응 | 사소한 것에 과민 반응 | "왜 그런 걸 물어!" (숨기는 것이 있음) |
| 과소 반응 | 큰 일에 무덤덤 | "알았어." (감정 억압 중) |
| 대리 화제 | 다른 것에 대해 말하면서 진짜를 표현 | "이 꽃 금방 시들겠네" = 이 관계도 |
| 행동 불일치 | 말과 행동이 다름 | "괜찮아" (눈물을 닦으며) |

**[인기]** 서브텍스트가 풍부한 대사 → 독자가 "진짜 의미"를 읽는 재미
**[비인기]** 모든 대사가 직설적 → 단조롭고 유치한 느낌

## 6. 장르별 대사 특성

| 장르 | 대사 특징 | 핵심 |
|------|----------|------|
| 판타지 | 세계관 용어 녹임, 격식/비격식 혼용 | 과도한 세계관 용어 주의 |
| 로맨스 | 서브텍스트 풍부, 감정 간접 표현 | "말하지 않는 것"이 핵심 |
| 액션/무협 | 짧고 강렬, 전투 중 최소 대사 | 전투 중 장문 대사 금지 |
| 미스터리 | 이중 의미, 단서 삽입 | 모든 대사가 단서일 수 있음 |
| 일상물 | 자연스러운 구어체, 사소한 수다 | 일상에서 의미 발견 |
| 호러 | 짧고 불안한 대사, 비명/침묵 | 공포는 설명하면 줄어듦 |
| 코미디 | 타이밍, 반복, 반전, 언어유희 | 대사의 "간격"이 웃음 포인트 |

## 7. 대사 자가 점검 체크리스트

- [ ] 지문 없이 대사만 읽었을 때 화자가 구분되는가?
- [ ] "너도 알다시피"형 설명 대사가 없는가?
- [ ] 모든 대사가 4대 기능 중 최소 1개를 수행하는가?
- [ ] 서브텍스트가 있는 대사가 전체의 30% 이상인가?
- [ ] 대화 리듬(빠름↔느림)이 장면에 맞는가?
- [ ] 침묵이 대사만큼 효과적으로 사용되고 있는가?
- [ ] 정보 전달이 갈등/감정과 결합되어 있는가?
- [ ] 캐릭터마다 고유 어구/말버릇이 있는가?
