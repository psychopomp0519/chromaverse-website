# 한국어 특화 기법

> 한국어로 소설을 쓴다는 것은 한국어의 고유한 리듬과 구조를 무기로 쓰는 것.
> 번역체를 걷어내고, 한국어만의 미학을 살리는 기술.

---

## 1. 종결어미의 서사적 활용

### 1.1 종결어미 = 장면의 톤

| 종결어미 | 분위기 | 적합 장면 |
|----------|--------|----------|
| ~했다/~였다 | 안정적, 서술적 | 일반 서술, 회고 |
| ~한다/~는다 | 현재감, 긴박함 | 액션, 현재 시점 |
| ~하는 것이다 | 설명적, 차분 | 세계관 서술 (주의: 과다 시 번역체) |
| ~해버렸다/~고 말았다 | 돌이킬 수 없는 행위 | 결정적 순간, 후회 |
| ~했을까 | 의문, 성찰 | 회고, 철학적 장면 |
| ~더라/~더군 | 회상, 경험 전달 | 1인칭 회고 |
| ~(이)라니 | 놀라움, 감탄 | 반전, 충격 |

### 1.2 종결어미 다양화 규칙

**[필수]** 같은 종결어미 3회 연속 반복 금지

```
나쁜 예:
"카이는 걸었다. 길은 어두웠다. 바람이 불었다." → ~다 3연속

좋은 예:
"카이는 걸었다. 길이 어둡다. 바람이 불어온다." → 과거→현재→현재 변환
```

**다양화 기법**:
- 서술문 + 의문문 + 감탄문 교대
- 장문(~하고 있었다) + 단문(멈췄다) 교대
- 종결어미 생략(명사형 종결) 활용: "끝없는 어둠." "차가운 바람."

## 2. 존댓말/반말 전환의 서사적 활용

### 2.1 호칭 체계와 관계 변화

| 전환 | 의미 | 서사적 효과 |
|------|------|-----------|
| 존대→반말 | 거리 좁힘, 친밀해짐 | 관계 발전의 마커 |
| 반말→존대 | 거리 두기, 차가워짐 | 갈등/단절의 신호 |
| 혼용 | 관계가 불안정/과도기 | 긴장감 |
| 일관된 존대 | 절대 넘지 않는 선 | 상하관계/벽 강조 |
| 갑작스런 반말 | 감정 폭발, 통제 상실 | 극적 전환점 |

### 2.2 호칭 변화 장면 설계

```
예시: 항상 "~님"으로 부르던 인물이 처음으로 이름을 부르는 순간

"...코다."
처음이었다. 그 이름을 '님'도 '씨'도 없이 부른 것은.
코다의 눈이 미세하게 흔들렸다.
→ 호칭 하나의 변화가 한 페이지의 묘사보다 강력한 감정 전달
```

## 3. 번역체 회피

### 3.1 흔한 번역체 패턴과 교정

| 번역체 | 자연스러운 한국어 | 원인 |
|--------|----------------|------|
| "~하는 것이 가능하다" | "~할 수 있다" | 영어 be able to 직역 |
| "~임에 틀림없다" | "~일 것이다/~겠지" | 일어 に違いない 직역 |
| "그녀는 ~했다" | "~는/이름은 ~했다" | '그녀' 과다 사용 (번역체) |
| "나는 ~라고 생각했다" | "~일까 싶었다/~겠지" | I thought 직역 |
| "~에 대해 생각하다" | "~을 생각하다" | about 직역 |
| "~하기 시작했다" | "~하기 시작" 없이 "~했다" | 과도한 '시작' 사용 |
| "~인 것 같다" 남발 | 단정 표현으로 교체 | 영어 seem to 영향 |
| "그리고 ~했다" 반복 | 접속 없이 연결 | and 직역 |

### 3.2 한국어다운 문장의 특징

| 특징 | 설명 |
|------|------|
| **주어 생략** | 한국어는 주어 생략이 자연스러움 ("나는" 반복 불필요) |
| **조사의 미세 차이** | "을/를" vs "은/는" — 초점의 차이 |
| **어순의 유연성** | 강조하고 싶은 요소를 앞으로 |
| **명사형 종결** | "끝없는 밤." — 동사 없이 정지된 이미지 |
| **연결어미의 풍부함** | ~며, ~고, ~는데, ~지만 → 미세한 논리 관계 표현 |

## 4. 사투리/방언 활용

### 4.1 사투리 = 캐릭터 차별화 도구

| 지역 | 특징 | 캐릭터 인상 |
|------|------|-----------|
| 경상도 | 억양 강함, 단호, 직설적 | 무뚝뚝하지만 속 깊은 |
| 전라도 | 끝올림, 정감 있음 | 인간적, 따뜻한 |
| 충청도 | 느리고 여유로운 | 느긋하지만 날카로운 |
| 제주도 | 독특한 어휘, 이국적 | 신비로운, 고유한 |
| 강원도 | 투박하고 소박한 | 순박한, 자연적 |

### 4.2 사투리 사용 규칙

**[필수]** 규칙:
1. **핵심 대사에만** — 모든 대사를 사투리로 쓰면 가독성 하락
2. **변환표 작성** — 캐릭터별 사투리 단어/어미 목록 유지
3. **일관성** — 같은 캐릭터의 사투리가 장면마다 달라지면 안 됨
4. **고증** — 실제 해당 지역 사투리와 맞는지 확인

**[금기]** 사투리 = 희화화 도구로만 쓰지 않기. 사투리 캐릭터에도 진지한 장면을 부여.

## 5. 세대별 말투

### 5.1 세대별 특성

| 세대 | 말투 특징 | 대사 예시 |
|------|----------|----------|
| 10대 | 줄임말, 유행어, 빠른 전환 | "ㄹㅇ? 대박인데" |
| 20대 | 반존대, 드립, 자기 비하 유머 | "그건 좀 아닌 것 같은데요..." |
| 30대 | 직장어, 완곡 표현, 피로감 | "그 건은 검토해 보겠습니다" |
| 40대 | 비유 증가, 경험 기반 | "젊었을 때는 나도..." |
| 50대+ | 격식체, 한자어, 관용구 | "사필귀정이라고 하지 않던가" |
| 60대+ | 옛 표현, 존대 습관, 느린 템포 | "그래, 그런 게여..." |

### 5.2 세대별 말투 활용 팁

- 세대 차이는 **대화의 갈등 요소**로 활용 가능
- 같은 단어를 세대별로 다르게 사용하면 자연스러운 세대 갈등 표현
- 유행어는 **시대 설정과 일치**해야 함 (2026년 소설에 2020년 유행어 X)

## 6. 한국어 문장 리듬

### 6.1 호흡 단위

한국어 문장의 자연스러운 호흡:
```
짧은 호흡 (4~10자): 긴박, 단절
"멈췄다. 돌아봤다. 아무도 없었다."

중간 호흡 (15~30자): 서술, 균형
"카이는 천천히 고개를 돌려 창 밖을 바라보았다."

긴 호흡 (35~50자): 몰입, 흐름
"그때의 빛은 마치 오래된 기억처럼 희미하게 흔들리며 방 안을 가득 채우고 있었다."
```

### 6.2 리듬 변화의 규칙

**[인기]** 한국어 소설의 좋은 리듬:
- 3~4개의 중간 호흡 후 1개의 짧은 호흡 → 강조
- 긴 호흡으로 분위기 조성 → 짧은 호흡으로 전환 → 충격
- 대화(짧은) ↔ 묘사(긴) 교대

```
좋은 예:
"길고 어두운 복도를 걸어가는 동안, 발소리만이 벽에 부딪혀 돌아왔다.
차갑다. 공기가, 벽이, 이 세계 전부가."
→ 긴 호흡 → 극단적 짧은 호흡 = 감정 전환
```

## 7. 조사의 미학

### 7.1 은/는 vs 이/가

| 조사 | 기능 | 효과 |
|------|------|------|
| **은/는** | 주제/대조 | "카이**는** 걸었다" → 다른 사람은 아닌, 카이가 |
| **이/가** | 초점/신정보 | "카이**가** 걸었다" → 걷고 있는 사람이 카이다 |

```
"카이는 멈추었다." → 카이에 대한 서술 (일반적)
"카이가 멈추었다." → 멈춘 사람이 카이임을 강조 (새 정보)
"멈추었다." → 주어 생략 (가장 몰입적)
```

### 7.2 조사 생략의 효과

```
"바람이 불었다." → 서술적
"바람, 불었다." → 시적, 파편적
"바람." → 극한의 절제
```
