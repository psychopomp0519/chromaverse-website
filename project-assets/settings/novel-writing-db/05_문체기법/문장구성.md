# 문장 구성

> 문장은 서사의 최소 단위. 문장의 길이, 구조, 리듬이 이야기의 톤을 결정한다.

---

## 1. 문장 길이와 효과

### 1.1 길이별 특성

| 길이 | 어절 수 | 효과 | 적합한 장면 |
|------|---------|------|------------|
| 극단문 | 1~2어절 | 충격, 정지, 강조 | 반전, 결정타, 죽음 |
| 단문 | 3~5어절 | 속도감, 긴장, 행동 | 전투, 추격, 위기 |
| 중문 | 6~15어절 | 균형, 서술, 정보 | 대화, 서술, 설명 |
| 장문 | 16~25어절 | 분위기, 몰입, 서정 | 풍경, 감정, 회상 |
| 초장문 | 26어절+ | 의식의 흐름, 압도 | 실험적 문학, 내면 폭발 |

### 1.2 리듬 변주

**[필수]** 같은 길이의 문장이 3개 이상 연속되면 리듬이 죽음

```
❌ 단조로운 리듬:
"그가 걸었다. 바람이 불었다. 해가 졌다. 밤이 왔다."
→ 균일한 단문 반복 = 무미건조

✅ 변주된 리듬:
"그가 걸었다. 끝이 보이지 않는 길 위로 바람이 불어왔고, 모래가 발목을 잡았다. 해가 졌다."
→ 단문-장문-단문 = 리듬감
```

### 1.3 강조를 위한 길이 대비

**장문 후 단문 = 강조**:
```
"오랜 세월 동안 누구도 그 문을 열지 못했다. 수백 명의 기사가 시도했고, 수천 번의 마법이 쏟아졌으며, 왕국 최고의 현자들이 머리를 맞대었다.
그가 손잡이를 잡았다.
문이 열렸다."
```

**단문 연쇄 후 장문 = 이완**:
```
"달렸다. 숨이 찼다. 뒤돌아볼 수 없었다. 다리가 풀렸다.
그리고 한참이 지나 — 나무 뒤에 몸을 숨기고, 거칠어진 숨을 고르며 — 그는 비로소 살아 있다는 것을 실감했다."
```

## 2. 문장 구조 기법

### 2.1 능동태 vs 수동태

| 태 | 효과 | 예시 |
|----|------|------|
| 능동 | 에너지, 행동, 직접성 | "그가 검을 뽑았다" |
| 수동 | 무력감, 거리감, 객관성 | "검이 뽑혔다" |

**[인기]** 원칙: 기본은 능동태, 수동태는 의도적 효과를 위해 사용
- 캐릭터가 무력한 상황 → 수동태
- 행동의 주체를 숨기고 싶을 때 → 수동태

### 2.2 문장 시작의 다양화

**[금기]** 같은 주어로 연속 시작:
```
❌ "그는 일어났다. 그는 옷을 입었다. 그는 문을 열었다."
✅ "일어나서 옷을 걸쳤다. 문이 열리자 차가운 공기가 밀려왔다."
```

**문장 시작 변주 도구**:
| 시작 | 예시 |
|------|------|
| 행동 | "검을 들어올리며 그가 말했다." |
| 감각 | "바람 냄새가 달라졌다." |
| 시간 | "해가 중천에 이르렀을 때," |
| 대사 | "'돌아가자.' 그가 말했다." |
| 장소 | "광장 한가운데 서 있는 것은" |
| 접속 | "하지만 그것은 시작에 불과했다." |

### 2.3 종결어미 다양화 (한국어 특화)

**[금기]** 같은 종결어미 3회 연속:
```
❌ "~했다. ~했다. ~했다."
✅ "~했다. ~이었다. ~였을까."
```

**주요 종결어미와 효과**:
| 종결어미 | 효과 |
|---------|------|
| ~했다/~했다 | 가장 기본, 서술적 |
| ~이었다/~였다 | 상태/존재 강조 |
| ~일까/~였을까 | 의문, 독자 참여 유도 |
| ~(이)다 | 단정, 강한 주장 |
| ~지 않았다 | 부정으로 긍정 강조 |
| ~인 것이다 | 설명적, 격식 |
| ~하고 있었다 | 진행, 현장감 |
| 체언 종결 | "소리. 그것뿐이었다." — 극적 강조 |

## 3. 비유와 수사

### 3.1 비유의 3대 원칙

1. **구체적**: 추상적 비유보다 감각적 비유
   - ❌ "슬픔이 물결처럼 밀려왔다"
   - ✅ "가슴에 무언가 차가운 것이 흘러내렸다. 삼킨 얼음물 같았다."

2. **신선한**: 관용적 비유 회피
   - ❌ "눈처럼 하얀 피부" (진부)
   - ✅ "돌 밑에서 자란 새싹 같은 창백함" (새로움)

3. **일관된**: 한 장면에서 비유의 이미지 계열 통일
   - ❌ "불같은 분노가 얼음처럼 가라앉았다" (불+얼음 혼재)
   - ✅ "불같은 분노가 타오르다 재가 되었다" (불 계열 유지)

### 3.2 비유 과다 경계

**[주의]** 비유는 양념 — 문장마다 비유를 넣으면 과식
- 핵심 장면에 1~2개 강력한 비유 > 매 문장 약한 비유 다수
- 비유 없는 직접 서술도 강력할 수 있음

## 4. 문체 유형별 특성

| 문체 | 특성 | 적합 장르 |
|------|------|----------|
| 건조체 | 감정 배제, 사실만 서술 | 미스터리, 스릴러, 하드보일드 |
| 서정체 | 감각적, 서정적, 아름다운 문장 | 문학, 판타지, 로맨스 |
| 구어체 | 말하듯 자연스러운 | 일상물, 1인칭 서술 |
| 격문체 | 짧고 강렬, 에너지 | 액션, 무협 |
| 유머체 | 가벼우면서 재치 | 코미디, 일상물 |
| 관조체 | 거리를 둔 관찰자 | 군상극, 사회 드라마 |
