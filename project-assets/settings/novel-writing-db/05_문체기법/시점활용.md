# 시점 활용

> 누구의 눈으로 이야기를 보여줄 것인가. 시점 선택이 서사의 모든 것을 바꾼다.

---

## 1. 시점 유형과 특성

### 1.1 1인칭 시점

**특성**: "나"가 직접 이야기. 내면 접근 자유, 정보 제한.

| 장점 | 단점 |
|------|------|
| 감정 이입 극대화 | 다른 인물 내면 접근 불가 |
| 독특한 목소리/개성 | 주인공 부재 장면 서술 어려움 |
| 비밀/반전에 유리 | 객관적 묘사 제한 |

**[인기]** 핵심: 화자의 개성이 강할수록 효과적
**[비인기]** 실패: "나는 ~했다. 나는 ~느꼈다" 반복 → 단조

**적합**: 미스터리, 성장물, 회고, 일기체, 서간체

### 1.2 3인칭 제한 시점 (Close Third)

**특성**: "그/그녀"로 서술하지만 특정 인물의 시야/생각에 제한.

| 장점 | 단점 |
|------|------|
| 1인칭의 몰입 + 3인칭의 유연성 | 시점 인물 외 내면 불가 |
| 시점 전환으로 다층 서사 가능 | 시점 혼란 주의 필요 |
| 가장 대중적, 읽기 편함 | — |

**[인기]** 현대 소설의 가장 인기 있는 시점
**[필수]** 시점 전환 시 장/챕터 단위로 → 한 장면 안에서 시점 전환 금지

**적합**: 모든 장르에 범용

### 1.3 3인칭 전지적 시점 (Omniscient)

**특성**: 모든 인물의 내면/모든 장소/과거와 미래에 접근 가능.

| 장점 | 단점 |
|------|------|
| 자유로운 정보 제공 | 서스펜스 구축 어려움 |
| 서사적 거리감, 서사시적 분위기 | 감정 이입 약화 |
| 아이러니/논평 가능 | "신 시점" 느낌 → 현대 독자에게 거리감 |

**[주의]** 전지적 ≠ 무작위 시점 전환. 전지적 화자도 **일관된 목소리**를 가져야 함.

**적합**: 대하소설, 군상극, 동화/우화, 고전적 서사

### 1.4 2인칭 시점

**특성**: "당신/너"로 서술. 독자를 캐릭터로 만듦.

| 장점 | 단점 |
|------|------|
| 즉각적 몰입감 | 오래 지속 어려움 |
| 실험적, 독특한 효과 | 독자가 거부감 느낄 수 있음 |

**적합**: 단편, 실험적 작품, 인터랙티브 소설, 게임 서사

### 1.5 다시점 (Multiple POV)

**특성**: 여러 인물의 시점을 교대.

**효과적 운영 규칙**:
1. 시점 인물은 3~5명 이내 (장편도 5명 초과 주의)
2. 시점 전환은 장/챕터 단위
3. 각 시점 인물의 목소리가 구분되어야 함
4. 모든 시점이 서사에 필수적이어야 함 (불필요한 시점 = 삭제)

## 2. 시점 선택 가이드

| 목적 | 추천 시점 | 이유 |
|------|----------|------|
| 독자 감정 이입 극대화 | 1인칭 / 3인칭 제한 | 내면 접근 |
| 복잡한 서사, 다수 인물 | 다시점 3인칭 제한 | 다양한 시각 |
| 미스터리/서스펜스 | 1인칭 / 제한 3인칭 | 정보 제한 → 긴장 |
| 서사시적 웅장함 | 전지적 3인칭 | 넓은 시야 |
| 실험적/문학적 | 2인칭 / 비선형 | 독특한 경험 |

## 3. 시점 실수와 금기

### 3.1 시점 침범 (Head-Hopping)

**[금기]** 한 장면 안에서 시점 전환:
```
❌ "그는 그녀가 미워졌다. 한편 그녀는 그의 마음도 모르고 창밖을 보았다."
→ 한 문장 만에 그→그녀로 시점 이동
```

### 3.2 시점 인물이 모를 정보 서술

**[금기]** 1인칭/제한 3인칭에서 시점 인물이 볼 수 없는 것 묘사:
```
❌ (1인칭) "내가 등을 돌리는 순간, 그는 주머니에서 칼을 꺼냈다."
→ 등을 돌렸으면 볼 수 없음
```

### 3.3 감정의 직접 판독

**[금기]** 제한 시점에서 타인의 감정을 확정적으로 서술:
```
❌ "그녀는 그가 슬퍼하고 있다는 것을 알았다."
✅ "그녀가 보기에 그는 슬퍼 보였다."
→ 관찰/추측으로 전달해야 시점 유지
```

## 4. 신뢰할 수 없는 화자 (Unreliable Narrator)

### 4.1 유형

| 유형 | 설명 | 효과 |
|------|------|------|
| 의도적 기만 | 의식적으로 사실을 숨김/왜곡 | 반전, 미스터리 |
| 인지적 한계 | 어린이, 정신 질환 등으로 정확한 인식 불가 | 행간 읽기의 재미 |
| 자기 기만 | 자신에게 거짓말 | 캐릭터 깊이, 비극 |
| 기억의 불완전 | 회고 시점에서 기억 왜곡 | 불확실성의 긴장 |

### 4.2 운영 규칙

**[필수]**:
- 독자에게 "이 화자를 의심할 수 있는" 단서를 제공
- 화자의 왜곡에도 일관된 패턴이 있어야 함
- 진실이 밝혀질 때 모든 것이 재해석 가능해야 함
