# 유머와 코미디 기법

> 긴장 속의 웃음이 가장 강력하다. 유머는 선택이 아닌 필수 기술.
> 웹소설에서 유머는 독자 유지율에 직접적 영향.

---

## 1. 유머의 서사적 역할

| 역할 | 설명 |
|------|------|
| 텐션 완화 | 고조된 긴장감을 잠시 이완 → 다음 긴장이 더 강력 |
| 캐릭터 호감 | 유머러스한 캐릭터 = 호감 상승 |
| 관계 표현 | 농담을 주고받는 관계 = 친밀도 표현 |
| 분위기 전환 | 무거운 장면 후 가벼움으로 리셋 |
| 재독 동기 | 재밌는 대사/장면 = 다시 읽고 싶음, 공유하고 싶음 |

---

## 2. 소설에서 사용 가능한 유머 기법

### 2.1 상황 유머 (Situational Comedy)
- 캐릭터가 의도하지 않은 상황의 우스꽝스러움
- **예**: 최강 마법사가 일상적인 일(요리, 빨래)에 무능
- **[인기]** 강한 캐릭터의 예상 밖 약점 = 가장 안전하고 효과적

### 2.2 대사 유머 (Verbal Humor)
- 재치있는 응수, 언어유희, 비꼼
- **예**: "너 나한테 반했지?" "...귀에 물이 들어간 건 맞아"
- **[팁]** 캐릭터 성격에 맞는 유머 스타일 부여

### 2.3 갭 모에 (Gap Humor)
- 기대와 현실의 불일치
- **예**: 무서워 보이는 전사가 고양이에 약함
- **[인기]** 웹소설에서 가장 사랑받는 유머 유형

### 2.4 쓴웃음/블랙 유머
- 어두운 상황 속 자조적 웃음
- **예**: 절망적 상황에서 "...적어도 날씨는 좋네"
- **[주의]** 톤에 맞게 사용, 부적절한 상황에서는 독자 반감

### 2.5 메타 유머/4th Wall
- 장르 문법을 자각하는 유머
- **예**: "이런 상황이면 보통 함정이 있는데" (그리고 정말 함정)
- **[주의]** 과다 시 몰입 파괴, 가끔 사용해야 효과적

### 2.6 반복/콜백 유머
- 이전에 나온 농담/상황이 다른 맥락에서 다시 등장
- **예**: 10화에서 한 실수가 50화에서 다시 언급될 때
- **[인기]** 독자가 "아, 그때 그거!" 하며 웃는 순간 = 충성도 상승

### 2.7 리액션 유머
- 과장된 반응, 탈력감, 무표정
- **예**: 주인공의 비상식적 행동에 주변이 경악
- **[인기]** 웹소설/만화에서 가장 흔하고 효과적

---

## 3. 유머 배치 타이밍

| 타이밍 | 효과 |
|--------|------|
| 전투 직전 | 긴장의 인간화 ("떨리는 거 아냐?") |
| 전투 직후 | 이완, 생존의 안도 |
| 일상/야영 장면 | 캐릭터 관계 심화 |
| 위기 중 | 긴장+웃음의 동시 효과 (고급 기법) |
| 장면 전환 시 | 분위기 리셋 |

**[필수]** 유머 빈도:
- 심각한 작품: 20~30화마다 유머 장면 1회
- 보통 작품: 5~10화마다 1회
- 가벼운 작품: 매 1~2화

---

## 4. 유머의 금기

| 금기 | 이유 |
|------|------|
| 감동 장면 직후 유머 | 감동 무력화 |
| 캐릭터 격하 유머 | 캐릭터 매력 하락 |
| 같은 농담 3회 이상 | 재미 소멸 |
| 민감한 주제 농담 | 독자 반감 |
| 유머가 플롯을 방해 | 진행 방해 → 불필요 |
