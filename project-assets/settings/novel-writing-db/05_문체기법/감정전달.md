# 감정 전달

> 독자의 가슴을 울리는 기술. 감정선 설계, 카타르시스, 감정의 간접 표현.

---

## 1. 감정 전달의 원칙

### 1.1 Show > Tell (감정 특화)

| Tell (직접) | Show (간접) |
|------------|------------|
| "그는 분노했다" | "그의 손이 떨리고 있었다. 이를 악물어 관자놀이가 움찔거렸다." |
| "그녀는 기뻤다" | "그녀는 아무 말 없이 웃었다. 눈에 물기가 맺혔지만 닦지 않았다." |
| "슬펐다" | "목이 조여들었다. 말이 나오지 않았다." |

### 1.2 감정의 간접 전달 채널

| 채널 | 방법 | 효과 |
|------|------|------|
| 신체 반응 | 심장, 호흡, 근육, 온도 변화 | 가장 직관적 |
| 행동 | 의식적/무의식적 행동 | 성격 반영 |
| 환경 투영 | 풍경/날씨에 감정 반영 | 분위기 조성 |
| 대사의 서브텍스트 | 말하지 않는 것으로 | 깊이, 재독 가치 |
| 물건/상징 | 특정 물건에 감정 투영 | 지속적 감정 환기 |
| 침묵/부재 | 말하지 않음, 행동하지 않음 | 때로 가장 강력 |

## 2. 감정선 설계

### 2.1 감정 곡선 모델

```
기대 → 불안 → 고조 → 절정 → 해소 → 여운
```

**[필수]** 감정은 "만들어지는" 것이 아니라 "쌓이는" 것:
- 감동적 장면이 감동적이려면 **이전 50화의 축적**이 필요
- 축적 없이 감동 시도 → "왜 울어야 하는지 모르겠다"

### 2.2 감정 투자 (Emotional Investment)

| 투자 단계 | 방법 | 역할 |
|----------|------|------|
| 공감 형성 | 캐릭터의 결핍/고통 제시 | "저 사람이 안됐다" |
| 호감 구축 | 작은 선행/유머/약점 보여주기 | "저 사람이 좋다" |
| 목표 공유 | 독자가 캐릭터의 목표를 응원 | "성공하면 좋겠다" |
| 위기 노출 | 소중한 것이 위협받음 | "제발 무사해라" |
| 카타르시스 | 위기 극복/목표 달성 | 감정 폭발 |

### 2.3 카타르시스 설계

**카타르시스 = 축적된 감정의 방출**

**[인기]** 카타르시스의 3요소:
1. **충분한 축적**: 감정을 충분히 쌓아야 터짐
2. **지연**: 해소를 최대한 늦춤 (참을수록 강력)
3. **트리거**: 사소한 계기가 큰 감정을 터뜨림

```
예: 내내 참고 있던 인물이 — 사소한 친절 한마디에 — 울음을 터뜨림
→ 독자도 함께 울게 됨
```

## 3. 감정 유형별 전달 기법

### 3.1 슬픔/상실
- **핵심**: 직접 "슬프다"고 쓰지 않을수록 강력
- **기법**: 일상의 부재 묘사 ("두 사람분 차를 끓이다 한 잔을 부었다"), 물건/장소/습관에 남은 흔적

### 3.2 기쁨/감동
- **핵심**: 기쁨의 전 단계(고통/부재)가 깊을수록 기쁨이 큼
- **기법**: 대비 활용 — 최악의 순간 직후 기쁜 순간, 미세한 반응(손떨림, 눈물, 웃음)

### 3.3 분노/정의감
- **핵심**: 독자가 먼저 분노해야 캐릭터의 분노에 공감
- **기법**: 불의를 충분히 보여준 후 → 정의 실현 = 사이다

### 3.4 두려움/불안
- **핵심**: 구체적 위협보다 모호한 위협이 더 무서움
- **기법**: 감각 제한(어둠, 안개), 정보 부족, 익숙한 것의 이상함

### 3.5 사랑/그리움
- **핵심**: 함께 있을 때보다 부재/거리에서 더 강하게 전달
- **기법**: 상대의 흔적, 기억의 감각화(향기/소리), 무의식적 행동

## 4. 감정 과잉/과소 진단

### 4.1 감정 과잉 (Melodrama)

| 증상 | 처방 |
|------|------|
| 매 장면 눈물/분노 | 감정의 경중 조절, 절제 |
| 독자가 공감 못하는데 캐릭터만 울음 | 감정 투자 단계 보충 |
| 감정 형용사 남발 ("너무" "정말" "미칠 듯이") | 형용사 삭제, 행동으로 |
| 배경 음악 없이도 울어야 함 | 장면의 내적 설득력 점검 |

### 4.2 감정 과소 (Coldness)

| 증상 | 처방 |
|------|------|
| 중요한 순간인데 감정이 안 느껴짐 | 장면 확대, 감각 묘사 추가 |
| 캐릭터가 기계적으로 행동 | 내면 반응 삽입 |
| 사건은 큰데 독자가 무관심 | 캐릭터-독자 간 감정 투자 부족 |

## 5. 눈물의 문법 (울리는 장면의 설계)

### 5.1 울리는 장면의 공식

```
1. 독자가 캐릭터를 좋아하게 만든다 (호감 구축)
2. 캐릭터에게 소중한 것을 보여준다 (스테이크 설정)
3. 소중한 것을 위협한다 (불안 축적)
4. 캐릭터가 소중한 것을 위해 무언가를 희생한다 (고구마)
5. 사소한 트리거가 축적된 감정을 터뜨린다 (카타르시스)
```

### 5.2 "작은 것이 큰 울림"의 법칙

```
대사 한마디: "잘했어." (인정받지 못한 인물에게)
행동 하나: 말없이 외투를 벗어 어깨에 걸쳐줌
물건 하나: 쓰다 남은 연필 (떠난 사람의 흔적)
```

**[인기]** 거창한 연설보다 **사소한 디테일**이 울림이 큼
