# 전투와 액션 장면 묘사

> 전투는 "움직임"이 아니라 "선택"의 연속이다.
> 독자는 누가 이기느냐가 아니라, 어떻게 이기느냐에 몰입한다.

---

## 1. 전투 장면의 구조

### 1.1 전투의 3층 구조

| 층 | 요소 | 역할 |
|----|------|------|
| **전술층** | 동작, 기술, 공격/방어 | "무엇을 하는가" |
| **전략층** | 판단, 계획, 기만 | "왜 그렇게 하는가" |
| **감정층** | 두려움, 각오, 분노, 희열 | "어떻게 느끼는가" |

**[인기]** 3층을 모두 사용하는 전투 = 명장면
**[비인기]** 전술층만 있는 전투 = "검을 휘둘렀다. 피했다. 다시 찔렀다." → 지루

### 1.2 전투 장면의 리듬

```
[준비] → [충돌] → [교착] → [반전] → [결정타] → [여파]

준비: 긴장감 축적, 전략 수립 (느린 문장)
충돌: 첫 교전, 빠른 전개 (짧은 문장)
교착: 주도권 다툼, 팽팽한 균형 (중간 문장)
반전: 형세 역전, 숨겨진 수 공개 (가속)
결정타: 클라이맥스의 한 순간 (시간 팽창)
여파: 결과와 대가 (느린 문장, 감정 처리)
```

## 2. 전투 묘사 기법

### 2.1 속도 조절

**가속 기법** (긴장감 ↑):
```
짧은 문장. 끊기는 호흡.
주어 생략.
동사 연속.

예시:
"뛰었다. 피했다. 발밑이 무너졌다. 구르는 사이, 검날이 스쳤다.
피가 튀었다—자기 것인지 상대 것인지 알 수 없었다."
```

**감속 기법** (임팩트 ↑):
```
긴 문장, 세밀한 묘사, 시간의 팽창.
0.1초를 반 페이지로 늘림.

예시:
"검끝이 떨렸다. 그 미세한 진동이 손목을 타고 팔꿈치까지 올라오는 동안,
카이의 눈은 상대의 발끝에 묻은 먼지가 오른쪽으로 쏠리는 것을 포착했다.
왼쪽이다. 상대는 왼쪽으로 빠질 것이다."
```

**[팁]** 교대 사용이 핵심:
- 초반 충돌: 가속
- 중간 교착: 가속↔감속 교대
- 결정타 직전: 극한 감속 (시간 정지)
- 결정타: 가속 1문장
- 여파: 감속

### 2.2 감각별 묘사

| 감각 | 전투에서의 활용 | 예시 |
|------|---------------|------|
| **시각** | 동작, 빛, 색채, 표정 | "검날에 반사된 불빛이 눈을 찔렀다" |
| **청각** | 충돌음, 비명, 호흡, 침묵 | "금속이 부딪히는 날카로운 비명" |
| **촉각** | 충격, 진동, 온도, 통증 | "충격이 팔을 타고 어깨까지 울렸다" |
| **후각** | 피, 화약, 땀, 흙 | "철 냄새가 혀끝까지 올라왔다" |
| **미각** | 피, 흙, 공포의 맛 | "입안에 번지는 쇳내" |
| **고유감각** | 균형, 속도감, 가속도 | "몸이 기울었다—중력이 달라진 것처럼" |

**[필수]** 전투 장면에서는 시각+청각+촉각 최소 3감각 사용
**[인기]** 고유감각(균형, 가속도)을 사용하면 몰입감 급상승

### 2.3 내면 독백과 전투의 병행

**좋은 예** (판단 과정을 보여줌):
```
"왼쪽 팔이 늦다. 부상인가, 페이크인가.
부상이라면 지금이 기회다. 하지만 페이크라면—"
검을 멈추었다. 그리고 그 선택이 옳았다는 것을,
상대의 눈에 스친 실망이 증명했다.
```

**나쁜 예** (전투 도중 장황한 회상):
```
"검이 날아온 순간, 카이는 어린 시절을 떠올렸다.
아버지가 처음 나무 검을 쥐어주던 날, 뒤뜰에는
이름 모를 꽃이 피어 있었고..." → 전투 긴장 파괴
```

**원칙**: 전투 중 내면 독백은 **전술적 판단**만. 감정/회상은 전투 전후에.

## 3. 전투 유형별 묘사법

### 3.1 일대일 결투

- 초점: 기술, 심리전, 상대 분석
- 감정: 존중, 두려움, 집중
- 묘사 비중: 전술 40% / 전략 30% / 감정 30%
- 시점: 주인공 내면 + 상대의 외면적 변화

### 3.2 다대다 전투 / 집단전

- 초점: 전체 전황 → 개인 → 전체 (줌인/줌아웃)
- 감정: 혼란, 연대, 각자의 사투
- 묘사 비중: 전술 30% / 전략 40% / 감정 30%
- **[팁]** 집단전에서는 "주인공 주변 반경 3m"에 집중 → 가끔 전체 조감

### 3.3 추격전

- 초점: 속도, 장애물, 선택의 연속
- 감정: 공포(도주자) / 흥분(추격자)
- 묘사: 환경 묘사 비중 ↑ (길, 장애물, 지형)
- **[팁]** 추격전은 문장 길이를 짧게 유지

### 3.4 마법/초능력 전투

- 초점: 능력의 시각적 표현, 규칙 내 창의적 활용
- 감정: 경외, 한계 인식, 각성
- 묘사: 에너지 흐름, 색채, 공간 변화
- **[필수]** 능력의 "대가"를 반드시 묘사 (체력 소모, 반동, 한계)

### 3.5 심리전/두뇌전

- 초점: 추론 과정, 상대의 표정 변화, 정보전
- 감정: 긴장, 자신감↔불안 교대
- 묘사: 대사 비중 ↑, 침묵의 묘사
- **[팁]** 독자에게 힌트를 주되 답은 주지 마라

## 4. 전투 묘사의 금기

| 금기 | 이유 | 대안 |
|------|------|------|
| **동작 나열** | "찔렀다. 막았다. 때렸다." = 게임 로그 | 감정/판단 사이에 동작 삽입 |
| **파워 인플레** | 매번 더 강한 기술 → 기준 상실 | 상황/조건으로 난이도 변화 |
| **무적 주인공** | 긴장감 제로 | 반드시 대가/부상/실패 포함 |
| **구경꾼 해설** | "대단하다! 저건 전설의 기술!" | 전투 참여자의 시점으로 |
| **전투 중 장황한 설명** | 기술/세계관 설명으로 전투 멈춤 | 전투 전후에 설명, 전투 중은 직관 |
| **승패 예측 가능** | 결과가 뻔한 전투 = 읽을 이유 없음 | 조건/변수 추가로 불확실성 유지 |

**[비인기]** 최악의 전투 묘사:
```
"카이가 마법을 시전했다. 이 마법은 3등급 화염 마법으로,
프리즈마 에너지를 외부로 방출하여 고열의 화염구를 생성하는
기술이다. 화염구는 상대를 향해 날아갔고, 상대는 방어 마법으로
막았다. 방어 마법은..." → 백과사전, 전투 아님
```

## 5. 전투 장면 워크시트

```
전투 유형: [결투/집단전/추격/마법전/심리전]
참여자:
위치/환경:
승패 조건:
감정 시작점 → 끝점:

전투 흐름:
  준비 — (무엇을 계획하는가)
  충돌 — (첫 교전의 결과)
  교착 — (어디서 균형이 잡히는가)
  반전 — (무엇이 형세를 바꾸는가)
  결정타 — (어떻게 끝나는가)
  여파 — (승자/패자의 반응)

대가: (부상? 소모? 관계 변화? 정보 획득?)
이 전투가 서사에 기여하는 것: (성장? 관계? 정보? 반전?)
```
