# 묘사 기법

> "보여주기(Show) vs 말하기(Tell)"의 실전 활용.
> 오감 묘사, 환경/인물/감정/동작 묘사의 기술.

---

## 1. Show vs Tell의 원칙

### 1.1 기본 원칙

| | Show (보여주기) | Tell (말하기) |
|-|----------------|--------------|
| **방법** | 행동/감각/대사로 전달 | 서술자가 직접 설명 |
| **효과** | 독자가 체험 → 강한 몰입 | 빠른 정보 전달 → 효율적 |
| **적합** | 핵심 장면, 감정, 관계 | 배경 요약, 시간 경과, 부차 정보 |

**예시**:
```
Tell: 그녀는 화가 났다.
Show: 그녀의 손이 테이블 위의 컵을 움켜쥐었다. 손가락 끝이 하얗게 변했다.

Tell: 그는 슬펐다.
Show: 그는 아무 말 없이 창밖을 바라보았다. 한참이 지나서야 고개를 돌렸는데, 눈이 붉었다.
```

### 1.2 Show와 Tell의 적절한 배합

**[필수]** 100% Show는 비효율적. Show와 Tell의 배합이 핵심.

| 상황 | Show/Tell | 이유 |
|------|-----------|------|
| 핵심 감정 장면 | Show 100% | 독자 체험 극대화 |
| 중요 대화 | Show 90% + Tell 10% | 간간이 감정 서술 보조 |
| 일반 서술 | Show 60% + Tell 40% | 효율적 진행 |
| 시간 경과/이동 | Tell 80% + Show 20% | 빠른 전환 |
| 배경 설명 | Tell 70% + Show 30% | 정보 효율 + 분위기 |

## 2. 오감 묘사

### 2.1 감각별 특성

| 감각 | 효과 | 자주 잊히는 정도 | 활용 팁 |
|------|------|-----------------|---------|
| **시각** | 장면 구축, 직관적 | 거의 안 잊힘 | 과도 의존 주의, 다른 감각과 조합 |
| **청각** | 분위기, 현장감 | 보통 | 침묵의 묘사도 강력 |
| **촉각** | 친밀감, 물리적 실재감 | 자주 잊힘 | 온도, 질감, 통증, 바람 |
| **후각** | 기억 환기, 강한 감정 연결 | 매우 자주 잊힘 | 후각은 기억과 연결 강함 |
| **미각** | 일상성, 편안함 | 자주 잊힘 | 음식 장면 외에도 활용 가능 |

### 2.2 복합 감각 묘사

**[인기]** 한 장면에 2~3개 감각 조합:
```
시각+청각: "안개가 자욱했다. 어디선가 까마귀 울음소리가 들렸다."
촉각+후각: "축축한 돌벽에 등을 기대자, 곰팡이 냄새가 올라왔다."
시각+촉각+청각: "금빛 햇살이 피부에 닿았다. 따뜻했다. 멀리서 종소리가 울렸다."
```

### 2.3 감각의 전이 (공감각)

```
"그녀의 목소리는 꿀처럼 달았다." (청각→미각)
"색이 차가웠다." (시각→촉각)
"그 침묵은 무거웠다." (청각→촉각)
```
→ 절제하여 사용하면 인상적, 과다 시 과장

## 3. 환경/풍경 묘사

### 3.1 기능적 환경 묘사

**[필수]** 풍경 묘사는 반드시 기능을 가져야 함:

| 기능 | 예시 |
|------|------|
| 분위기 설정 | 어두운 숲 → 불안 |
| 캐릭터 심리 반영 | 비 오는 날 → 슬픔 |
| 복선/정보 전달 | 높은 성벽 → 폐쇄성, 전쟁 대비 |
| 시간/계절 표현 | 낙엽 → 가을, 시간의 흐름 |
| 대비/상징 | 전투 후 핀 꽃 → 희망/생명 |

### 3.2 카메라 기법

| 기법 | 설명 | 효과 |
|------|------|------|
| 와이드 샷 | 전체 풍경 → 세부 | 스케일감, 도입 |
| 클로즈업 | 디테일 하나에 집중 | 강조, 복선 |
| 팬 | 시선이 이동하며 훑음 | 공간감 |
| 주관적 | 캐릭터의 눈으로 | 감정 이입 |

```
와이드→클로즈업 예시:
"들판이 끝없이 펼쳐져 있었다. 지평선 위로 구름이 빠르게 흘러갔다.
— 그런데 발밑의 풀이 이상했다. 끝이 검게 타들어 있었다."
```

## 4. 인물 묘사

### 4.1 외형 묘사 원칙

**[금기]** 경찰 수배서식 묘사:
```
❌ "그녀는 키가 170cm이고, 갈색 머리에, 파란 눈을 가졌으며, 마른 체형이었다."
```

**[인기]** 특징적 인상 + 행동 결합:
```
✅ "그녀가 고개를 돌렸을 때, 가장 먼저 눈에 들어온 것은 이마 위의 흉터였다.
   오래된 것이었다. 본인은 신경 쓰지 않는 듯 머리카락으로 덮지도 않았다."
→ 외형 + 성격(당당함) + 역사(과거 사건) 동시 전달
```

### 4.2 묘사 타이밍
- 첫 등장 시: 1~2가지 인상적 특징만
- 이후: 행동/대사 속에서 점진적으로 추가
- **절대 하지 말 것**: 거울 앞에서 자기 외모 설명

## 5. 동작 묘사

### 5.1 미세 동작의 힘

**[인기]** 큰 동작보다 미세한 동작이 캐릭터를 살림:
```
"그는 화가 났다" → 추상적
"그의 턱 근육이 움찔거렸다" → 구체적, 시각적
```

**미세 동작 라이브러리**:
| 감정 | 미세 동작 |
|------|----------|
| 긴장 | 손가락 꼼지락, 입술 깨물기, 어깨 경직 |
| 불안 | 시선 회피, 발 끝 두드리기, 팔짱 |
| 호감 | 몸 기울이기, 미소, 시선 유지 |
| 거짓 | 코 만지기, 시선 우상방, 말 빨라짐 |
| 분노 | 주먹 움켜쥐기, 호흡 거칠어짐, 턱 굳어짐 |
| 슬픔 | 고개 숙이기, 느린 동작, 한숨 |

## 6. 감정 묘사

### 6.1 감정의 물리화

> 감정을 "느꼈다"가 아닌 **신체 반응으로 전환**

| 감정 | 물리적 표현 |
|------|------------|
| 두려움 | "등줄기를 타고 찬 것이 흘러내렸다" |
| 기쁨 | "가슴 안에서 뭔가 부풀어 오르는 느낌" |
| 분노 | "관자놀이가 뜨거워졌다" |
| 슬픔 | "목 안이 조여들었다" |
| 놀라움 | "숨이 멎었다" |
| 사랑 | "심장이 갈비뼈를 때렸다" |

### 6.2 감정의 층위

**[인기]** 복합 감정이 단일 감정보다 현실적이고 깊음:
```
단일: "그는 슬펐다."
복합: "슬프면서도 한편으로는 안도했다. 그리고 그 안도감이 다시 슬프게 만들었다."
```
