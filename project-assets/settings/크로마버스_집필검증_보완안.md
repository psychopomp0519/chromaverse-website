## 추가 검증사항 (12항목)

### 1. 화별 기능 검증

- **검증**: 집필 완료 후, 이 화의 기능을 한 문장으로 적는다
- [ ] 이 화를 삭제하면 전체 서사에 구멍이 생기는가?
- [ ] 이 화의 기능은 다음 셋 중 최소 2개를 충족하는가? (플롯 전진 / 캐릭터 변화 / 세계관 심화)
- [ ] 이 화에서 "상태 변화"가 있는가? (화의 시작과 끝에서 무엇이 달라졌는가를 명시)
- **막는 실패**: "풍경화" — 분위기는 좋지만 사건 전진이 없어 독자 이탈

### 2. 장면 단위 변화량 검증

- **검증**: 각 장면(Scene)에 대해
- [ ] 장면 시작 상태와 종료 상태가 다른가? (정보/감정/관계/상황 중 최소 1개)
- [ ] 장면 내에 미시 갈등(micro-conflict)이 있는가?
- [ ] 장면을 삭제하면 다음 장면이 성립하지 않는가? (아니라면 통합/삭제 검토)
- [ ] 장면이 종료될 때 다음 장면으로의 추진력이 남는가?
- **막는 실패**: "장면은 예쁜데 읽고 나면 뭐가 진행됐는지 모르겠다"

### 3. 설명 대 사건 비율 검증 ★즉시 적용

- **검증**: 각 화를 문단 단위로 분류
  - [E] 설명/묘사/내면 독백
  - [A] 사건/행동/대화
- [ ] 1~4화: E ≤ 50%, A ≥ 50%
- [ ] 5화 이후: E ≤ 40%, A ≥ 60%
- [ ] 설명 블록이 5줄 이상 연속되는 곳이 있는가? → 중간에 행동/대화 삽입
- **막는 실패**: "설정은 탄탄한데 재미를 너무 늦게 느낀다"

### 4. 카이의 능동성 강도 검증 ★즉시 적용

- **검증**:
- [ ] 이 화에서 카이가 스스로 개시한(외부 압력이 아닌) 선택이 최소 1회 있는가?
- [ ] 그 선택에 가시적 결과(상태 변화)가 뒤따르는가?
- [ ] 그 선택이 카이의 동기를 드러내는가?
- [ ] 카이의 행동이 70% 이상 반응적인가? → 경고
- **능동성 유형** (매 화 최소 1개):
  - 규정/기록을 무기로 사용
  - 위험을 감수하고 기록을 선택
  - 타인을 위해 자신을 노출
  - 데이터에 기반한 판단/주장
- **막는 실패**: "캐릭터는 좋은데 행동성이 약하다"

### 5. 선택의 대가 검증

- **검증**:
- [ ] 이 화에서 주요 선택에 대가가 존재하는가?
- [ ] 대가의 유형은? (시간/자원/관계/안전/정보)
- [ ] 대가가 이후 화에서 회수되는가?
- [ ] 대가 없이 성공하는 장면이 연속 2회 이상인가? → 긴장 감소 경고
- **막는 실패**: "위기인 것 같은데 긴장이 안 된다"

### 6. 화말 훅 유형 검증 ★즉시 적용

- **유형 분류**: 정보형 / 위기형 / 관계형 / 반전형 / 도착형
- [ ] 직전 2화와 같은 유형의 훅인가? → 변경 필요
- [ ] 훅이 2화 이내에 해소/진행되는가?
- [ ] 훅의 강도가 화 내 감정 고점과 연결되는가?
- **막는 실패**: "후킹은 있는데 매번 같은 방식이라 단조롭다"

### 7. 반복 표현/자기복제 검증

- **검증**:
- [ ] "기록관은 ~한다" 형태가 3화 이내에 2회 이상 반복되는가?
- [ ] "서첩에 적었다" 기록 행위가 한 화 내 5회 이상인가?
- [ ] 의미심장한 마감 문장이 연속 3개 문단에 나타나는가?
- [ ] 동일한 비유/은유가 5화 이내에 재사용되는가?
- **누적 추적 대상**: "정확한 것은 버리지 않는다", "기록관은 기록한다" 변형, "물음표, 별표", "괜찮아요"
- **막는 실패**: "좋은 문장이 많지만 전체 리듬이 무거워진다"

### 8. 인물 기능 중복 검증

- **검증**:
- [ ] 이 화에서 각 인물이 고유한 기능을 수행하는가?
- [ ] 두 인물이 같은 기능을 하고 있는가?
- [ ] A의 대사를 B에게 옮겨도 어색하지 않다면 → 기능 중복
- **현재 기능 맵**: 카이(기록/데이터 저항), 레이(안정화/역설적 강인함), 닉스(긴장/자기혐오 속 인간성), 엔(완충/날카로운 관찰), 코다(압도적 능력/침묵)
- **막는 실패**: "캐릭터가 많은데 다 비슷하게 느껴진다"

### 9. 독자 오해 관리 검증

- **검증**:
- [ ] 의도적으로 숨긴 정보가 "궁금하다"를 유발하는가, "뭔 소리인지 모르겠다"를 유발하는가?
- [ ] 의도하지 않은 오해가 발생할 수 있는 서술이 있는가? (시간순서 모호, 주어 불명확)
- [ ] 독자가 이 화 시점에서 알아야 하는 최소 정보가 제공되었는가?
- [ ] 미해결 질문이 동시에 7개 이상인가? → 정보 과부하 경고
- **막는 실패**: "미스터리를 숨겼는데 혼란만 생김"

### 10. 감정 곡선 검증

- **검증**: 각 화의 감정을 5단계로 기록 (1=평온 ~ 5=절정)
- [ ] 화 내 최소 1개 감정 고점(4 이상)이 있는가?
- [ ] 고점 전에 저점(2 이하)이 존재하는가?
- [ ] 화가 평평한 감정선(3-3-3-3)으로 끝나는가? → 경고
- [ ] 직전 3화와 감정 곡선 패턴이 동일한가? → 변주 필요
- [ ] 고구마가 5화 연속 해소 없이 이어지는가? → 필수 사이다 삽입
- **막는 실패**: "좋은 문장인데 감정적으로 밋밋하다"

### 11. 물리적 동선 검증

- **검증**:
- [ ] 캐릭터의 위치 이동이 시간순으로 모순 없는가?
- [ ] 이동 소요 시간이 서사적 시간과 맞는가?
- [ ] 이전 화 끝의 위치와 이번 화 시작의 위치가 일치하는가?
- [ ] 공간 제약(잠긴 문, 무너진 벽)이 다음 장면에서 유지되는가?
- **추적**: 매 화 시작/종료 위치 기록
- **막는 실패**: "아까 문 잠갔다고 했는데 어떻게 나왔지?"

### 12. 숫자/수치 시스템 정합성 검증

- **검증**:
- [ ] 레이 크로마 총합이 직전 화와 일치하는가? (레이_크로마_추적.md 대조)
- [ ] 승급 심사 카운트다운이 정확한가?
- [ ] 닉스의 빛 흡수 반경이 이전 화 수치와 모순되지 않는가?
- [ ] 새 크로마 값이 계층 범위에 맞는가? (탄사이 100~299, 츄사이 300~549 등)
- [ ] 시간 경과 표현이 실제 화수 간 시간과 일치하는가?
- [ ] 레스 지출이 보유량과 맞는가?
- **막는 실패**: "수치 기반 세계관인데 숫자가 안 맞다"

## 진행 방향 보완

### A. 3화 묶음 검증 도입

매 화 검증에 더해, 3화를 묶어 추가 검증 1회 실시.

| 시점 | 검증 내용 |
|------|----------|
| 매 화 | 플롯 대조, 설정 정합, 독자 관점, 추적 업데이트 (기존) |
| 3화 묶음 | 감정 곡선 흐름, 훅 유형 다양성, 설명:사건 비율 추세, 카이 능동성 누적, 반복 표현 |

### B. 작가용 vs 독자용 검증 분리

| 관점 | 질문 |
|------|------|
| 작가용 | 설정 맞는가? 용어 맞는가? 숫자 맞는가? 플롯 일치하는가? |
| 독자용 | 넘기고 싶은가? 카이를 응원하는가? 다음 화 궁금한가? |

**독자용 핵심 5문** (매 화 필수):
1. 첫 3줄을 읽고 계속 읽고 싶은가?
2. 가장 기억에 남는 한 장면은?
3. 카이가 이 화에서 무엇을 했는가? (관찰 말고)
4. 화 끝에서 다음 화를 바로 펼치고 싶은가?
5. 이 화를 한 문장으로 요약하면?

### C. 중도 이탈 방지 "3화 단위 후킹 체크"

| 구간 | 체크 |
|------|------|
| 1~3화 | 카이 호감 근거 + 세계관 특별함 1개+ |
| 4~6화 | 레이(핵심 이벤트) 존재감이 서사 지배 |
| 7~9화 | 긴장 에스컬레이션 가속 + 카이 능동적 선택 반복 |
| 10~12화 | 새 인물/세력으로 스케일 확장 |

### D. Scene-Sequel 패턴 적용

- **Scene**(능동): 목표 → 갈등 → 재앙/결과
- **Sequel**(반응): 반응 → 딜레마 → 결정 → 다음 Scene

비율: 일반 화 6:4, 위기 화 8:2, 회복 화 4:6

### E. 문장 밀도 리듬 가이드

무거운 문장(비유, 주제, 의미) vs 가벼운 문장(행동 신호, 전환, 단순 사실)

리듬: `가벼운 — 가벼운 — 무거운 — 가벼운 — 가벼운 — 무거운(핵심)`

- [ ] 무거운 문장 3개 연속인가? → 사이에 가벼운 문장 삽입
- [ ] 화 전체 무거운 비율 75% 초과인가? → 경고 (1~6화: 60% 이하 권장)
