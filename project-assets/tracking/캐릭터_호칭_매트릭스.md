# 캐릭터 호칭 매트릭스

> 행: 부르는 사람 / 열: 불리는 사람 / 값: 사용 호칭
> 호칭 변화 시 "(N화~)" 표기로 변경 시점 기록

## 파티 '노이즈' 내부

| 부르는 사람 \ 대상 | 카이 | 코다 | 닉스 | 레이 |
|-------------------|------|------|------|------|
| **카이** (서술/내면) | — | 코다 | 닉스 | 레이 |
| **코다** | 카이 / 스크라이브 | — | 닉스 | 레이 |
| **닉스** | (이름 안 부름) / 기록관 | 코다 | — | 레이 |
| **레이** | 카이 씨 | 코다 씨 | 닉스 씨 | — |

## 1~2화 실제 사용 호칭

| 부르는 사람 | 카이를 | 비고 |
|-----------|--------|------|
| **엔** (선배) | "카이" / "넌" | 비격식, 친밀 |
| **카이** (서술) | 엔을 "엔 선배" "엔" | 존칭+이름 |
| **노점 아저씨** | 직접 호명 없음 | "살이 없잖아" (2인칭 회피) |
| **아이 어머니** | 호명 없음 | "모르는 사람" |
| **츄사이 의뢰인** | "탄사이" (비하) | 이름이 아닌 계층으로 지칭 |

## 3~5화 추가 인물

| 부르는 사람 | 카이를 | 비고 |
|-----------|--------|------|
| **호쿠 할아버지** | "자네" | 카겐 이웃 노인, 3화 등장 |
| **코드 밖 여자** | 호명 없음 | "코드에서 왔어?" "기록해 줘" |
| **카이** (서술) | 레이를 "이 렌" | 아직 이름 모름, 5화 첫 대면 |
| **레이** | 자칭 없음 | "이름이... 있나요, 저?" |
