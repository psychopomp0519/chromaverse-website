# 독자 인지 추적

> 독자가 각 화 시점에서 무엇을 알고, 모르고, 추측하고, 느끼는지를 추적한다.
> 매 화 집필 완료 후 업데이트.

**통합 대상**: 이 문서는 기존 3개 추적 파일의 역할을 흡수한다.
- ~~용어_도입_추적.md~~ → 제4장 "용어 이해도"
- ~~감정곡선_추적.md~~ → 제6장 "감정 상태"
- ~~위기_에스컬레이션_추적.md~~ → 제1장/제2장 "독자가 아는 것/모르는 것"

**별도 유지 파일** (이 문서와 용도가 다름):
- `레이_크로마_추적.md` — 설정 내부 수치 추적
- `캐릭터_호칭_매트릭스.md` — 호칭 레퍼런스
- `플롯_작성_진행현황.md` — 프로젝트 관리

#### 1. 독자가 아는 것

- **[신규]** 카이의 동기: "세상을 적고 싶다" — 존재 증명으로서의 기록
- **[신규]** 카이는 오차 제로, 3년간 400명 측정. 승급 심사 22일 전
- **[신규]** 이 세계에는 렌(종족), 이로몬(색채 표지), 이로쥬(색채 생물)가 있음
- **[신규]** 이로몬 밝기로 계층이 나뉨: 탄사이(낮음) < 츄사이(높음)
- **[신규]** 코드(도시 단위), 레스(화폐), 공명 구조(건축 에너지)
- **[신규]** 상부에서 극비 밀봉 서신이 카이에게 직접 내려옴
- **[신규]** 크로마스톰이 이번 달만 세 번째 — 빈도 급증 중
- **[신규]** 광명회 소성당이 존재 — 계층 구조와 연관

#### 2. 독자가 모르는 것

- 밀봉 서신의 내용 → 2화
- 왜 견습인 카이에게 극비 임무가 내려왔는가 → 2~3화

- 카이의 스승 — 필사본만 남아 있음 → 대막1 후반~
- 엔 선배의 숨겨진 면 — 발끝 습관, 극비 서신에 대한 불안 → 대막1 후반
