누적 합계: 1~11화 총 3회 사용. 규칙(에피소드 아크 10회 이내) 충분히 여유 있음.

## 제6장. 매 화 훅(갈고리) 현황

| 화 | 훅 내용 | 다음 화 연결 |
|----|---------|-------------|
| 1화 | 엔: '이상한 의뢰가 들어왔어' + 밀봉 서류 | → 2화: 이상한 의뢰 내용 공개 |
| 2화 | '정식 기록에 남기지 말 것' — 기록 금지 명령 | → 3화: 카이의 내적 갈등 심화 |
| 3화 | 집 문 앞에 두 번째 밀봉 서신 도착 | → 4화: 제3구역으로 출발 |
| 4화 | 스톰 잔해 속 에너지 방향성 집중 + 렌즈 진동 | → 5화: 레이 발견 |
| 5화 | 레이 눈을 뜸: '...여기가 어디예요?' | → 6화: 레이 측정 + 정체 탐구 |
| 6화 | 레이가 카이에게 '적어주셔서 감사해요' + 미소 | → 7화: 레이와의 관계 심화 |
| 7화 | 상부 서신: '즉시 보고하라. 대상은 격리 조치한다' | → 8화: 상부 대응 |
| 8화 | 엔: '상부에서 사람을 보냈어. 직접.' | → 9화: 관리와 대면 |
| 9화 | 관리 손이 레이에 가까워지자 이로몬 요동 + 렌즈 진동 | → 10화: 현상 분석 |
| 10화 | 카이 실험: 총합 역치 확인 + 레이 이름 기원 기억 | → 11화: 시한 내 추가 관찰 |
| **11화** | **공명탑 맥동 정지. 코드 외곽에서 무너지는 소리. "코드가 무너지고 있다."** | **→ 12화: 코드 붕괴** |
