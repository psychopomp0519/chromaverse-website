## Lessons Learned

### What Went Well

1. **A-B 분리 전략 효과**
   - A항목(설정 결정)을 먼저 문서에 반영한 후 플롯에 적용하는 방식이 정합성 유지에 효과적
   - 설정 우선(설정집 → 마스터 문서 → 플롯) 원칙이 작업 효율성과 품질 향상에 기여

2. **1화 단위 분리의 서사 강화**
   - 묶음 화수 24화를 개별화하면서 각 화에 고유한 서사 포인트 추가
   - 초반(1~40화)에 비해 후반부(41~80화)의 강도가 높아지는 자연스러운 에스컬레이션

3. **캐릭터 역할의 점진적 성장**
   - 카이의 역할(분석 → 지휘 → 외교)이 화수별로 단계적으로 성장하도록 표현
   - 4인 팀의 역할 분담(코다 조율, 닉스 흡수, 레이 정화, 카이 분석/안내)이 명확

4. **생활풍습 자연 도입**
   - 세계관 노출이 아닌 "일상 속 자연스러운 세계관 체감"에 성공
   - 초반 3화에 의식주/교통/통신/경제 풍습을 부자연스럽지 않게 삽입

### Areas for Improvement

1. **코다-레이 관계 심화 필요**
   - 캐릭터 관계 점수(8.5/10)
   - 대막2 이후에 더 깊은 감정적 연결 필요
   - 제안: 대막1 후반부에 복선 강화

2. **개별 화의 디테일 보강**
   - 42화: 종자 측정 장면의 긴장감 강화
   - 49화: 새로운 패턴 발견의 임팩트 확대
   - 76화: 최종 선택 앞의 카이 심리 표현 심화

3. **61화 정보량 조절**
   - 페이스 점수(8.5/10)
   - 전투 장면에 RGB 채널명, 역할, 외부 영향 등이 동시 노출
   - 제안: 61~73화 전투 시퀀스에 호흡 장면 추가

4. **대막2 정합성 사전 검증**
   - 현재 대막2 플롯과의 인수인계 검증 미실시
