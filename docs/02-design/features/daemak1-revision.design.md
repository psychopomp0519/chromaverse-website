### S-02: RGB 총합 분포

**대상 파일**: 세계관설정집 4.1 섹션
**추가 내용**:

**대상 파일**: 세계관설정집 새 섹션 (5.X 또는 적절한 위치)
**추가 내용**: A10 확정 사항 전체 (의/식/주/수입/교통/통신)

### S-05: 카이 집필 시점/이유/대상

**대상 파일**: 마스터 문서 3.1~3.2 섹션

- 57화 "700년 뒤" → 기록의 잔영이 시간 초월 직관 부여 (복선 주석 추가)

### S-06: 코덱스 설정

**대상 파일**: 마스터 문서 22.1 카이 캐릭터 시트 + 부록A

- 코덱스: 기록관 정식 자격자 전용 참고서. 세계 각지의 크로마 데이터/역사/종족 정보 집성

### S-07: 메타 프레임

**대상 파일**: 마스터 문서 새 섹션 (23.X 대막7 후일담 근처)

## 2. Phase 2-1: 묶음 화수 분리 (11개 → 24화 개별화)

### 분리 원칙
- 각 화에 **고유 제목** 부여
- 각 화에 **독립적 서사 전개** (도입-전개-훅)
- 기존 묶음 내용을 분배하되, 부족하면 **신규 서사 추가** (분량 축소 불가)
- 태그(비밀/사이다/고구마/훅 등)도 화별로 재분배

### 분리 상세

| 묶음 | 분리 후 | 각 화 서사 방향 |
|------|---------|---------------|
| **43~44화** | 43화: 소규모 코드 도착 + 병든 컨덕터 발견 / 44화: 코다의 일시 조율 + 블랑 서명 감지 | 기존 내용 분배 |
| **45~46화** | 45화: 블랑의 영향력 증거 수집 / 46화: 코다의 첫 신념 표출 + 카이의 질문 | 기존 내용 분배 |
| **48~49화** | 48화: 4인 관계 변화 (닉스→레이 교류) / 49화: 새 코드 도착 + 새로운 디소넌스 감지 | 기존 내용 분배 |
| **51~53화** | 51화: 전투 시작 + 주민 대피 + 카이 첫 지휘(어설픔) / 52화: 닉스 K 충돌 + 레이 정화 복귀 / 53화: 코다 조율 완성 + 외부 에너지 '의지' 발견 | 전투 시퀀스 3단계 |
| **55~56화** | 55화: 코다 쓰러짐 + **카이 감정 폭발**(펜 부러짐) + 레이 위로 / 56화: 레이 치료 시도 실패(공명 차단 후유증) + 코다 회복 + 첫마디 | 감정 절정→회복 2단계 |
| **57~58화** | 57화: 카이-코다 심야 대화 (집필 복선) / 58화: 닉스-레이 대화 ("아무것도 아닌 게 편하다" + 닉스 울림 해석) | 대화 씬 분리 |
| **59~60화** | 59화: 일상 + 코다 이름 부르기 + 이전 호칭 명시 / 60화: 코다의 엔지 행선지 제안 + 3권 마무리 훅 | 일상→전환 |
| **62~63화** | 62화: 코다 과거 2차 편린 / 63화: 닉스-코다 **거울상** 발견 + 엔지 진입 | A9 반영 |
| **64~66화** | 64화: 엔지 전장 참혹함 + 카이 기록 시도(떨림) / 65화: 코다 조율 중재 거부 / 66화: 닉스 어린 렌 구출 + 플린트 등장 예고 | 전쟁 시퀀스 3단계 |
| **68~70화** | 68화: 내전의 정보전 + 카이 증거 수집 / 69화: 카이가 양측 기록 비교로 광명회 개입 증거 발견 / 70화: 플린트에게 증거 제시 + 플린트 결심 | 탐정→증거→결단 3단계 |
| **71~73화** | 71화: 대규모 조율 준비 + **카이 외교 심리 묘사** / 72화: 조율 실행 + 4인 역할 상세 / 73화: 조율 성공 + 세이지 급보 | 준비→실행→결과 3단계 |

## 3. Phase 2-2: A항목 플롯 반영

### A-11: 프롤로그 (번역자 서문)

**위치**: 1화 이전, 플롯 파일 최상단
**내용**:
```
### 프롤로그: 번역자 서문

"나는 이 서첩을 발견한 사람입니다.
어떻게 이것이 내 손에 닿았는지는 이 기록의 마지막에 적겠습니다.
다만 한 가지 — 이것은 다른 세계의 기록입니다.
기록한 이의 이름은 카이. 직업은 크로마 스크라이브.
그가 무엇을 보았고, 무엇을 적었는지, 이제부터 읽게 됩니다."
```

### A-01: 레스/하쿠세이 체계 반영

**수정 대상 화수**:
- **42화**: "레스 광맥에서 하쿠세이를 채집" → 하쿠세이(원석)와 레스(화폐)의 차이를 카이가 설명. "하쿠세이는 원석이다. 코드의 정제소에서 가공해야 레스가 된다."
- **43~44화** (분리 후): 소규모 코드에 정제소가 없어 하쿠세이를 직접 물물교환하는 장면. 경제 격차 체감

### A-02: RGB 채널명 명시

**수정 원칙**: 능력 사용 시 채널명을 자연스럽게 삽입
- "R 에너지" → "아도르(열)" 또는 "R 채널 아도르"
- "G 에너지" → "비탈리스(생)"
- "B 에너지" → "코그니스(식)"
- 첫 등장 시 카이가 "기록관 용어로는~"이라며 도입

**적용 화수**: 17화(레이 능력), 22화(닉스 폭주), 34화(스톰), 39화(첫 협력), 51~53화(대규모 전투), 64~66화(엔지), 71~73화(대규모 조율)

### A-03: 보이드 호칭/효과 단계적 공개

| 화수 | 공개 내용 | 연출 |
|------|----------|------|
| 5~6화 | 레이 발견 시 주변 빛이 약해지는 이상 감지 | 카이 측정 이상 |
| 13~15화 | 닉스 근처에서 "존재 희석" 현상 | 주민들 두려움, 카이 관찰 |
| 22~23화 | K 폭주 시 보이드의 파괴력 체감 | 빛이 빨려 들어감 |

| 50화 | 보이드와 다른 외부 에너지(CMYK)의 차이 감지 | 카이 분석 |
| 54화 | 보더 너머 에너지와 보이드의 관련 암시 | 코다 언급 |

### A-05: 여행 동기 점진적 전환

| 권 | 화수 범위 | 동기 단계 | 반영 방법 |

### A-06: 노이즈 결성

**대상 화수**: 77화 "이름을 붙이다"
**수정 내용**:

## 4. Phase 2-3: B항목 개별 수정

### B-01: 5화 측정 대상 = 레이 알아낸 방법

**현재**: 5화는 크로마스톰 잔향에서 레이를 발견. 6화에서 측정.
**문제**: 4화의 "출처 불명 렌 극비 측정" 지시와 5화의 레이 발견이 어떻게 연결되는지 불명확.
**수정**: 4화 밀봉 서신에 "크로마스톰 잔해 지역에서 비정상 에너지 반응 감지. 해당 지점 탐색 후 대상 측정"이라는 구체적 지시 추가. 5화에서 카이가 지시받은 좌표를 따라가다가 레이를 발견.

### B-02: 38화 계급 차이 연출

**현재**: 탄사이가 츄사이 상위에게 질문하는 것이 놀라운 일이라는 기술만 있음.
**수정**: 38화에 추가 — 주변 렌들의 반응("탄사이가 감히?"), 카이 자신의 내면("손이 떨렸다. 하지만 기록관은 질문해야 한다"), 상대방의 동요("...뭐라고?")를 구체적으로 묘사.

### B-03: 성별 힌트 추가

| 캐릭터 | 현재 | 추가 위치 | 연출 방법 |
|--------|------|----------|----------|
| 카이 | 미표시 | 1~2화 | 신체 묘사(목소리, 체격) 또는 타인의 호칭으로 자연스럽게 |
| 닉스 | 미표시 | 15화 첫 등장 | 외모 묘사에 성별 힌트 포함 |
| 레이 | 23화 "여자" | 유지 | — |
| 코다 | 31화 "남자" | 유지 | — |

### B-04: 58화 닉스 울림 이유

**수정**: 닉스 내면 해석 추가
- 레이의 "아무것도 아닌 게 편하다" → 닉스에게 울림을 주는 이유: 닉스도 보이드 때문에 "없는 존재"가 되고 싶었던 적이 있음. 존재 자체가 타인에게 해를 끼치므로 "아무것도 아닌" 상태를 갈망. 레이가 그것을 아무렇지 않게 말한 것이 닉스의 고독과 공명.
- 카이의 관찰: "닉스가 처음으로 누군가의 말에 무장해제된 것을 보았다."

### B-05: 59~60화 이전 호칭

**수정**: 59화에 추가
- 카이가 코다를 이전에는 "당신" 또는 "조율사"라고 불렀다는 것을 명시
- "카이가 처음으로 '코다'라고 이름을 부른다. 이전까지는 줄곧 '당신' 또는 '조율사'였다."

### B-07 + B-11: 전투 역할 명확화 + 해결 과정 보강

**전투 장면 수정 원칙**:
1. 각 인물이 **구체적으로 무엇을 하는지** 명시 (A7 역할 단계 참조)
2. 적이 **구체적 존재**로 등장 (에너지 현상뿐 아니라 실체적 위협)
3. 해결 과정에 **기승전결** 부여 (위기→시도→실패→재시도→성공)

**주요 전투 화수별 수정**:

| 화수 | 현재 문제 | 수정 방향 |
|------|----------|----------|
| 16~17화 | 자경단 포위→레이 빛→해결. 단순 | 카이가 규정으로 시간 번 것이 기승, 자경단 재돌입이 전, 레이 발현이 결 |
| 22~23화 | 닉스 폭주→레이 안정. 단순 | 폭주 규모 구체화, 카이가 패턴 분석으로 레이에게 접근 타이밍 알려줌(미숙하지만) |
| 51~53화 | 4인 팀플레이지만 역할 모호 | 각 화에서 **한 인물씩 핵심 기여** — 51화 카이(어설픈 지휘), 52화 닉스+레이(K 제어), 53화 코다(조율 완성) |
| 64~66화 | 전쟁 묘사이나 4인 역할 불명확 | 전투가 아닌 **관찰/중재/구출**로 역할 분화 |

### B-08: 71화 카이 외교 심리

**수정**: 71화 (분리 후) 카이의 외교 과정 심리 묘사 추가
- 탄사이 기록관이 양측 대코드 지도자 앞에 서는 긴장
- "내 말에 무게가 있을까" → "기록에 무게가 있다. 나 자신이 아니라."
- 코덱스의 외교 사례를 떠올리며 전략을 세우는 장면 (비밀리에)

### B-09: 레이 치료 시 RGB 감소 — 첫 치료에서만인 이유

**수정 대상**: 17화(첫 치료) + 22~23화(이후 치료) + 26화(줄어드는 빛)
**설명 추가**:
- 17화: 첫 발현은 레이가 **무의식적으로 자신의 크로마를 소모**하여 정화. 비효율적이라 총합 감소.
- 22~23화 이후: 레이가 능력을 학습하면서 **외부 에너지를 매개**로 정화. 자신의 크로마는 촉매 역할만. 단, 완전히 소모가 없는 것은 아님 — 극미량 감소가 누적.
- 26화 "줄어드는 빛": 카이가 미세한 총합 감소를 측정으로 감지. "매번 줄어든다. 아주 조금씩."

### B-10: 80화 "기록에 자신을 포함" 근거

**수정**: 80화에 근거 강화
- 대막1 전체에서 카이가 "이건 내 이야기가 아니다"를 반복해왔음을 상기
- 80화에서 카이가 서첩을 넘기며 자신의 행동을 발견: 55화에서 펜이 부러진 기록, 66화에서 미소 지은 기록, 75화에서 "선택"한 기록
- "나는 기록에 자신을 넣지 않으려 했다. 하지만 서첩은 이미 알고 있었다. 이건 — 내 이야기이기도 하다."

### B-12: 이로쥬/보쿠쥬 조기 등장

**현재**: 이로쥬 19화(울음소리만), 보쿠쥬 미등장
**수정**:
- **3화**: 코드 풍경에 이로쥬 묘사 추가 — "아이들이 작은 이로쥬(색채 생물)와 노는 장면. 붉은 이로쥬가 아이의 이로몬에 공명하며 빛난다"
- **5화**: 크로마스톰 잔해에서 다친 이로쥬 목격 — 세계 위기의 간접 증거
- **19화**: 이로쥬 울음소리를 "단순 울음"에서 "오염된 이로쥬의 비정상적 울음"으로 구체화
- **34화**: 크로마스톰 중 보쿠쥬(또는 CMYK 생물)의 그림자 목격 — 아주 짧게. "스톰 속에서 빛을 흡수하는 형체가 보였다. 카이가 기록하려 했지만 사라졌다."

**검증 항목**:
